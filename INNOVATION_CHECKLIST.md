# 创新性检查清单

每个方案在进入正式评分前必须逐项通过本清单。

## A. 机制创新

- [ ] 选择性机制是否超越了普通疏水分配、静电吸附或尺寸筛分？
- [ ] 是否提出了可被因果对照推翻的新识别原理？
- [ ] 该机制在目标水质场景中是否有独特的适用性（而非通用吸附）？

## B. 材料架构创新

- [ ] 材料架构是否超越了载体替换或孔径调参？
- [ ] 是否引入了新的空间组织方式（如预组织位点、受限腔体、梯度结构）？
- [ ] 合成路线是否具有方法学贡献（而非仅套用已有模板）？

## C. 仿生转译创新

- [ ] 生物原型到人工材料的转译是否超越了官能团类比？
- [ ] 是否提取了生物原型中不可由常规化学直觉得到的设计原则？
- [ ] 转译后的材料特征是否保留了原型的核心功能逻辑？

## D. 选择性策略创新

- [ ] 选择性来源是否可由单一物化参数（logKow、电荷、分子量）单调解释？若是则不通过
- [ ] 是否设计了至少两个正交识别维度的协同？
- [ ] 竞争物谱是否包含结构近似物而非仅不同类物质？

## E. 先例区分

- [ ] 是否完成了系统先例检索？
- [ ] 与最近先例的区别是否可被实验量化？
- [ ] 若先例已存在，本方案的增量贡献是否足以支撑独立发表？

## 判定规则

- A-E 五大维度中，至少一个维度必须展示明确的、可被实验验证的创新性
- 纯载体替换、参数优化、已发表平台的简单复用不算创新
- 未通过的维度必须标注原因和可能的改进方向
