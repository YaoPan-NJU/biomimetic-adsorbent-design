# Biomimetic Selective Adsorbent Design

Private research workspace for an independent, experiment-ready biomimetic adsorbent design. Codex conducts the research and adversarial design process; Pan Yao reviews the final Chinese package before laboratory handoff to Zhou Jiaqi.

This repository is deliberately independent of ADRMATS. Start with `AGENTS.md`.

## Research flow

1. Lock common engineering bottlenecks and ten pollutant candidates in secondary effluent.
2. Audit BMDL by paired model-only versus BMDL-assisted evaluation.
3. Develop five concepts through isolated designer, attacker, and reviewer rounds.
4. Select one primary and one backup design.
5. Produce an experiment-ready Chinese report and evidence appendices.
