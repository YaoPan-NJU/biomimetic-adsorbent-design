# 交付包索引

本目录是潘尧审阅实验方案的入口。项目没有产生实验数据，所有容量、选择性、稳定性和成本数值均为待实验检验的门槛或设计变量。

## 核心交付

- `MAIN_REPORT_ZH.md`：中文主报告，主案与备选均已完成设计终审。
- `attachments/FINAL_SCORECARD.md`：候选漏斗、各轮评分和终选理由。
- `attachments/REVIEW_ISSUE_LEDGER.md`：攻击、复审、修订和剩余实验风险。
- `attachments/REVISION_LOG.md`：方案版本与重要改动。
- `attachments/USER_REVIEW_LOG.md`：潘尧已提出的范围决定和最终审阅状态。
- `attachments/EXPERIMENT_EXECUTION_INDEX.md`：实验执行文件和附件路径。

## 事实源

- 污染物筛选：`research/pollutants/SCREENING_REPORT.md`
- 文献证据：`research/evidence/evidence_ledger.csv`
- BMDL A/B 评价：`research/bmdl/BMDL_PAIRED_EVALUATION.md`
- 主方案：`rounds/round_3/D1A_FINAL.md`
- 备选方案：`rounds/round_3/D1B_FINAL.md`
- 全部攻击和裁决：`rounds/round_1/`、`rounds/round_2/`、`rounds/round_3/`

最终批准人为潘尧。在 `USER_REVIEW_LOG.md` 记录明确批准前，不向周佳奇交接实验。
