# 实验执行索引

## 主方案 D1-A

唯一执行文件为 `rounds/round_3/D1A_FINAL.md`。该文件冻结受体、锁开体、扰乱对照、支架配方、肽合成、固定化、可溶受体门禁、柱实验、再生、稳定性、成本和失败合同。

实验开始顺序必须遵守文件中的 Gate M0 → S1/L0/D1 → 固定化 → 二级出水与柱 → 再生循环。前一门失败时不继续消耗后续材料。

## 备选方案 D1-B

唯一执行文件为 `rounds/round_3/D1B_FINAL.md`。D1-B 只在 D1-A 的水相预组织、合格锁开体或温和再生门失败时启动；它不是与 D1-A 同时铺开的第二条主线。

启动前必须关闭蛋白身份、apo 分数、sortase 共价结点、最低床负载、成本和分析方法门。任何一项未闭合，不得表达或装柱。
