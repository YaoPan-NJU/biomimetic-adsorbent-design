# 自适应仿生吸附方案发现与实验释放计划

状态：现行执行计划。取代固定“污染物数—方案数—主方案—备选方案”漏斗，但不删除任何历史方案、评分或否决记录。

## 1. 目标

Codex 持续寻找能够同时满足以下三项要求的仿生选择性吸附材料：

1. 仿生故事完整、诚实且有直接证据；
2. 人工识别单元、材料结构、合成和因果实验具有足够科学性；
3. 能够在市政二级出水场景中进行台式工程验证。

不预设污染物数量、候选池数量、最终方案数量或主备标签。通过几套就向潘尧滚动交付几套；没有方案通过时如实报告，不以降门槛、补数量或继承历史高分制造“赢家”。

## 2. 三道不可补偿的硬门

### Gate A：仿生故事完整

必须同时具备：

- 在检索原型前已经冻结的二级出水工程问题和选择性障碍；
- 可观察、可核验的生物吸附、富集、隔离、屏障、分配或选择性传输过程，或者与其物理功能直接同构的天然过程；
- 直接一手证据说明生物特征确实参与该功能；
- 可转译到非蛋白人工材料的功能原理；
- 至少一个六字段硬对应：生物特征、来源证据、材料实现、预期功能、测量、因果对照；
- 清楚说明生物过程与目标污染物之间是直接证据、近似物证据还是设计推断。

优先级为：目标污染物的实际生物吸附/富集 > 同类化合物的实际过程 > 物理功能严格相似的天然过程。

污染物与毒性靶蛋白、酶或核糖体的结合结构只能作为几何或化学辅助证据，不能单独通过 Gate A。生物来源聚合物、形貌相似和泛化“口袋”类比也不能通过。

### Gate B：材料与实验科学性

必须同时具备：

- 确切的非蛋白材料实体；依赖离散识别时给出人工识别单元结构、分子量、质子化状态、构型和合成入口；
- 完整体、删除体、位置/距离错配体，以及匹配总电荷、疏水量、芳香面积和孔结构的普通材料对照；
- 水相作用、溶解/聚集、化学稳定性和分析可辨识性论证；
- 目标物与关键结构类似物、同系物、无机离子、DOM和表面活性物质的竞争设计；
- 能排除仅由 `logD`、`pKa`、总电荷、碳链长度、芳香面积或孔容造成的单调吸附；
- 核心材料组合与现有技术的逐项先例攻击；
- 每项核心主张对应成功阈值、失败阈值和终止条件。

如果人工识别单元仍只是官能团清单、材料只是一张概念图、关键作用只在有机溶剂成立，或者没有可制造的匹配对照，Gate B 失败。

选择性层级必须在方案冻结时写明。若工程目标是共同去除一类均具有环境意义的污染物，类内高度分辨不是硬门；材料可以对ROX、ERY、CLA和AZI共同富集，但必须证明这种类别去除在二级出水中优于对DOM、盐和无关微污染物的普通吸附，并用类外阳离子药物排除“只要带正电就吸附”的解释。只有当处理目标要求保留某些同系物，或核心论文主张是单分子识别时，才要求类内分辨。若材料只是广谱富集多类有机微污染物，则不得声称大环内酯类别识别。

### Gate C：工程验证可执行

必须同时具备：

- 台式合成或制备 SOP，包括比例、当量、浓度、温度、时间、气氛、洗涤、干燥、纯化、批次质控和安全处置；
- 按总干材料质量、床层体积和可达位点数进行质量账本，不隐藏惰性载体；
- 超纯水机制门和市政二级出水主验证；
- 痕量浓度下的自由浓度质量平衡，以及同系物、盐、DOM和表面活性物质竞争；
- 批量动力学、竞争选择性、连续床突破、再生、浸出和结构稳定性；
- 与 GAC、离子交换、成熟主客体材料或目标污染物直接先例的同条件基准；
- 失败后的明确处置，不允许通过提高投加量、降低盐度或更换措辞掩盖机制失败。

Gate C 评价的是“能否形成有解释力的工程实验”，不是要求纸面阶段已经取得实验性能。

## 3. 实验释放级别

### E0：概念阶段

生物线索或材料方向存在，但确切结构、对照或SOP不完整。E0不进入正式推荐清单。

### E1：机制验证就绪

最低正式交付级别。必须已经冻结：

- 确切人工识别单元、可溶模型或最小材料实体；
- 完整体、删除体、错配体和普通吸附对照；
- 最小规模合成/制备步骤与身份确认方法；
- 第一轮水相竞争实验、分析方法、样本量和预注册停止阈值；
- 化学安全、浸出和废物处置要求。

潘尧审阅通过后，可以直接开展第一阶段机制实验。E1通过不等于完整材料性能已被证实。

### E2：完整材料验证就绪

在E1基础上，进一步冻结最终材料SOP、批次质控、二级出水竞争、连续床、再生、浸出、稳定性和工程基准矩阵。潘尧审阅通过后，可直接进入完整台式材料验证。

任何推荐方案必须标明E1或E2，不再使用含糊的“实验可行”描述。

## 4. 自适应发现流程

### Phase 0：工程问题与污染物入场

- 以 `pollutant_kb` 为主要种子，不按文献条数或数据库覆盖率排序；
- 独立核验二级出水检出、环境/健康风险和实际选择性难题；
- 对每个入场污染物先冻结“目标—竞争物—传统材料缺口”问题陈述；
- 同时冻结选择性层级：单污染物、污染物类别或广谱有机微污染物富集；该层级决定竞争物和论文主张；
- 不限制入场数量，失败污染物保留否决理由。

### Phase 1：真实生物过程搜索

按功能检索而非按蛋白名称检索，重点覆盖：

- 活性污泥、生物膜、好氧颗粒污泥和厌氧污泥中的固液分配、快速吸附、解吸滞后和组分富集；
- 生物膜、EPS、细胞壁、膜界面和黏液屏障的污染物吸附或隔离；
- 生物体、组织界面和天然多相结构中的可逆富集或分配；
- 天然选择性传输、尺寸门控、头基定位、亲疏水分区和可逆释放；
- 植物表面、微生物表面、天然分泌基质和生物矿化界面的污染物捕获；
- 与目标污染物具有同一选择性障碍的天然底物处理过程。

蛋白复合物结构可以在真实生物过程已成立后用于细化机理，但不得成为检索起点和唯一故事来源。

污水生物处理文献先按证据层级筛选：总体去除率或污泥检出只产生搜索线索；水相—污泥质量平衡和灭活/抑制对照可支持吸附过程；EPS去除/回补、组分分级、酶处理、离子强度扰动或表面性质变化能够把作用定位到具体组分；只有可重复的因果扰动才可支持硬功能对应。必须排除生物降解、共代谢、挥发、沉淀、胶体截留、细胞摄取和分析损失。EPS提取造成的细胞破损、絮体解体和离子桥联改变必须单独审计。

每条生物线索记录直接性、因果强度、物种/体系、水相条件、目标适用边界和可转译特征。未能给出材料硬对应的线索不生成方案。

每套主方案优先使用一个主生物原型，最多增加一个承担不同必要功能的支持原型。增加原型不能代替证据补强，也不能仅为了提高仿生维度数量。

### Phase 2：非蛋白材料转译

每个通过Gate A的生物原型可以生成零个、一个或多个材料方案，但不同方案必须在人工识别单元或材料机制上实质不同。设计文件必须从确切实体开始，而不是从载体名称或官能团列表开始。

优先选择：

- 自支撑多孔聚合物或低惰性质量架构；
- 共价稳定的分级、梯度或两亲纳米域；
- 能制造位置删除、空间错配和等组成随机体的模块化单元；
- 在中性水、盐和DOM条件下仍有可检验作用的化学；
- 可以成珠、成颗粒、成膜或装柱的结构。

材料平台先做工程筛选：多孔碳应作为低成本平台或强制基准进入比较；多孔聚合物只有在共价定位、成珠或功能协同带来必要增益时升级；MIP、COF、PAF和MOF必须逐案通过原料、模板、溶剂、步数、产率、水稳定、浸出、成型和再生成本门，不按材料名称预设高低等级。

### Phase 3：证据与先例攻击

每套方案在写完整材料SOP前先接受一次独立攻击：

- 生物功能是否真实且必要；
- 污水生物过程线索是否已经从总体去除推进到具体组分或界面，并排除降解、摄取和沉淀等替代解释；
- 人工材料是否真的实现同一功能；
- 选择性是否可由普通吸附替代解释；
- 关键化学是否在水中成立；
- 核心材料组合是否已有直接先例；
- 惰性质量、合成步数、浸出、DOM污染和再生是否使工程主张失效。

出现无法关闭的critical或high问题即终止或降为工程对照，不分配固定修订轮数。

### Phase 4：实验就绪设计

通过前三阶段的方案进入E1包编制。E1经独立Reviewer确认无critical/high后立即滚动提交潘尧审阅，不等待凑齐一批。只有确有价值且纸面风险已关闭的方案继续扩展为E2包。

## 5. 候选状态与排序

候选只允许以下状态：

- `lead_only`：证据线索，尚未形成材料；
- `concept_valid`：Gate A通过，Gate B/C未完成；
- `E1_ready`：三道硬门通过，机制验证包完整；
- `E2_ready`：完整材料验证包通过；
- `control_only`：可作工程或机制基准，不承担仿生主张；
- `terminated`：存在不可关闭问题；
- `historical`：旧合同产物。

总分只用于对同一释放级别内的方案排序，不能抵消任何硬门失败。正式推荐清单只包含 `E1_ready` 和 `E2_ready`；近失方案和否决方案进入附件。

## 6. 每套正式候选的交付包

1. 一页决策摘要与释放级别；
2. 工程痛点、污染物、竞争障碍和环境证据；
3. 生物原型及证据等级；
4. “生物特征→人工材料实现→功能结果→表征→因果对照”矩阵；
5. 确切材料组成、空间结构、人工识别单元和选择性机制；
6. 完整体、删除体、错配体、随机体和工程基准；
7. 台式SOP、质量账本、理论上限、批次质控和安全事项；
8. 第一阶段机制实验或完整材料实验矩阵；
9. 成功、失败、终止和替代路线；
10. 直接先例攻击、证据台账和独立审查意见。

## 7. 当前重启入口

旧方案不享有优先晋级权，只作为证据和失败模式输入。首轮执行同时做两件事：

1. 对 `pollutant_kb` 中已通过环境门的有机/新污染物重新进行真实生物过程搜索；
2. 把 `PFAS-M1`、`ROX-BF1` 和 `S1-SYN` 作为压力测试线索，分别检验实际膜富集、实际生物膜吸附和B类机制启发能否满足新的Gate A。

`S1-SYN` 若只有FABP证据而没有可独立成立的实际生物吸附/富集过程，只能保留为机制探针或辅助设计，不进入正式推荐。`PFAS-M1` 若只得到链长单调分配，`ROX-BF1` 若只得到阳离子药物类别吸附，也不能进入E1。

## 8. 停止与滚动交付

- 单方案在任一硬门失败时立即归档，不为凑数量修补；
- 某污染物的高质量原型、材料转译和水相选择性路径均耗尽后，记录证据饱和并转向其他污染物；
- 第一套E1方案形成后立即交付潘尧审阅，不等待第二套；
- 后续每形成一套E1/E2方案即追加交付；
- 项目何时停止、是否设置主备标签，由潘尧根据已通过方案决定，而不是由预设数量决定。
