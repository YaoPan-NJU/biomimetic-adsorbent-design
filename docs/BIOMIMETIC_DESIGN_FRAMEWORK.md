# 仿生选择性吸附材料设计框架

状态：`main` 约束性框架，2026-07-18 自适应实验释放修订版。

本框架吸收 `worktree-framework-correction` 分支中关于工程立论、仿生强度分级、材料可制造性和术语的有效内容，同时服从潘尧随后确认的项目规则：至少一个硬功能对应；动态对应可选；蛋白、折叠肽和易失活生物组分不进入默认材料；工程难题须在原型检索前形成可审查问题陈述，不能根据找到的生物结构反向制造痛点。

## 1. 项目主线

设计必须保持以下顺序：

> 二级出水中的工程难题 → 目标污染物及竞争障碍 → 传统材料的能力缺口 → 生物原型或生物机制参考 → 可转译设计原理 → 人工材料特征 → 选择性机制 → 因果对照与失败门

工程难题与污染物选择可以迭代校正，但在正式检索生物原型前，必须冻结当前版本的问题陈述。禁止先发现一个有趣蛋白，再把它能够提供的功能包装成项目原本就存在的工程痛点。

### 1.1 发现策略

设计发现以尚未解决的工程问题为主干，例如DOM竞争、痕量浓度下的有效传质、温和再生、固定床寿命、成型和成本。生物原型检索是解决这些问题的方法，不是先找一个能结合污染物的蛋白再反推用途。具体污染物仍然必不可少，但它主要用于冻结环境边界、分子性质、竞争物和验证场景，不能作为“自然界应当存在专属原型”的前提。

每套主方案原则上只保留一个主生物原型；只有第二个原型承担不同、必要且可独立验证的功能时，才增加一个支持原型。更多原型不得用来弥补单个原型证据不足，也不得把论文写成松散的类比集合。

## 2. 生物证据的三种身份

### A. 功能仿生原型

生物体系在天然功能中解决了与本项目相似的工程问题，且有直接结构、扰动或机制证据。例如对长链两亲分子的疏水区容纳、极性头基定位、选择性通道传输或可逆再生。

这类原型可以支持硬功能对应，但仍只转译设计原理，不复制蛋白序列或折叠。

### B. 生物机制参考

生物体系能够结合目标污染物，但该结合可能是毒性靶点、偶然亲和或外源化合物作用，并非生物体系进化来处理污染物。例如 ERRγ–BPA、核糖体–大环内酯和 FABP4–PFOA 结构。

这类证据可以约束距离、极性环境、占位和竞争物选择，但不能自动称为“自然界的解决方案”，也不能仅凭一张复合物结构获得高仿生分。按照当前实验释放合同，B类证据单独存在时不能通过仿生故事硬门；必须另有实际生物吸附、富集、隔离、屏障、分配、选择性传输过程或直接同构的天然功能证据。

### C. 理性化学设计

方案仅依据配位、氢键、卤键、分子印迹、孔径筛分或静电作用设计，没有可审计的生物功能对应。此类路线可以作为工程基准、机制对照或独立理性设计论文方向，但不能包装成本项目的仿生主方案。

每个方案必须明确标注 A、B 或 C。不得通过寻找一个泛化“酶口袋”类比，把 C 类路线事后改写成 A 类。

### 2.1 污水生物处理过程作为原型发现入口

活性污泥、生物膜、好氧颗粒污泥、厌氧污泥和其他生物处理体系，本身就是与目标水质条件接近的生物多相材料。论文中出现的污染物固液分配、污泥富集、快速初始去除、解吸滞后或不同污泥体系间差异，可以作为寻找生物原型的重要入口。检索时优先追问：

- 污染物主要进入EPS、细胞表面、细胞内部、脂质区域还是无机絮体；
- EPS中的蛋白样、多糖样、腐殖质样和脂质样组分，哪一类与分配变化相关；
- 表面电荷、疏水微区、Ca/Mg桥联、孔隙结构、絮体尺度和扩散距离是否参与；
- 吸附是否可逆，是否存在快速表面分配和慢速内部扩散两个阶段；
- 同类污染物是否共同富集，类外化合物是否表现不同。

证据按四级使用：

| 级别 | 可接受证据 | 在设计中的身份 |
|---|---|---|
| `WW-0` | 仅有处理去除率、出水下降或污泥中检出 | 搜索线索，不能证明吸附，更不能直接作为仿生原型 |
| `WW-1` | 有水相—污泥质量平衡、短时动力学、灭活或抑制对照，能够区分吸附与生物降解 | 生物过程证据，可生成待验证机制假说 |
| `WW-2` | 通过EPS去除/回补、组分分级、酶处理、离子强度或表面性质扰动，把富集定位到具体生物组分或界面 | 候选功能仿生原型，可进入材料转译设计 |
| `WW-3` | 具体生物特征的扰动引起目标分配可重复改变，并排除孔量、总有机质、沉淀和分析损失等替代解释 | 可支持硬功能对应，但仍须在人工材料中重新做因果验证 |

必须区分吸附、生物降解、共代谢、挥发、光解、共沉淀、胶体截留、细胞摄取和取样分析损失。仅凭“活性污泥去除率高”不能认定EPS吸附；污泥中浓度升高也可能来自细胞摄取或无机相结合。EPS提取还可能破坏絮体、释放胞内物质或改变离子桥联，因此提取前后差异必须与质量回收、细胞破损指标和结构变化一起解释。

只有定位到可测量的生物功能后，才进行人工材料转译。例如：

- EPS中的水化负电—低极性微区协同，可转译为稳定的两亲多孔网络；
- 絮体外层快速捕获、内部慢扩散，可转译为入口与内部区域分工；
- Ca/Mg参与的可逆桥联，可启发可再生的多点作用，但必须检查真实水竞争和金属浸出；
- 颗粒污泥的分级孔道和抗剪切结构，可启发可装柱颗粒，而不能仅仿外观。

最终材料仍以非生物、耐污水、可放大吸附剂为默认。发现EPS承担作用，不等于直接把提取EPS加入材料；应优先转译其功能组织方式，并设置去除该特征、等组成随机化和普通工程材料对照。

## 3. 硬功能对应

项目通过门要求至少一个硬功能对应，字段为：

- 生物特征及证据；
- 可转译设计原理；
- 人工材料实现；
- 预期选择性功能；
- 可直接测量指标；
- 删除、错配或锁定因果对照。

第二个独立硬对应可以增加论文价值，但不是强制门槛。动态行为仅在确实解决传质、选择性或再生问题且具有独立测量和因果对照时计分；不得为了凑“多维仿生”重新引入蛋白、肽链或通用刺激响应单元。

## 4. 从生物原理到人工材料

允许转译：

- 亲疏水区域的空间分工；
- 极性锚点与非极性腔/通道的协同；
- 分级孔道、入口和内部区域的功能分工；
- 经直接证据支持的可逆占位或动态切换；
- 多组分协同所需的相对位置和功能关系。

默认禁止：

- 蛋白质、表达构建体、折叠肽或蛋白衍生生物杂化材料；
- 复制氨基酸序列或完整三维折叠；
- 把埃级坐标等同于宏观材料中已经实现的位点几何；
- 把生物来源材料本身当作仿生硬对应；
- 用容量提升替代选择性和因果证明。

人工材料可以是自支撑多孔聚合物、COF、稳定MOF、工程多孔碳或其他可制造架构。载体不预选，所有容量按总干材料质量和床层体积报告。

### 4.1 材料体系筛选顺序

材料类别不是固定排名，但必须先比较工程成熟度，再讨论结构新颖性：

1. **工程多孔碳优先进入基准或平台筛选。** GAC、PAC和可成粒生物炭具有成本、水稳定性、固定床和再生先例；当简单表面处理或孔结构调整足以实现所需功能时，不应先上复杂框架材料。
2. **多孔聚合物用于需要共价固定、成珠或同一网络内功能协同时。** 其额外价值必须通过匹配的商业树脂、物理混合物和GAC对照证明。
3. **MIP、COF、PAF或MOF逐案准入。** 不预设MIP天然高选择、COF成本适中，也不因名称一概淘汰MOF；必须核算单体/模板/溶剂、合成步数、产率、水稳定、浸出、成型和再生。高成本、水不稳定或只能以纳米粉末使用的体系默认淘汰。
4. **最终成本按处理功能比较。** 同时报每千克成品材料成本和每立方米达标处理水的材料、再生及废液处置成本；任意的单价阈值不能替代明确工程基准。

### 4.1.1 离子交换材料边界

本项目的主材料是仿生吸附材料，不是阴离子交换、阳离子交换或混合模式离子交换树脂。离子交换材料只作为工程基准和机制对照。判断依据是主要机制而非名称：若容量主要由可达IEC、Donnan分配、对离子释放或盐置换解释，即使材料具有核壳、梯度、长烷基或空间分区，也按离子交换主路线淘汰。

带电位点可以辅助一个有独立生物依据的仿生功能，但必须在总电荷匹配的删除/错配对照中证明空间组织或协同识别的独立贡献，并用对离子质量平衡和盐依赖排除主要离子交换机制。完整判据见`docs/MATERIAL_SCOPE_BOUNDARY_ION_EXCHANGE.md`。

### 4.2 局部几何约束的正确用法

不复制蛋白埃级结构，并不等于人工材料不能形成可重复的局部几何。刚性共价拓扑、环化、预组织骨架或定义明确的配位结构可以产生局部距离和取向约束，但“反应具有常见几何”不能直接证明材料已经获得选择性人工识别单元。

任何硼酸/二醇、金属/配体、氢键阵列或其他局部约束在进入方案前，必须同时回答：

- 相关pH和二级出水中实际存在的化学形态是什么；
- 水、盐、DOM和竞争配体是否破坏或取代该作用；
- 约束是否位于污染物可到达的位置，而非只存在于干态平均结构中；
- 删除、位置错配和等组成随机对照能否区分几何贡献与总电荷、疏水性和孔结构；
- 金属或小分子组分是否浸出，循环后局部结构是否保留；
- 哪种直接表征证明了局部关系，不能只引用理想键长或配位数。

未通过这些问题时，只能写“候选局部化学约束”，不能写成成熟、可靠或目标专属的空间识别。

## 5. 人工识别单元门

遵守 `docs/TERMINOLOGY.md`。人工小分子、主体腔、官能团组合和孔口结构统一称为人工识别单元。

当选择性主张依赖一个离散化学结构时，必须先完成：

1. 确切结构、分子量、质子化状态和合成入口；
2. 完整体、删除体、位置或距离错配体；
3. 相关水相中的正交结合或分配测量；
4. 目标与结构类似物、共存离子和DOM的竞争；
5. 排除聚集、沉淀、普通疏水性和总电荷解释；
6. 通过后才增加聚合手柄或固定化路线。

孔道型材料若无法先做离散模型，也必须以等孔径化学随机体、位点删除体和位置错配体形成同等强度的因果对照。

## 6. 选择性与环境门

环境意义和二级出水相关性是评分前硬门。未通过时不进入材料评分。

选择性首先服务于处理目标，而不是追求越窄越好。若同一类别中的多种污染物都具有明确环境或健康风险，并且工程上本就希望共同去除，可以把该类别作为目标集合。此时不要求材料在同系物之间分出高低，类内分辨只作为附加结果；真正的硬门是目标类别相对二级出水背景物和无关污染物仍有优先富集或更慢突破。只有当某个同系物需要保留、其他同系物不应去除，或论文核心机制明确声称单分子识别时，才把类内选择性升为硬门。

每套方案在实验设计前必须明确选择性层级：

1. `single_compound`：确有单污染物区分需求，必须证明对关键同系物的分辨；
2. `pollutant_class`：同类高风险污染物共同去除，类内分辨不作硬门；
3. `broad_micropollutant_enrichment`：面向多类有机微污染物的广谱去除，不能再声称类别识别，只能评价相对DOM、盐和常规基质的有效处理。

选择性层级一旦冻结，生物故事、材料机制、竞争物、成功阈值和论文措辞必须使用同一层级，不能用单体机理证据支持类别主张，也不能用类别共同吸附结果包装成单体特异识别。

选择性必须在真实竞争条件中成立，至少包含：

- 同类污染物或位置/链长类似物；
- 二级出水主要无机离子；
- DOM与典型表面活性物质；
- 痕量目标浓度及自由浓度质量平衡；
- 超纯水机制对照与真实二级出水验证。

选择性只随 `logD`、`pKa`、总电荷、芳香面积、孔容或碳链长度单调变化时，不能归因于仿生人工识别单元。

## 7. 方案审查模板

每套方案进入深设计前必须回答：

1. 工程难题是否在原型检索前被独立提出并有实际水处理意义？
2. 生物证据属于功能仿生原型、生物机制参考还是理性化学设计？
3. 至少一个硬功能对应是否六字段齐备？
4. 材料是否为非蛋白、可制造且质量效率可核算？
5. 哪个实验能够区分声称机制与普通吸附解释？
6. 哪项失败会立即终止，而不是更换措辞或骨架挽救？
7. 现有技术是否已经实现核心材料组合？
8. 是否在进入完整SOP前核查了同功能材料家族，而不只是完全相同的化合物名称？
9. 若原型来自污水生物处理，是否已经区分吸附、降解、摄取、沉淀和分析损失，并把作用定位到可扰动的生物组分或界面？

## 7.1 自适应方案数量与实验释放

项目不再预设污染物数量、方案库规模、最终方案数量或主备标签。每个方案独立通过仿生故事、材料与实验科学性、工程验证三道不可补偿硬门；通过一套即交付一套，不等待凑齐固定批次。

正式推荐的最低级别为 `E1`：确切最小材料实体、完整体/删除体/错配体/普通材料对照、最小合成步骤、分析方法和预注册停止阈值均已冻结，可以直接开展第一阶段机制验证。`E2` 进一步要求完整材料SOP、批次质控、二级出水、连续床、再生、浸出和稳定性矩阵。具体规则见 `docs/ADAPTIVE_EXPERIMENT_READY_RESEARCH_PLAN.md`。

## 8. 三个当前案例的正确边界

### PFOA–FABP4

FABP4天然识别脂肪酸，可为“极性头基定位＋疏水链容纳＋空间预组织”提供功能类比；PFOA复合物属于直接生物机制参考。晶体双占位不能自动证明痕量水中双位点工作，也不能直接导出一个非肽人工识别单元。确切人工结构未冻结前不得继承高分。

### BPA–ERRγ

ERRγ–BPA是毒理学结合参考，不是自然界处理BPA的方案。结构和构效证据当前只支持一处主极性锚加疏水芳香环境，不支持把两个酚羟基同时锚定写成必要功能。任何BPA材料必须遵守后续非对称修正。

### ROX–核糖体与肟醚化学

核糖体是ROX的生物靶标，但没有ROX专属肟醚接触几何；不能声称核糖体进化来识别ROX。针对ROX 9-O-甲肟醚的氢键或铜配位属于值得检验的理性化学设计线索，除非另有真实生物功能对应，否则只能作为C类路线或机制对照。

## 9. 与分支框架的差异

`worktree-framework-correction` 分支提出的以下内容不进入 `main` 约束：

- 强制至少两个硬对应；
- 强制结构加动态多维组合；
- 要求生物原型必须进化来解决同一污染问题；
- 先由生物能力倒推工程痛点；
- 未经核验的BPA降解酶双端夹持和核糖体进化识别ROX示例。

这些要求分别与当前合同、动态可选规则、外源污染物事实、问题冻结纪律和证据真实性规则冲突。
