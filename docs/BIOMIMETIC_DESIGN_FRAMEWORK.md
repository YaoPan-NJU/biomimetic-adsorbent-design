# 仿生吸附材料设计框架（2026-07-18 修正版）

> 本文档为项目子 Agent 提供统一的仿生设计框架参考。所有设计方案必须遵循本框架。

---

## 一、核心立论：为什么要仿生

仿生的独特价值必须落在**传统合成方法做不到的事情**上，至少是其中之一：

- **结构层级性**：多尺度有序孔道（如木质部导管层级）
- **动态响应/可逆切换**：环境刺激下构象变化
- **多组分协同自组装**：像蛋白质折叠那样精确组装

**判据**：如果讲不清"仿生带来传统方法做不到的能力"，仿生就只是包装，审稿人一眼看穿。

## 二、三层方法架构

| 层次 | 问题 | 思路 | 优先级 |
|------|------|------|--------|
| Why（立论层） | 为什么需要仿生 | **思路二**：解决工程共性难题 | **主干** |
| How（方法层） | 如何仿生 | **思路一**：机制匹配寻找生物原型 | **方法** |
| Case（验证层） | 针对具体污染物 | 思路三：针对特定污染物找特定原型 | 可选，倾向放弃 |

### 思路二（自上而下）— 主干路线

**逻辑链**：
1. 设定要解决的**共性工程难题**（如孔道问题、亲疏水性、阵列结构）
2. 传统方法解决不了
3. 生物界进化已解决
4. 借鉴生物的**设计原理**

**优势**：逻辑链最硬、切入点多、可成系列工作。

### 思路一（自下而上）— 方法落地

**流程**：
1. 分析污染物性质
2. 匹配它如何被吸附去除（如氨基对羧基、静电吸引等**基团匹配**）
3. 通过这些**化学/物理机制**，去自然界寻找具备**相同机制**的生物原型
4. 研究生物原型**如何实现该机制**
5. 模仿其**基团或结构**，转化为设计思路

**关键**：必须走到**结构/动态层面**，不能停在官能团。

### 思路三（特定污染物）— 明确降级

> "针对某种特定污染物，寻找能特定吸附它的生物原型进行仿生设计。逻辑牵强，'因为难去除，所以自然界有现成方案'这一步不成立。倾向放弃，或仅作个案补充。"
> —— `docs/source/仿生吸附材料设计任务总纲.md`

**禁止使用思路三作为主干路线。**

## 三、多维度对应判据

### 四个维度

| 层次 | 维度 | 对应的痛点 |
|------|------|-----------|
| **静态结构** | 分级孔道、亲疏水 | "污染物为什么难去除"（本征吸附能力不足） |
| **动态行为** | 刺激响应、再生 | "为什么难以设计吸附剂"（工程瓶颈） |

### 硬对应与软对应

- **硬对应**：论文必须证明，建议 **至少 2 个**
- **软对应**：顺带提，可不深入
- **原则**：把硬对应做实比四个都浅做更值

### 仿生保真度要求

- 不是单维度仿生（只抄结构或只抄机制）
- 而是**多维度协同仿生**（结构 + 机制 + 动态行为）
- 明确列出仿了哪几个维度、每个维度达到什么保真度
- 这本身就是论文的 contribution

## 四、痛点-仿生因果链

### 正确的逻辑链

> 污染物难去除的根因 → 该根因恰是传统合成解决不了的 → 某生物原型在进化中恰好解决了该根因 → 故模仿其对应维度

### 关键原则

1. **痛点不是写作技巧**，而是从生物原型和仿生能力**倒推**出来的逻辑锚点
2. 顺序是：
   - 先定能仿什么生物原型（哪几个维度能真仿到）
   - 再倒推痛点清单
   - 再写引言
3. **对应不是并列清单**，而是一条贯通的因果链

### 错误示例

❌ **错误**：先有方案"模仿 FABP4 蛋白"，再凑痛点"PFOA 难去除"
✅ **正确**：先明确"PFOA 难去除的根因是缺乏对全氟链的选择性识别机制"，再找"自然界如何解决长链疏水分子识别"，提取设计原理"疏水腔 + 极性锚定 + 空间预组织"

## 五、材料复杂度约束

### 不要求

- 复制蛋白质的精确三维结构
- 要求 Å 级精确空间控制
- 引入蛋白质、折叠肽或生物构建体（除非潘瑶明确重开 scope）

### 要求

- 提取**设计原理**（如"疏水腔 + 极性锚定"）
- 用**可合成的材料**实现（COF、超交联聚合物、MOF 等）
- 允许结构柔性，不要求原子级精确复制

### 示例

**不要求**：复制 FABP4 的精确氨基酸序列和折叠结构
**要求**：在 COF 中实现"疏水环境 + 定向极性位点"的**空间排布原理**

## 六、术语标准

### 必须使用

- **人工识别单元**（artificial recognition unit）：指人工合成的分子识别结构
- **设计原理**（design principle）：从生物原型提取的可转译机制
- **硬对应**（hard correspondence）：必须实验证明的仿生维度
- **软对应**（soft correspondence）：可选的仿生维度

### 禁止使用

- **受体**（receptor）：容易与生物蛋白受体混淆，改用"人工识别单元"
- **结合口袋**（binding pocket）：过于蛋白质化，改用"识别位点"或"识别区域"

## 七、评估检查清单

每个设计方案必须回答以下问题：

### 1. 工程难题（Why）

- [ ] 解决了什么传统合成方法解决不了的工程难题？
- [ ] 这个难题是否明确、具体、可验证？
- [ ] 传统方法为什么解决不了？（不是"做得不好"，而是"原理上做不到"）

### 2. 生物原型（How）

- [ ] 生物原型是否真实存在？
- [ ] 生物原型是否进化出来解决类似问题？（不是碰巧结合）
- [ ] 是否有直接证据（晶体结构、突变研究、机理研究）？
- [ ] 是否属于"自然界的解决方案"，而非"碰巧能结合"？

### 3. 多维度对应（What）

- [ ] 仿生了几个维度？（至少 2 个硬对应）
- [ ] 每个维度的仿生保真度如何？
- [ ] 是否明确列出了硬对应和软对应？

### 4. 设计原理转译（Translation）

- [ ] 是否提取了可转译的设计原理？
- [ ] 是否避免了直接复制蛋白质结构？
- [ ] 材料复杂度是否可控？（不要求 Å 级精确）

### 5. 痛点-仿生因果链（Logic）

- [ ] 痛点是否从仿生能力倒推出来？
- [ ] 因果链是否贯通（不是并列清单）？
- [ ] 是否能写成："污染物难去除的根因 → 传统合成解决不了 → 生物原型进化解决了该根因 → 故模仿其对应维度"？

### 6. 诚实标注（Honesty）

- [ ] 是否明确标注了仿生强度？（真仿生 / 仿生启发 / 理性设计）
- [ ] 是否承认了设计的不确定性？
- [ ] 是否区分了已验证事实和设计假说？

## 八、正确 vs 错误示例

### 示例 1：PFOA 选择性吸附

❌ **错误（思路三）**：
> "FABP4 蛋白能结合 PFOA（PDB 9MIW），所以我们模仿 FABP4 设计吸附材料。"
>
> **问题**：FABP4 进化用于结合脂肪酸，不是为解决 PFOA 污染。这是"碰巧能结合"，不是"自然界的解决方案"。

✅ **正确（思路二 + 思路一）**：
> "PFOA 难去除的根因是缺乏对全氟链的选择性识别机制，传统疏水作用无法区分 PFOA 与其他 PFAS。自然界中，脂肪酸转运系统（如 FABP 家族）进化出了对长链疏水分子 + 羧酸头基的选择性识别，采用'疏水腔 + 极性锚定 + 空间预组织'设计原理。我们提取该原理，在 COF 中实现类似的空间排布，解决全氟链选择性识别的工程难题。"
>
> **优势**：明确了工程难题、传统方法的局限、生物原型的进化目的、以及设计原理的转译。

### 示例 2：BPA 选择性吸附

❌ **错误（思路三）**：
> "ERRγ 受体能结合 BPA（PDB 2E2R），所以我们模仿 ERRγ 设计吸附材料。"
>
> **问题**：ERRγ 结合 BPA 是内分泌干扰（毒性机制），不是解决方案。ERRγ 进化用于结合天然激素。

✅ **正确（思路二 + 思路一）**：
> "BPA 难去除的根因是其两个酚羟基的几何排列导致常规吸附剂无法区分 BPA 与类似物（BPF、BPS）。自然界中，某些细菌进化出了 BPA 降解酶（如细胞色素 P450），其底物结合口袋采用'刚性芳香夹持 + 双端极性识别'设计原理。我们提取该原理，在超交联聚合物中实现类似的几何约束，解决双酚类选择性识别的工程难题。"
>
> **优势**：生物原型（降解酶）是自然界处理 BPA 的真实方案，不是毒性受体。

### 示例 3：ROX 选择性吸附

❌ **错误（思路三）**：
> "我们发现 ROX 的肟基可以与铜配位，所以设计铜 MOF 来吸附 ROX。"
>
> **问题**：这不是生物原型，是配位化学。ROX 的自然识别机制是结合细菌核糖体。

✅ **正确（思路二 + 思路一）**：
> "ROX 难去除的根因是其大环内酯结构与多种抗生素类似，常规吸附剂无法区分。自然界中，细菌 50S 核糖体的多肽出口隧道（NPET）进化出了对 14 元大环内酯的特异性识别，采用'尺寸匹配狭缝 + 多点氢键网络'设计原理。我们提取该原理，在分子印迹聚合物中实现类似的尺寸和氢键约束，解决大环内酯选择性识别的工程难题。"
>
> **优势**：核糖体是 ROX 的天然靶标，是真实的生物识别机制。

## 九、可选参考：生物富集启发式

文档 `research/evidence/ORGAN_ENRICHMENT_BIOMIMETIC_HEURISTIC.md` 提供了一条**可选的原型发现启发式**：

> 研究污染物在生物体内的富集机制（如 PFOA 在肝脏通过 FABP 结合富集），理解其分子机制，然后在污染物进入生物体之前拦截。

**定位**：
- **可选参考**，不是评分项
- 不改变主干路线
- 不强制所有方案使用

**使用原则**：
- 必须有直接结合证据（不能仅凭"在某器官检出"）
- 必须排除替代解释（血流、代谢、非特异分配等）
- 必须把机制抽象为可制造的设计原理

## 十、总结：仿生的判据

一个方案是**真正的仿生**，当且仅当：

1. **解决了传统合成方法解决不了的工程难题**
2. **找到了自然界进化出来解决类似问题的生物原型**（不是碰巧能结合的蛋白）
3. **提取了可转译的设计原理**（不是直接复制蛋白质）
4. **实现了至少 2 个硬对应维度**（多维度协同仿生）
5. **痛点从仿生能力倒推，形成贯通因果链**
6. **材料复杂度可控**（不要求 Å 级精确）
7. **诚实标注仿生强度**（真仿生 / 仿生启发 / 理性设计）

**不符合以上任一条，就是"包装成仿生"，审稿人一眼看穿。**
