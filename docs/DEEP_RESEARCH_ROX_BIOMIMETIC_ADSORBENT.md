# ROX仿生选择性吸附材料深度研究报告

日期：2026-07-18
状态：候选A证据收敛并完成首轮攻击；非印迹和非生物膜路线已重新开放
候选A：`ROX-BF1 / ROX-BioMIP-G`（concept_valid，不是既定主方案）

## 执行摘要

本轮没有找到一个既有直接生物证据、又能在水中专一识别ROX的蛋白外天然位点。最强真实原型是污水生物膜/EPS：在1 μg/L条件下，它确实吸附ROX、红霉素和克拉霉素，且膜厚、孔隙、密度和扩散显著影响分配；但三种大环内酯在厚膜中的Kd几乎相同。这一结果同时给出了项目最有说服力的仿生起点和最清楚的天然缺口。

对人工识别化学的比较显示，Cu/Zn配位实验主要在甲醇中成立，ROX也不优于CLA/AZI；单一氢键侧袋会被水强烈竞争；未改性环糊精能包合ROX却没有近邻选择性证据。现有表面印迹虽然选择性有限，但它是唯一已有完整ROX材料制造先例、能直接进入水相验证的路线。

因此最合理的方案不是“找一个神奇官能团”，而是一个分工清楚的混合设计：

- 仿生模块：水化负电分级基质和受控薄层传质，来源于真实生物膜吸附；
- 人工选择模块：ROX-d7表面印迹空腔，完整读取ROX形状，重点检验C9醚肟侧链亚腔；
- 工程模块：多孔聚合物微球和超薄pSBMA表皮，面向二级出水抗DOM和填充床。

候选A的设计、SOP草案、对照矩阵和停止阈值见 `rounds/rox_bf1_deep_design_1/DESIGN.md`。隔离攻击发现其质量账本、抗污层锚定、空间对照和痕量分析尚未闭合，因此当前仅为`concept_valid`。同时继续检索细胞酸性区室离子陷获、膜分配/跨膜富集、真实选择性转运以及非印迹主客体路线；最终方案不预设为候选A。

替代原型和材料路线的同门比较已经单独记录在`docs/ROX_ALTERNATIVE_PROTOTYPE_AND_DESIGN_TOURNAMENT.md`。其中溶酶体离子陷获提供了更强的ROX直接生物故事，但单独微胶囊路线因AZI反例和梯度—传质矛盾被终止为主方案；ROX印迹选择通透层与酸性接收相串联被保留为高上限膜分离旁路。

## 研究问题

1. ROX是否通过环境意义硬门？
2. 是否存在真实、可转译且不是蛋白结合位点包装的生物吸附原型？
3. 哪种非蛋白人工识别化学最可能在中性水中区分ROX与ERY/CLA/AZI？
4. 如何把材料做成有合成路径、匹配对照、痕量分析和连续床验证的实体？

## 主要结论

### 环境意义：通过

ROX在污水处理出水中持续检出，部分常规设施去除有限或出现负去除；区域调查还把ROX列入主要残留或生态风险贡献物。既有数据支持把0.1–1.0 μg/L作为机制和加标验证范围，以约0.2 μg/L作为接近真实二级出水的核心点。环境意义不是评分项，而是已经通过的入场硬门。

### 仿生故事：通过，但必须分层

真实生物膜的可核验功能是阳离子大环内酯的水化负电富集和扩散—分配耦合，而不是ROX专一识别。人工材料翻译的是功能同构：水化边界、内部负电吸附域、可达孔隙和薄层传质。ROX选择性由独立的人工印迹空腔补足。

这一分层比核糖体或蛋白口袋故事更诚实：核糖体结构并不显示ROX侧链提供强专属接触，而且天然核糖体对克拉霉素的结合可以强于ROX；它只能作为结构边界证据，不能承担仿生原型。

### 人工识别路线：印迹胜出，但不是已经证明成功

| 路线 | 水相ROX直接证据 | 近邻选择性 | 工程风险 | 裁决 |
|---|---:|---:|---:|---|
| Cu/Zn配位 | 弱；正式比较在甲醇 | ROX不优于CLA/AZI | 竞争、浸出 | 排除主方案 |
| 单一氢键侧袋 | 无 | 未知 | 水合竞争强 | 只作高风险探针 |
| β-CD/HP-β-CD | 有包合证据 | 无 | 可能只提高类别分配 | 不作ROX识别单元 |
| ROX表面印迹 | 有多项材料先例 | 已报道但有限 | 模板泄漏、DOM、重复先例 | 作为人工选择模块 |

### 设计创新点不应写成“又一个ROX-MIP”

直接先例已经包括硅胶表面MIP和COF表面MIP。拟投稿创新必须落在以下可检验组合，而非名词组合：

1. 真实生物膜功能启发的外部水化边界—内部负电识别域空间顺序；
2. 以ROX-d7使模板来源信号可区分、可审计，同时单独核查d0杂质、回交换和MRM串扰；同位素模板不能自动消除泄漏假阳性；
3. 用ERY-d3模板空腔作为同源C9侧链删除体，而不是只做NIP；
4. 用等组成随机体和空间反转体证明分级结构必要性；
5. 在真实二级出水、同系物共存和连续床条件下同时报告选择系数与质量平衡。

## 决策

`ROX-BF1`首轮隔离攻击未通过E1，保留为候选A并关闭可修复问题。ROX方案池重新开放；只有在不同仿生原型和材料路线完成同门比较后，才决定哪个方案进入E1。
