## 2. 已吸收

### 2.1 工程难题为主干

`main`进一步明确：先冻结DOM竞争、痕量传质、温和再生、固定床寿命、成型和成本等未解决问题；再用真实生物过程寻找解决原理。具体污染物用于环境与实验边界，不能用来假设自然界必然存在专属原型。

### 2.2 生物原型数量收敛

每套方案优先一个主原型，最多增加一个承担不同必要功能的支持原型。该规则防止用多个弱类比拼接故事，但不恢复“必须两个硬对应”或“必须动态仿生”的旧要求。

### 2.3 成本、水稳定和放大优先

工程多孔碳优先作为低成本平台或强制基准。多孔聚合物只有在共价定位、成珠或同一网络协同确有必要时升级。任何高级框架材料都必须核算原料、模板、溶剂、步骤、产率、水稳定、浸出、成型和再生。

### 2.4 局部几何约束可以设计，但必须验证

刚性共价拓扑、环化、预组织骨架或定义明确的配位结构可以形成局部约束。`main`吸收这一点，但增加水相形态、DOM/盐竞争、湿态可达性、错配对照、浸出和直接结构测量六道门，防止把理想键长直接写成材料选择性。

### 2.5 现有技术检索扩大到材料家族

先例攻击不能只找完全同名材料，还要检查相同功能家族是否已经实现核心组合。例如混合模式树脂、膜界面模拟、环糊精网络、MIP和胍基聚合物可能已经覆盖大部分声称的新意。

### 2.6 选择性围绕需要共同去除的类别定义

GLM提交说明中的“对DOM/离子有选择，而不一定区分其他PFAS”被提升为通用规则：若同一类别中的污染物均有环境意义并需要共同去除，可以把整个类别作为目标集合，不必执着于同系物之间的选择性。实验硬门改为目标类别相对DOM、盐和无关污染物的有效富集，并用适当类外竞争物排除普通电荷或疏水吸附。

该规则不等于所有方案都可以宽泛声称选择性。每套方案必须先冻结以下一种层级：单污染物、污染物类别、广谱有机微污染物。单污染物主张才要求类内分辨；类别主张允许共同去除；广谱材料不得再包装成类别特异识别。当前MAC-BI1明确属于`pollutant_class`。

## 3. 未直接吸收或已改写

### 3.1 GLM的`PROJECT_STATE.yaml`

分支记录的PFOA重设计、62/100和下一动作与`main`当前MAC-BI1大环内酯路线冲突，全部不合并。`main`仍是唯一事实源。

### 3.2 活性炭固定为所有方案主材料

不接受材料类别一刀切。活性炭具有最强工程成熟度，应优先做平台或基准；但如果选择性机制需要共价定位、同一网络协同或可制造错配对照，多孔聚合物可能更合适。当前MAC-BI1因此不会被活性炭优先规则自动否决。

### 3.3 COF、MIP和MOF的类别结论

不接受“COF成本中等”“MIP选择性高且制备简单”或“所有MOF成本高、水稳定性差”的概括。具体配方差异太大，只能逐案评价。高成本、水不稳定、粉体难回收或依赖大量有机溶剂的方案默认淘汰。

### 3.4 三种精确识别化学的无条件可靠性

- 硼酸/二醇结合高度依赖硼酸结构、二醇构型和pH，不能由五元环示意直接推出中性水选择性；
- Zn或Cu羧酸配位会同时面对无机离子、DOM、其他羧酸和金属浸出，不能据此声称PFOA头基专属；
- 氢键阵列在水中受到强烈竞争，理想长度和角度不等于材料中已经实现稳定识别。

这些内容被改写为“候选局部化学约束＋六项验证门”。

### 3.5 固定`<5元/g`成本线

该数值只出现在提交说明中，没有成本口径、原料等级或处理寿命依据，因此不作为硬门。项目改为同时比较成品材料每千克成本和每立方米达标处理水的材料、再生与废液处置成本。

### 3.6 放弃特定污染物路线

不把具体污染物从项目中删除。正确改写是：具体污染物不能成为寻找专属生物结合位点的逻辑前提，但仍必须用于环境意义、竞争物、分析方法和实验验证。

### 3.7 分支中的旧仿生硬门

不吸收“至少两个硬对应”“必须同时有结构和动态仿生”“先选原型再倒推工程痛点”。`main`继续要求至少一个落实到人工材料、可测量且有因果对照的硬功能对应；动态只在真正解决选择性、传质或再生问题时增加；工程问题必须先冻结。这样保留完整论文故事，同时避免为了凑仿生维度引入蛋白、肽链或无关刺激响应。

## 4. 对当前MAC-BI1的实际影响

- GAC继续作为必须比较的工程基准，并保留为低成本备选平台；
- MAn–St–DVB微球只有在商业材料预筛显示基础化学值得继续、且一体化网络优于物理混合物时才合成；
- 生物膜作为主原型，阴离子磷脂界面只承担不同的分子作用解释，不再继续增加第三个原型；
- 羧酸与芳香结构共存仍是候选功能组合，不能在湿态结构和交互实验前写成精确空间识别。

本次整合不改变`MAC-BI1`的`E1_not_ready`状态和唯一下一动作。
