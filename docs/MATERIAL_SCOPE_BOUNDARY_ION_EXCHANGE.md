# 仿生吸附材料与离子交换材料的项目边界

日期：2026-07-19。该边界来自潘尧的明确课题范围决定，对后续所有污染物和材料方案生效。

## 1. 核心决定

本项目研究仿生吸附材料，不以阴离子交换树脂、阳离子交换树脂或混合模式离子交换树脂作为主材料。离子交换材料可以作为工程基准和机制对照，但不能进入正式推荐方案，也不能通过核壳、梯度、孔道或“生物启发”措辞重新包装成仿生主方案。

这一边界不仅按材料名称判断，也按主要去除机制判断。材料即使不叫“树脂”，只要目标污染物的主要容量来自固定电荷位点与对离子的化学计量交换，就属于本项目排除的主路线。

## 2. 判定为离子交换主路线的信号

出现以下任一核心特征，应默认按离子交换路线审查：

- 永久季铵、质子化多胺、磺酸、羧酸等固定电荷位点构成主要容量来源；
- 吸附量或柱容量主要随可达离子交换容量IEC变化；
- 目标阴离子或阳离子通过释放Cl⁻、OH⁻、Na⁺、H⁺等对离子完成近化学计量交换；
- 高浓度盐是主要解吸手段，盐浓度与竞争离子决定主要性能；
- 材料的主体、官能团与工作方式可归入成熟强碱、弱碱、强酸、弱酸或混合模式交换介质；
- 删除所谓仿生空间结构后，只要IEC和孔结构相同，主要吸附性能仍被保留。

核壳、径向梯度、长烷基季铵、强弱碱分区或疏水PS-DVB骨架不能自动改变上述分类。

## 3. 允许带电位点的条件

带电作用并非一律禁止。电荷可以作为仿生人工识别单元中的一个辅助作用，但必须同时满足：

1. 主要论文主张来自有直接生物依据的空间组织、界面过程、可逆构象/装配、协同多点识别或其他非离子交换功能；
2. 目标捕获不能由总IEC、简单Donnan分配或对离子释放解释；
3. 删除、位置错配或结构扰乱对照在总电荷基本匹配时显著破坏目标功能；
4. 用对离子质量平衡、盐依赖、同电荷非目标物和商业离子交换材料对照，证明主要机制不是化学计量离子交换；
5. 再生策略不以高盐置换作为唯一或主要功能来源。

无法同时满足时，该材料只能作为离子交换或普通静电吸附对照。

## 4. PFHxS方向的直接后果

`PFH-1`的弱碱外层和永久季铵内核均以离子交换为主要捕获基础。径向分区改变的是交换位点的空间排列，没有改变材料的根本类别。因此PFH-1因课题范围冲突终止，不再开展M0制造验证。

`PFH-2`和`PFH-3`已因各自科学问题终止。长烷基季铵PS-DVB同系列继续只作离子交换/疏水混合模式对照。

PFHxS重新设计必须回到真实生物吸附、界面富集、隔离、运输或排斥过程，寻找能够转译为非离子交换主机制的人工材料。商业PFAS阴离子交换树脂保留为强制工程基准，以证明新材料与隔壁课题组方向的实质区别。

该规则也追溯适用于ROX方案。`MAC-BI1`和`MAC-IF1`以羧酸/羧酸盐弱阳离子交换与疏水分配为主要材料基础，不再作为ROX主候选；它们只保留为混合模式对照，相关B0/M0不再执行。

## 5. 后续方案的准入问题

每个新方案必须先回答：

1. 若把材料交给离子交换领域研究者，他们是否会自然地把它归类为离子交换或混合模式树脂？
2. 主要容量是否可以由可达IEC和对离子交换质量平衡预测？
3. 仿生结构被删除而总电荷保留时，功能是否显著消失？
4. 与商业离子交换材料相比，新材料新增的生物启发功能是什么，如何直接测量？
5. Introduction能否不依赖“改进离子交换树脂”的叙述而独立成立？

前两题任一为“是”，或后三题不能给出可证伪答案，方案不得进入E1。
