# ROX主线修订：面向大环内酯类的低成本仿生吸附颗粒

日期：2026-07-18

状态：`concept_valid / commercial_prescreen_protocol_not_frozen / E1_not_ready`。隔离攻击发现3项critical和5项high问题；本文件给出修订后的研究方向，不是已经通过审查的E1实验SOP。

## 0. 完整方案叙述

本方案面向市政污水二沉池出水中ROX、ERY、CLA和AZI等大环内酯抗生素的共同去除。目标不是把四种结构相近的大环内酯彼此分开，而是让固体吸附剂在DOM、无机盐和其他有机微污染物大量共存时，仍对大环内酯这一类分子表现出优先富集。ROX是代表性目标物和论文分析重点，类内高度选择性属于额外结果，不是方案成败的前提。

仿生起点来自两个相互补充的真实过程。第一，污水生物膜能够在低浓度水相中共同吸附ERY、CLA和ROX，生物膜厚度和可达孔隙会明显改变分配，说明水化多孔基质能够富集这一类阳离子大分子。第二，ROX、ERY和AZI能够在不依赖蛋白的条件下直接进入阴离子磷脂界面，作用同时涉及带电区域和低极性区域。前者证明“大环内酯在真实生物多相材料中能够共同富集”，后者解释“负电区域和低极性区域为什么需要共同存在”。吞噬细胞酸性区室只作为类别富集的补充证据，不转译为需要维持内部低pH的密闭颗粒。

人工材料`MAC-BI1`拟采用马来酸酐、苯乙烯和少量DVB制成毫米级多孔微球，再把可接近的酸酐水解成羧酸/羧酸盐。苯乙烯来源的芳香结构提供低极性作用，水解马来酸酐提供负电位点，DVB维持颗粒与孔道。设计只主张这两类结构存在于同一交联网络，不预先声称它们严格交替、有序排列或已经复制磷脂膜。是否真正形成协同，必须通过同体系组成梯度、水解梯度和匹配对照测出来。

预期吸附过程分为三个可检验环节：大环内酯先进入可达孔道；芳香区域降低其从孔内返回水相的倾向；羧酸盐与质子化糖基进一步稳定占据。该解释面临一个直接反例：普萘洛尔、文拉法辛和西酞普兰同样是疏水阳离子，而且尺寸更小、扩散可能更快。如果材料对这些药物的增强与对大环内酯相同，就不能称为大环内酯类别优先，也不能把普通混合模式树脂包装成仿生识别。

合成路线原则上只有悬浮成球和稀碱水解两个主要反应，产物可以筛分、回收并装入固定床，不使用抗体、蛋白、多肽、二维材料、贵金属或同位素模板。已有文献证明相应交联多孔颗粒可以做到约0.35–1.0 mm，但原配方中的二氧六环、湿态强度、可接近酸位点和残余单体尚未解决，因此不能直接把旧配方当成放大SOP。

为降低失败成本，方案不从合成新材料开始。第一步先用商业大孔非离子树脂、弱酸树脂、二者物理混合物、商业混合模式树脂和指定GAC做等摩尔竞争预筛。如果这些现成材料所代表的“疏水孔道＋羧酸位点”基础化学对大环内酯没有可重复的类别偏好，MAC-BI1主线停止。如果出现类别偏好，才进入新微球合成，并进一步证明一体化交联网络是否比物理混合物带来额外效果。

论文成立需要同时满足三点：仿生原型与材料功能对应有直接证据；一体化颗粒的增益能够由因果对照归到两类区域协同，而不是孔容或总电荷；真实二沉池出水固定床在所需剂量、床寿命、DOM共吸附、再生或处理成本中至少一项优于明确型号的GAC和商业混合模式树脂。任一条件失败都应降低结论或终止路线，而不是依靠增加投加量或更换表述维持故事。

## 1. 用户边界与问题重述

ROX仍作为代表性目标物，但材料不必在ROX、ERY、CLA和AZI之间实现很高的类内分辨率。这四种大环内酯均具有环境意义，能够共同去除是合格结果。新的选择性要求是：在市政污水二沉池出水中，材料对大环内酯类的吸附应明显优于对DOM、无机盐和无关有机微污染物的普通吸附。

最终产品必须是可回收颗粒或可装柱的固体吸附剂。膜分离、酸性接收液、抗体、蛋白、多肽、昂贵二维材料、贵金属和复杂定制主体不进入当前方案。优先接受简单悬浮聚合、成球、水解、一次表面处理或马弗炉热处理。

## 2. 为什么放宽类内选择性会改善仿生故事

受控生物膜实验在1 μg/L下观察到ERY、CLA和ROX的显著吸附。500 μm生物膜的模型平衡分配系数约为CLA 10.90、ERY 10.68、ROX 10.80 L/g；相对50 μm生物膜，CLA和ERY约提高26–32倍，ROX也表现出明显厚度效应。天然系统呈现的是大环内酯类别富集，而不是ROX专一识别。

含磷脂酰肌醇的阴离子脂质体还会直接结合ERY、ROX和AZI。现有研究把这一作用解释为带正电的糖基与阴离子磷酸头基作用，同时分子的低极性部分进入膜内。该证据不证明污水吸附性能，但提供了更清楚的材料转译原理：负电区域与低极性区域需要在同一个可达界面共同工作。

吞噬细胞酸性区室仍可作为大环内酯类能够被生物体系富集的支持证据，但不再直接转译为密闭酸性微胶囊。开放式固体颗粒难以长期维持内部低pH；强行复制酸陷获会同时带来传质慢、难装柱和酸耗问题。

## 3. 当前首选材料：MAC-BI1多孔两亲性微球

### 3.1 材料实体

候选材料为马来酸酐、苯乙烯和少量DVB交联形成的毫米级多孔微球，成球后用稀碱把可接近的酸酐转成羧酸：

- 苯乙烯来源结构提供低极性芳香区域；
- 水解后的马来酸酐提供羧酸/羧酸盐位点；
- DVB维持颗粒和孔道的机械稳定性；
- 目标粒径为约0.35–1.0 mm，可直接筛分并评估装柱。

线性苯乙烯—马来酸酐共聚物具有形成交替序列的倾向，但不能把这一结论直接外推到DVB交联的非均相微球。现阶段只能写成“同一交联网络内同时存在芳香结构和羧酸位点”；局部序列、位点分布和可接近程度必须通过固态核磁、滴定和吸附对照验证，不能声称已经形成有序磷脂膜式排列。

### 3.2 为什么优先于高磺化树脂和改性活性炭

高磺化树脂主要依靠离子交换。二沉池出水中的Na、Ca、Mg和DOM远高于ng/L级大环内酯，容易掩盖或竞争纯静电作用。部分磺化PS-DVB工艺成熟，但如果不能证明负电与疏水区域的协同，它只是普通阳离子交换树脂。

中孔GAC拥有最强的真实二级出水和连续床工程证据，因此必须作为基准；但简单氧化GAC属于常规改性，单独承担仿生主张和论文新意的能力较弱。MAC-BI1的价值在于用一个可直接成球的交联网络检验“同一界面中的低极性区域和负电位点是否产生协同”，而不是仅提高酸基数量。

## 4. 合成可行性与尚未关闭的问题

1987年的直接成珠先例使用甘油连续相、羟乙基纤维素和NaCl稳定悬浮；马来酸酐、苯乙烯、DVB、过氧化苯甲酰及甲苯/二氧六环组成分散相，在约70 ℃、氮气、200 rpm下反应2 h，得到约0.35–1.0 mm多孔颗粒；随后以0.1 N NaOH、90 ℃、3 h检验酸酐水解。来源：https://doi.org/10.1002/app.1987.070340125

该路线只有“悬浮成球—稀碱水解”两个化学操作，原料没有生物试剂、二维材料或贵金属，原则上符合成本边界。但以下问题未关闭：

- 文献配方使用较多甘油、甲苯和二氧六环，放大前需要验证连续相回收及用更安全致孔剂替代二氧六环的可能性；
- 低DVB有利于酸酐进入网络，却可能降低湿态抗压强度；提高DVB又会降低酸酐含量；
- 文献约只有一半酸酐单元可被水或胺接近，不能按投料量计算有效羧酸位点；
- 颗粒尚无大环内酯吸附、真实二沉池出水、再生或长期固定床数据。

因此当前冻结的是小规模配方筛选和停止条件，不冻结公斤级SOP。

## 5. 先做商业材料化学预筛，再决定是否合成

MAC-BI1最可能的失败原因是羧酸和芳香孔道只能吸附一般疏水阳离子，对大环内酯类没有额外偏好。直接优化新微球会浪费合成资源。因此第一步应先用现成材料回答“这组简单化学是否值得做成一个新颗粒”：

| 组别 | 现成材料 | 回答的问题 |
|---|---|---|
| P1 | 商业大孔非离子树脂 | 单独的疏水孔道能做到多少 |
| P2 | 商业弱酸阳离子交换树脂 | 单独的羧酸位点能做到多少 |
| P3 | P1与P2按酸容量和质量配平的物理混合物 | 两种普通作用相加是否已有类别偏好 |
| P4 | 商业混合模式树脂 | 成熟的一体化疏水—离子交换材料能做到多少 |
| P5 | 明确型号的中孔GAC | 固定床工程基准 |

ROX、ERY、CLA和AZI与普萘洛尔、文拉法辛和西酞普兰按等摩尔竞争。若P3和P4均未表现出可重复的大环内酯类别偏好，则没有充分依据期待相同基础化学的MAC-BI1自然获得类别选择，主路线停止或退为普通工程树脂。

只有商业材料预筛通过后，才合成下列材料：

| 组别 | 材料 | 回答的问题 |
|---|---|---|
| T | 水解MAn–St–DVB一体微球 | 低极性区域与羧酸位点共同存在时的总效果 |
| C1 | 无MAn的St–DVB多孔微球 | 只有疏水孔道能做到多少 |
| C2 | 酸容量接近的商用弱酸树脂 | 只有酸性位点能做到多少 |
| C3 | C1与C2的物理混合物 | 两种普通材料相加是否已足够 |
| C4 | 商业中孔GAC | 当前工程基准 |
| C5 | 商业大孔非离子树脂或混合模式树脂 | 判断新材料是否只是已有树脂的重复 |

此外必须在同一MAn–St–DVB合成体系中设置马来酸酐投料梯度和部分水解梯度，尽量分别改变芳香结构比例与可接近羧酸量；同步测湿态孔容、溶胀、扩散、酸容量和粒径。只有双因素分析显示“酸位点量×疏水保留能力”存在正交互，且T在这些变量尽量匹配后仍优于C1、C2和C3，才能说明同一颗粒中的两类区域产生了协同。若T与C3相同，仿生结构主张失败，即使容量较高也只能作为普通混合模式树脂。

## 6. 第一轮吸附与机制门

目标组合为ROX、ERY、CLA和AZI。类外竞争物至少包括三种同为阳离子、但不是大环内酯的药物，例如普萘洛尔、文拉法辛和西酞普兰；另加入一种中性和一种阴离子微污染物，防止把总疏水性或总电荷误写成类别识别。

主实验使用同一批本地二沉池出水，并记录DOC、pH、电导率、Ca、Mg和主要无机离子。超纯水只作为机制对照。E1前还必须冻结两个浓度层级、材料剂量、液固比、接触时间、重复数、等摩尔竞争浓度、空白容器损失、过滤前后差异、LC-MS/MS定量限和基质回收率。

第一轮按三个连续门判断：

1. **化学可行门：**四种大环内酯均出现可重复吸附，完整进出质量回收为80–120%；同时报告单位材料吸附量和分配系数，不能只报告去除率。
2. **类别偏好门：**材料改性带来的增益对四种大环内酯显著高于对三种类外阳离子药物；双因素交互项和95%置信区间必须支持这一差异。
3. **工程价值门：**在至少三个日期的二沉池出水中，达到同一出水目标时，MAC-BI1至少在所需剂量、固定床寿命、再生恢复率、DOM共吸附量或处理成本中的一项显著优于指定GAC和商业混合模式树脂，其余指标没有不可接受退化。

若只在mg/L投料、低盐纯水或很高吸附剂剂量下成立，路线停止。痕量实验必须用LC-MS/MS做完整进出质量核算，不能只凭进水下降判断吸附。

## 7. 再生与连续床边界

第一轮只筛选低成本再生液：稀盐酸、NaCl以及不高于20%的乙醇组合。甲醇、乙腈和大量有机洗脱剂不作为工程默认方案。每轮同时计算解吸质量回收、下一轮吸附恢复、再生液体积、浓缩倍数、中和及清洗用水、残余苯乙烯和细颗粒浸出。五次再生后若性能低于初始值80%，或污染物/材料质量去向无法闭合，停止扩大。

通过第一轮后，颗粒以0.5–1.5 mm规格进入固定床，起始空床接触时间10–20 min。真实二沉池出水连续运行5000床体积、平均去除80%、相对GAC床寿命提高30%、单周期粉化损失低于0.5%和再生液体积低于处理水量1%，目前均作为远期工程目标；E1结果和指定GAC基准形成后再冻结正式E2通过值。

这些数值是预注册的继续/停止门，不是已经取得的材料性能。

## 8. 类别印迹为什么暂缓

泰乐菌素模板印迹材料已经能够同时富集十种大环内酯，证明“类别印迹”在化学上成立；ERY印迹材料也在5–100 μg/L范围报告过吸附和循环。问题是现有路线通常使用氯仿、乙腈、甲醇、DMSO和较高模板量，且缺少二级出水连续床与ng/L模板泄漏闭环。

类别印迹只在以下情况下重新进入：

1. MAC-BI1能够高效去除四种大环内酯，但同样强烈吸附类外阳离子药物；
2. 模板改为普通ERY或泰乐菌素，不使用ROX-d7作为规模制备模板；
3. 聚合与洗脱能够改成乙醇/水或低比例乙醇体系；
4. 稀酸/盐/乙醇能够再生，连续空白出水中模板低于LC-MS/MS定量限；
5. 印迹带来的类别偏好或床寿命增益足以补偿额外步骤和成本。

直接证据及边界见：https://doi.org/10.3390/molecules23051172、https://doi.org/10.1016/j.envpol.2023.122425、https://doi.org/10.1002/jctb.7884。

## 9. 成本与放大判据

- 第一阶段不使用模板、抗体、蛋白、多肽、二维材料或贵金属；
- 合成原则上不超过成球和水解两个主要反应；
- 小试颗粒收率目标至少80%，目标粒径收率和湿态强度必须单独报告；
- 放大前必须核算甘油和致孔剂回收，不得把溶剂费用与废液处理隐藏在“廉价单体”后面；
- 初步材料制备成本以不超过商业GAC两倍为门，最终按每立方米实际处理水的材料、再生和废液处置成本比较；
- 如果材料消耗高于10–20 mg/L PAC工程基准且没有更长寿命或更好的大环内酯去除，停止放大。

## 10. 当前裁决

`MAC-BI1`是本轮最值得继续检验的低成本仿生固体候选，但尚未达到E1。隔离攻击确认它仍可能只是普通混合模式树脂，当前最重要的动作不是立即合成，而是先用商业弱酸树脂、非离子大孔树脂、二者物理混合物、商业混合模式树脂和GAC完成等摩尔竞争预筛。预筛出现大环内酯类别偏好后，才值得冻结MAn–St–DVB小试配方。

中孔GAC是必须战胜的工程基准，也是失败时最可靠的备选底座。类别印迹是条件触发的第二阶段增强模块，不进入第一版材料。ROX-LYS-MIM膜路线因用户明确要求固体吸附剂而退出当前范围。

1. **图1：工程问题与仿生映射。** 二沉池出水中的大环内酯、DOM和盐背景；生物膜类别富集与阴离子磷脂界面；对应到多孔交联网络中的芳香结构、羧酸位点和可达孔道。
