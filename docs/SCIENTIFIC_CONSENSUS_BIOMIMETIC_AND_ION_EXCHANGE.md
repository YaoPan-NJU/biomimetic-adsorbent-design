# 仿生吸附材料、静电吸引与离子交换的科学共识

日期：2026-07-19。该文件记录潘尧与Codex达成的科学分类共识，必须先于项目准入判断使用。

## 1. 两个名称描述不同维度

“仿生”描述设计知识的来源与结构—行为—功能转译；“离子交换”描述目标离子进入材料时发生的物理化学过程。两者不是互斥类别。一种材料可以同时具有仿生属性和离子交换属性，科学写作中必须同时如实标注，不能用“仿生”覆盖或取消其离子交换材料身份。

依据[ISO 18458](https://www.iso.org/standard/62500.html)，仿生要求分析生物系统、抽象其功能或运行原理，并把模型转移和应用到技术方案。仿生不是某一种固定作用力，也不要求使用蛋白、肽、细胞或天然材料。[Vincent等](https://doi.org/10.1098/rsif.2006.0127)同样把核心放在生物学知识向技术系统的转移。

依据[IUPAC Gold Book](https://doi.org/10.1351/goldbook.I03167)，离子交换是目标离子的吸附伴随等电荷当量的其他离子解吸或置换。固态有机聚合物型离子交换剂可称为离子交换树脂；更广义的交换剂还包括无机物等。[IUPAC cation exchanger](https://doi.org/10.1351/goldbook.C00909)。

## 2. 静电吸引不等于离子交换

静电吸引回答“为什么带电目标在某个界面或口袋附近更稳定”；离子交换回答“目标离子进入时，哪一种原有离子按多少电荷当量被置换出来”。

材料可以通过局部电场、离子—偶极作用、外层离子对或多位点静电协同吸附目标物，而不发生可测的等当量对离子置换。反过来，只要目标离子进入固定电荷相时等当量置换Cl⁻、Na⁺、H⁺等原有对离子，该过程就属于离子交换，即使同时存在疏水、空腔或形状互补作用。

Donnan分配描述固定电荷相与外液之间为满足电化学平衡和电中性形成的离子分布；它与离子交换材料密切相关，但不等同于一个孤立的等当量交换事件。盐敏感、带电、可再生、Langmuir拟合或ζ电位变化均不能单独证明离子交换。

## 3. 仿生吸附材料的科学定义

本项目采用以下定义：

> 仿生吸附材料是以有证据的生物模型为知识来源，将该模型中与物质捕获、识别、分配、传递或抗干扰有关的结构—行为—功能原理经过抽象和工程转译，构造成能够吸附目标物的技术材料。其仿生属性来自可追溯、可检验的机制映射，而不是生物来源、外形相似或某一种常见作用力本身。

语义最低要求包括：明确生物模型、生物功能有证据、写清抽象原理以及材料实现。若仿生是论文主创新，还必须用删除、打乱、反向或等组成对照证明被转译的结构对材料功能具有独立因果贡献。后者是强科学判据，不是ISO词义本身逐字规定的条件。

## 4. 二维分类

| 材料或过程 | 仿生属性 | 离子交换属性 | 准确名称 |
|---|---|---|---|
| 普通季铵PS-DVB吸附PFHxS并等当量释放Cl⁻ | 无 | 明确 | 阴离子交换树脂 |
| 有完整生物结构—功能转译的带电交换材料 | 可成立 | 明确 | 仿生离子交换材料，不得省略“离子交换” |
| 仿生口袋以形状、氢键和局部静电场协同捕获目标，未出现等当量反离子置换 | 可成立 | 未证实或很弱 | 含静电贡献的仿生吸附材料 |
| 生物来源壳聚糖仅靠质子化胺交换阴离子 | 不自动成立 | 可成立 | 生物基弱碱阴离子交换/吸附材料 |
| 普通多孔碳依靠疏水和色散作用 | 无 | 无 | 多孔碳吸附剂 |

已有论文直接使用`bio-inspired anion-exchange membrane`这一并列命名，说明生物启发和离子交换可以科学共存；该案例不单独证明其满足ISO意义上的全部严格仿生步骤。[Zhao等](https://doi.org/10.1038/srep37285)。

## 5. 离子交换的判定实验

把材料转为已知单一对离子型，在目标吸附过程中同步测量全部相关离子：

\[
R=\frac{\text{释放对离子的电荷当量}}{\text{吸附目标离子的电荷当量}}
\]

- 在较宽负载范围内`R≈1`，且容量受可达IEC约束，强烈支持离子交换。
- 目标显著吸附但没有相应对离子释放，应进一步考虑主客体识别、疏水分配、表面络合或其他机制。
- 只有部分目标吸附伴随对离子释放，应标注为混合机制，并定量讨论离子交换贡献。

辅助实验包括IEC—容量关系、Donnan模型、外盐浓度、同电荷竞争物、表面谱学和再生行为，但均不能替代完整离子物料衡算。

## 6. 仿生因果判定

至少比较：

- 完整仿生结构；
- 总电荷、IEC、孔容和润湿性匹配但空间位点随机化的材料；
- 删除关键生物对应特征的材料；
- 位置错配或反向结构；
- 普通离子交换、静电吸附和多孔碳基准。

若完整体优势可由IEC、孔容、疏水性或扩散解释，仿生机制不成立。若完整体在这些变量匹配后仍显著占优，才能支持空间组织、协同识别或界面过程的独立贡献；若同时发生等当量对离子置换，准确分类仍然是“仿生离子交换材料”。

## 7. PFH-1的最终科学分类

PFH-1由大孔PS-DVB树脂、可质子化叔胺外层和永久季铵内核组成。PFHxS/PFSA的预期主要捕获过程是固定正电位点上的阴离子交换，容量以IEC为核心变量，再叠加疏水分配和颗粒扩散。因此其必须优先称为：

> 弱碱外层—强碱内核的分区型阴离子交换树脂。

“生物转运启发”可以描述设计来源，但目前只是未经因果验证的次级限定。最完整的诚实名称是：

> 生物转运启发的分区型阴离子交换树脂。

它现在不能简称为“仿生吸附材料”。即使未来证明径向顺序具有独立贡献，它也只会升级为“仿生/生物启发的离子交换材料”，不会失去离子交换树脂属性。

## 8. 科学分类与项目准入的顺序

任何新方案先完成上述科学分类，再应用`docs/MATERIAL_SCOPE_BOUNDARY_ION_EXCHANGE.md`的项目边界。项目可以为了避免方向重合而排除以离子交换树脂为主要材料身份的方案，但不得把这种项目选择误写成“仿生材料科学上不能发生离子交换”。
