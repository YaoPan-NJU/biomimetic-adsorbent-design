# 项目术语约定

本文件用于避免把人工材料设计误读为蛋白质或肽链方案。后续面向潘尧的方案、评分、报告和交接必须遵守以下用词。

## 强制用词

- **生物原型：** 用于提供机制证据的蛋白质、核糖体、酶、组织微环境或其他生物识别体系。例如 ERRγ、FABP 和核糖体。生物原型不是默认材料组成。
- **人工识别单元：** 人工合成的小分子、主体分子、固定官能团组合或确定孔口结构，用于在人工材料中识别目标污染物。未完成水相验证时必须加“候选”或“设想”。
- **已验证人工识别单元：** 只有在相关水相中，以正交方法和删除/错配对照证明目标结合及选择性后才能使用的称谓。
- **材料识别位点：** 人工识别单元被聚合、接枝或固定后，在材料中的可达功能位置。
- **仿生吸附材料：** 将生物原型中有证据支持的功能规律转译为人工识别单元、孔道或界面后的非蛋白人工材料。

## 禁止用词与边界

- 不用无修饰的“受体”指代人工小分子、环糊精、杯芳烃、官能团组合或材料位点。
- “生物受体”只在确实讨论受体蛋白且名称准确时使用；核糖体、酶和结合蛋白分别按其实际类别称呼，不统称为受体。
- 不因引用蛋白结构而把蛋白序列、折叠、肽链、表达构建体或易失活生物组分带入材料。
- 不把“候选人工识别单元”写成已经具有选择性的单元；可合成、可聚合或有计算结合均不等于已验证。
- 旧轮次中的历史措辞为保持审计链不批量改写；引用旧文件时必须按本约定重新表述。

## 推荐表达示例

- 正确：ERRγ 是 BPA 机制研究的生物原型；P1-MAPy 是待验证的人工识别单元候选。
- 正确：β-CD 提供人工主体腔，但 BPA 选择性增量尚未建立。
- 错误：P1-MAPy 是 BPA 受体。
- 错误：材料复制了 ERRγ 受体结构。
