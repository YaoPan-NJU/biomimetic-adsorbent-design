# 匿名方案 A-Q

本文件只含待评分的概念正文。不得从写作风格推断生成方法；请仅按统一量表和证据台账评分。

## 1. PFOA：脂肪酸结合蛋白启发的头尾协同分子夹

### 1.1 工程问题与真实生物原型

PFOA 在二级出水中与 PFOS、PFHxS、PFBS、HFPO-DA、天然脂肪酸和 DOM 共存。单纯疏水腔容易偏好所有长链有机物，单纯正电位点又会被背景阴离子占据。因此，材料必须同时读取羧酸头基位置和全氟链长度，并证明入口闭合对选择性的独立贡献。

- **生物原型**：人肝脂肪酸结合蛋白（hL-FABP）。
- **事实**：美国九座市政污水处理设施的最终出水调查中，PFOA、PFOS、PFBS 和 PFHxS 均被检出；该频率不能外推到所有污水厂。[E001]
- **带边界事实**：分子模拟显示，Arg122 替换会扩大 hL-FABP 外部口袋，并扰动 PFOA 的疏水堆积和氢键作用。该证据是模拟结果，不是突变结合实验，也不证明人工动态机制。[E002]
- **现实水体边界**：交联 β-环糊精材料已能在市政污水中去除 PFAS 和药物，说明普通主体-客体路线已有强竞争者。[E003]

### 1.2 材料架构

构建可装填于流通柱的层级多孔二氧化硅单体。孔壁仅使用一层薄粘附底层固定亲水聚合物刷，聚合物刷末端形成两个相邻模块：

1. **头基模块**：在口袋入口固定胍基样氢键/静电位点，模拟 Arg122 所在外部区域对 PFOA 羧酸头基的定位。
2. **链段模块**：入口后设置长度受限、低极性的狭长腔体，提供针对 PFOA 链段的形状和疏水堆积环境，不采用可脱落含氟小分子作为亲和配体。
3. **柔性盖片**：链段模块位于可动盖片上。PFOA 同时占据头基和链段模块时，盖片靠近孔壁形成局部闭合；降低协同作用的温和洗脱条件使盖片重新打开。

层级孔只承担通量和位点可达性，薄粘附层只承担接枝。两者都不得被解释为 PFOA 选择性的来源。

### 1.3 静态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | hL-FABP 外部口袋的 Arg122 相关区域与内部疏水堆积共同影响 PFOA 的模拟结合。[E002] |
| 材料实现 | 胍基样入口位点固定羧酸头基，狭长低极性腔读取链段长度，形成头尾协同口袋。 |
| 预期功能 | 同时抑制普通阴离子、短链 PFAS 和天然脂肪酸的非特异竞争。PFOA/PFOS 的头基区分仍须实验决定。 |
| 测量 | PFOA/PFOS、PFOA/PFHxS、PFOA/PFBS、PFOA/HFPO-DA、PFOA/脂肪酸选择性系数；结合热；原位红外；单位位点穿透容量。 |
| 因果对照 | 胍基改为中性基团；缩短或延长链段腔；保持官能团数量但随机分布；支撑体和粘附底层空白。 |

### 1.4 动态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | hL-FABP 为带入口的可达性受限配体口袋；当前证据未直接核实 PFOA 诱导的特定开闭过程，因此动态生物依据仍是待补缺口。[E002] |
| 材料实现 | 在盖片与孔壁放置荧光供受体，PFOA 跨接头基与链段模块时产生距离缩短信号；洗脱时返回开态。 |
| 预期功能 | 开态提高进入速率，闭态降低被 DOM 和相近 PFAS 置换的速率。 |
| 测量 | 时间分辨 FRET、吸附/解吸速率、盖片距离与占位量的同步曲线、至少五次开闭循环。 |
| 因果对照 | 共价锁开盖片；缺失链段模块的盖片；铰链长度错配；同离子强度但无 PFOA 的响应对照。 |

### 1.5 可证伪预测

1. PFOA 应同时产生双模块占位和盖片闭合，PFBS 只能弱占据或不能稳定闭合。
2. 若随机位点材料达到相同的 PFOA/PFBS 与 PFOA/脂肪酸选择性，预组织口袋主张失败。
3. 若锁开材料与可动材料在二级出水中具有相同抗置换能力，动态闭合没有独立作用。
4. 若空白粘附层贡献大部分 PFOA 去除，则接枝底层造成非特异吸附，概念不成立。

### 1.6 风险

- hL-FABP 证据主要来自分子模拟，缺少直接 PFOA 诱导动态证据。
- PFOA、PFOS 和长链脂肪酸的尺寸/疏水性接近，头基模块可能无法提供足够区分。
- 层级支撑和粘附底层可能增加非特异吸附，掩盖分子夹的贡献。
- 过度刚化可提高几何保持但降低传质，柔性过高又会丢失预组织优势。

## 2. PFOS：双区两亲口袋与配体锁定盖片

### 2.1 工程问题与真实生物原型

PFOS 需要在 PFOA、PFHxS、PFBS、其他表面活性有机物和 DOM 背景中被选择性捕获。既有壳聚糖印迹材料已经证明静态 PFOS 印迹不是新路线，因此本概念把创新点限定为磺酸头基/链长双区读取与配体锁定盖片的因果耦合。[E004]

- **生物原型**：脂肪酸结合蛋白家族的两亲性配体口袋。
- **事实**：PFOS 在一项九座市政污水处理设施调查的最终出水中被检出；不能外推普遍频率。[E001]
- **证据边界**：当前直接生物识别来源只涉及 hL-FABP 与 PFOA 的模拟结合，不足以证明 PFOS 适用。[E002]
- **先前技术事实**：壳聚糖分子印迹材料早已用于 PFOS 选择性吸附；主要结果来自以合成水为主的研究。[E004]

### 2.2 材料架构

在中性亲水多孔凝胶中构建长度可调的两亲口袋：入口为定距的双阳离子/氢键站，用于识别体积和电荷分布更大的磺酸头基；内部为与 C8 链长匹配的狭长腔。柔性盖片携带腔体末端的第二识别站，只有长链 PFOS 能同时占据入口和末端并稳定闭合。外层采用亲水抗污刷，避免把高疏水表面当作选择性来源。

### 2.3 静态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | 脂肪酸结合蛋白以入口极性区和内部低极性腔共同容纳两亲配体；PFOS 对该原型的适用性待验证。[E002] |
| 材料实现 | 磺酸头基双点站与 C8 狭长腔形成串联口袋，入口间距和腔长分别作为可调变量。 |
| 预期功能 | 头基站区分 PFOS/PFOA，腔长区分 PFOS/PFHxS/PFBS；两项都成功才可宣称 PFOS 选择性。 |
| 测量 | PFOS/PFOA、PFOS/PFHxS、PFOS/PFBS 分配选择性；竞争等温线；穿透次序；DOM 存在下位点保持率。 |
| 因果对照 | 单阳离子入口；C4/C6/C10 错配腔；随机官能团凝胶；已知静态 PFOS 印迹材料对照。[E004] |

### 2.4 动态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | 两亲配体口袋需要调节入口可达性，但当前没有 PFOS 诱导蛋白开闭的直接证据。 |
| 材料实现 | PFOS 的长链同时连接入口和盖片末端站，使盖片闭合；短链 PFBS 达不到末端站，预计不形成稳定闭合。 |
| 预期功能 | 将链长读取转化为可测构象事件，并降低长链目标被短链共存物置换。 |
| 测量 | FRET、单颗粒尺寸、闭合寿命、不同链长 PFAS 的闭合阈值、再生开态恢复率。 |
| 因果对照 | 锁开盖片；去除末端站；改变铰链长度；亲水外层移除对照。 |

### 2.5 可证伪预测

1. PFOS 的闭合寿命应高于 PFHxS 和 PFBS；若只与总疏水性相关而不随口袋长度匹配变化，链长读取主张失败。
2. 单阳离子入口若保持相同 PFOS/PFOA 选择性，双点磺酸识别没有独立贡献。
3. 锁开材料若不损失动态穿透选择性，则盖片只增加复杂度。
4. 若 DOM 去除亲水外层后没有显著恶化选择性，外层抗污功能不能作为核心设计贡献。

### 2.6 风险

- 缺少 PFOS 与脂肪酸结合蛋白的直接结构证据，原型适用性弱于 PFOA。
- 双阳离子站可能仍被硫酸盐和有机磺酸盐占据。
- PFOS 印迹研究成熟，静态部分容易与既有路线重叠。[E004]
- 所谓动态闭合可能只是 PFOS 表面活性引发的非特异凝胶收缩。

### 2.7 自评

| 维度 | 分数 | 说明 |
|---|---:|---|
| 科学正确性 | 11/15 | 两亲口袋合理，但 PFOS 生物识别缺直接证据。 |
| 因果闭环 | 17/20 | 头基、链长、盖片和抗污层均可拆分。 |
| 仿生保真度 | 10/15 | 原型为家族级类比，动态未被生物证据锁定。 |
| 选择性机制 | 12/15 | 能测试头基与链长，背景阴离子风险高。 |
| 原创性 | 11/15 | 动态链长门控有区别，静态 PFOS 印迹已拥挤。 |
| 证据质量 | 6/10 | 场景和先前技术较清晰，原型证据间接。 |
| 可证伪性 | 10/10 | 链长序列、锁开与凝胶收缩对照可直接裁决。 |
| **总分** | **77/100** | 不达通过线，须补 PFOS 直接生物结合证据。 |

## 3. HFPO-DA（GenX）：短链醚型全氟羧酸的分裂式极性口袋

### 3.1 工程问题与真实生物原型

HFPO-DA 链更短并含醚连接，常规长链疏水腔的驱动力可能不足。设计必须同时读取羧酸头基、醚氧局部极性和短链轮廓，且与 PFOA、PFHxA、PFBA 及其他醚型 PFAS 比较。

- **生物原型**：hL-FABP 的头基定位区与可塑疏水口袋。
- **事实**：HFPO-DA 在美国九座市政污水处理设施最终出水调查中的检出频率为 67%；该频率不能推广到其他地区或工艺。[E001]
- **带边界事实**：hL-FABP 的 PFOA 模拟结合受 Arg122 和外部口袋尺寸影响，但没有 HFPO-DA 直接结合证据。[E002]
- **现实水体边界**：普通主体-客体材料在市政污水中对 PFAS 的亲和力受疏水性影响，提示短链 HFPO-DA 可能处于劣势。[E003]

### 3.2 材料架构

构建两个相对的短臂半口袋。第一半口袋以胍基样位点定位羧酸头基；第二半口袋包含一个定向氢键供体/受体对，尝试读取醚氧附近的局部极性；两者之间仅保留适配短链轮廓的浅低极性区。HFPO-DA 三点同时占据后拉近两个短臂，形成局部闭合。材料不使用更长的疏水隧道，以免自动偏向 PFOA。

### 3.3 静态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | hL-FABP 通过头基附近相互作用与口袋内部堆积识别 PFOA；向 HFPO-DA 转译属于未经验证的可塑口袋假说。[E002] |
| 材料实现 | 羧酸站、醚氧站和短链浅腔构成三点分裂式口袋。 |
| 预期功能 | 减少对总疏水性的依赖，使 HFPO-DA 相对 PFOA 和普通短链全氟羧酸获得几何/极性识别。 |
| 测量 | HFPO-DA/PFOA、HFPO-DA/PFHxA、HFPO-DA/PFBA 选择性；结合热；醚氧站删除前后的光谱和穿透差异。 |
| 因果对照 | 删除醚氧站；延长疏水腔；打乱三点距离；仅保留羧酸站。 |

### 3.4 动态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | 脂肪酸结合口袋具有受限入口和可塑内部空间；HFPO-DA 诱导闭合未有直接证据。 |
| 材料实现 | HFPO-DA 同时桥接两个短臂时产生 FRET 增强；PFOA 因腔体过浅、PFBA 因跨度不足而难以形成同等闭合。 |
| 预期功能 | 把“短链+醚氧”联合匹配转化为构象判别，而不是仅依靠疏水分配。 |
| 测量 | FRET、闭合寿命、不同 PFAS 的结合-构象耦合、开闭再生循环。 |
| 因果对照 | 锁开短臂；醚氧站删除；臂长错配；同浓度非氟醚羧酸对照。 |

### 3.5 可证伪预测

1. 删除醚氧站应显著降低 HFPO-DA 相对普通短链全氟羧酸的选择性；若无变化，醚识别假说失败。
2. 延长疏水腔若使 PFOA 取代 HFPO-DA 成为优先目标，将支持链长几何是关键变量。
3. 若非氟醚羧酸产生相同闭合，材料只识别羧酸+醚，不具 HFPO-DA 选择性。
4. 若锁开不改变真实水体选择性，则动态三点闭合无效。

### 3.6 风险

- 当前没有 HFPO-DA 生物识别原型的直接证据，整个口袋是跨目标转译。
- 水相中醚氧氢键读取可能太弱，三点口袋可能退化为羧酸离子交换。
- HFPO-DA 痕量分析和标准品要求较高。
- 对其他醚型 PFAS 的选择性边界需要额外标准品，实验负担较大。

| 维度 | 分数 | 说明 |
|---|---:|---|
| 科学正确性 | 10/15 | 污染物特征明确，生物映射为间接假说。 |
| 因果闭环 | 17/20 | 三点拆分和臂长对照可裁决核心机制。 |
| 仿生保真度 | 9/15 | 借用可塑口袋原则，缺少 HFPO-DA 天然原型。 |
| 选择性机制 | 12/15 | 明确区分链长和醚氧，但水相作用可能过弱。 |
| 原创性 | 13/15 | 针对短链醚型 PFAS 的动态分裂口袋较少见。 |
| 证据质量 | 5/10 | 仅场景直接，识别证据来自 PFOA 外推。 |
| 可证伪性 | 10/10 | 醚站删除、臂长和非氟醚对照直接。 |
| **总分** | **76/100** | 创意高但证据弱，不宜直接进入实验级方案。 |

## 4. 克拉霉素：核糖体出口隧道启发的串联识别纳米孔

### 4.1 工程问题与真实生物原型

克拉霉素必须与红霉素、罗红霉素、阿奇霉素等大环内酯及 DOM 中的大分子有机物区分。普通疏水腔只能识别大环骨架，无法读取环尺寸、糖基排列和入口方向，因此采用沿纳米孔轴向排列的串联识别站。

- **生物原型**：细菌核糖体大亚基的肽链出口隧道及其大环内酯结合区。
- **带边界事实**：一项瑞士 Glatt Valley 研究报告夏季处理出水中克拉霉素为 57–330 ng/L；未核对工艺终点前不能重新标注为特定二级出水。[E005]
- **已筛查事实**：四种大环内酯占据细菌大核糖体亚基出口隧道中的明确位置。[E006]
- **证据边界**：E006 支持静态几何，不足以支持材料闭合或克拉霉素特异动态机制。

### 4.2 材料架构

制备亲水多孔膜，在孔径与大环内酯尺度相近的短纳米孔内设置三个轴向站点：入口站读取可质子化糖基的电荷/氢键方向；中段站提供对 14 元大环骨架的限域；末端站读取克拉霉素取代基的空间占位。入口连接一条柔性环形刷，只有目标以正确方向同时占据三站时才收缩孔口并延长停留时间。

### 4.3 静态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | 大环内酯在核糖体出口隧道的定义位置通过限域和多点接触结合。[E006] |
| 材料实现 | 沿短纳米孔轴向布置糖基站、大环限域站和末端取代基站，复制串联几何读取。 |
| 预期功能 | 通过入口方向、环尺寸和末端占位共同区分十四元与十五元大环内酯。具体克拉霉素/红霉素区分仍是待验证假说。 |
| 测量 | 克拉霉素/红霉素、克拉霉素/罗红霉素、克拉霉素/阿奇霉素选择性；孔内停留时间；竞争穿透；原位光谱。 |
| 因果对照 | 逐一删除三个站点；扩大孔径；随机分布相同官能团；仅有疏水孔的对照。 |

### 4.4 动态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | 大环内酯占据出口隧道后形成限域阻断；现有证据未证明一个可转译的配体诱导闭合过程。[E006] |
| 材料实现 | 三站同时占据时，入口柔性环形刷因多点桥接而收缩，降低孔口可达性；温和改变离子环境或竞争洗脱解除桥接。 |
| 预期功能 | 把正确轴向占位转化为可测门控信号，减少相近大环内酯和 DOM 的置换。 |
| 测量 | 孔口 FRET、离子探针通量、单分子/群体停留时间、吸附-门控同步性和再生循环。 |
| 因果对照 | 共价锁开入口刷；去除末端站；使用环尺寸错配孔；非结合大分子引起的堵孔对照。 |

### 4.5 可证伪预测

1. 克拉霉素应产生三站占位与入口收缩的同步信号；阿奇霉素因环尺寸不同应呈不同停留/门控曲线。
2. 删除任一关键站点后若仍保持克拉霉素选择性，串联识别链不成立。
3. 若 DOM 或任一大分子抗生素都能引起相同孔口收缩，则测到的是堵孔而非仿生门控。
4. 锁开膜若具有相同选择性，则动态环形刷不必要。

### 4.6 风险

- 核糖体结构来源不能直接支持克拉霉素诱导入口闭合，动态对应证据弱。
- 纳米孔合成误差可能大于几种大环内酯的几何差异。
- 痕量浓度下 FRET 与停留时间测量的灵敏度要求高。
- 普通药物分子印迹已有成熟路线，三站几何必须优于随机印迹才有论文价值。[E020]

## 5. O-去甲基文拉法辛（ODV）：SERT 启发的离子耦合交替可达口袋

### 5.1 工程问题与真实生物原型

ODV 必须与母体文拉法辛、其他芳香胺类药物和 DOM 芳香位点区分。ODV 特异识别证据不足，因此本概念把最关键的静态差异设为酚羟基读取，把动态差异设为离子耦合口袋可达性；两项均须通过母体药物对照裁决。

- **生物原型**：人血清素转运体（SERT）。
- **已筛查事实**：ODV 在微藻二级处理系统中表现出低生物降解和有限去除；目标地区二级出水仍需实测。[E007]
- **带边界事实**：SERT 执行 Na⁺/Cl⁻ 依赖的血清素再摄取，2016 年结构显示向外开放状态及由 TM1、3、6、8、10 围成的中央配体位点。该来源不解析完整交替可达循环，也不证明 ODV 特异结合。[E008]
- **先前技术边界**：药物分子印迹路线成熟，真实水体基质仍是主要障碍。[E020]

### 5.2 材料架构

构建含中央口袋的双门纳米凝胶。中央口袋包含三个空间分离模块：负电/氢键模块读取可质子化胺，低极性模块容纳芳香/环状骨架，定向供受体模块读取 ODV 的酚羟基。外门和内门采用离子响应可逆交联，代表向外可达和向内可达两个状态；目标结合后外门收缩，改变盐条件或施加温和竞争洗脱使口袋重新向外开放并释放。

### 5.3 静态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | SERT 中央配体位点由多个跨膜螺旋围成并用于芳香胺类配体识别；ODV 适用性未由 E008 直接证明。[E008] |
| 材料实现 | 胺模块、疏水骨架模块和酚羟基模块形成三点中央口袋。 |
| 预期功能 | 酚羟基模块提供相对母体文拉法辛的主要差异读取，三点几何抑制普通芳香胺和 DOM。 |
| 测量 | ODV/文拉法辛、ODV/其他芳香胺药物、ODV/DOM 模拟物选择性；结合热；光谱位移；固定床竞争穿透。 |
| 因果对照 | 删除酚羟基模块；删除胺模块；保持组成但随机排列；文拉法辛模板错配口袋。 |

### 5.4 动态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | SERT 具有 Na⁺/Cl⁻ 依赖和向外开放中央位点；完整交替可达过程及 ODV 耦合尚未由当前证据锁定。[E008] |
| 材料实现 | 外门和内门分别布置 FRET 对；在代表性离子条件下外门开放，ODV 三点占据后外门收缩。再生时改变离子交联状态，使外门重新开放并释放。 |
| 预期功能 | 入口开态保持传质，结合后闭态降低被 DOM 置换；离子条件成为可控再生变量。 |
| 测量 | 双门 FRET、Na⁺/Cl⁻ 梯度响应、结合-收缩同步性、锁开/锁闭状态的吸附和解吸动力学。 |
| 因果对照 | 离子不敏感凝胶；外门锁开；内门删除；无 ODV 的等离子强度对照。 |

### 5.5 可证伪预测

1. 删除酚羟基模块应使 ODV/文拉法辛选择性趋近于 1；若不变，ODV 特异读取主张失败。
2. ODV 结合与外门收缩应在相同浓度区间发生；若盐本身引起同等收缩，则响应不是配体耦合。
3. 锁开材料若仍保持真实二级出水中的选择性和抗置换能力，动态门控没有必要性。
4. 若其他芳香胺药物产生相同三点结合和双门响应，该材料只能识别类别，不能宣称 ODV 选择性。

### 5.6 风险

- SERT 结构不能证明 ODV 是其配体，静态原型是机制类比。
- 二级出水离子强度变化可能引起非特异凝胶收缩，淹没目标响应。
- ODV 与文拉法辛的差异位点少，酚羟基模块必须提供可重复的增量选择性。
- 双门材料复杂度高，若单门或静态材料性能相同，应主动简化或淘汰动态叙事。
