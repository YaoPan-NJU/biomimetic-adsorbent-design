# 方案批次 B

## 共同边界

本文件给出五个待攻击的材料概念。每个概念均区分事实层、受限线索和转译假说。事实层仅复述证据台账的支持范围；受限线索不能独立支撑硬对应；转译假说必须由材料表征、竞争实验和因果对照检验。共同应用场景为市政污水处理厂二级出水，超纯水用于本征机制实验。

## B1. SMX：DHPS式“PABA位点匹配—柔性环封口”双层纳米口袋

### 事实层、受限线索与转译假说

- **事实层：** SMX 及其乙酰代谢物在多座 WWTP 中高频检出，SMX 对藻类的风险评估结果具有关注价值（E009，`verified_with_boundary`）。DHPS 结构研究支持 PABA/磺胺类识别和耐药机制，但当前台账没有直接核实可移植的可逆环闭合（E010，`screened_abstract`）。
- **受限线索：** 壳聚糖、植物多酚和聚多巴胺能够提供氢键、芳香作用和 pH 依赖电荷，但现有输入对 SMX 均无直接证据，相关机制来自染料或广谱吸附体系。
- **转译假说：** 将 DHPS 的底物口袋与柔性封口环分别实现为刚性内腔和可移动外层。聚多巴胺仅作为薄附着底层，不承担 SMX 识别；识别层由定向酰胺/脲位点、芳香壁和异恶唑端部体积截止位点组成。

### 真实生物原型

二氢蝶酸合成酶（DHPS）的 PABA/磺胺结合位点。该原型对磺胺骨架识别有结构依据；“配体进入后柔性环封口并可逆释放”仍是待补原始证据的动态候选。

### 静态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | DHPS 对 PABA/磺胺类的位点几何与化学识别（E010）。 |
| 材料实现 | 在刚性纳米腔内依序布置磺酰胺氢键位点、芳香夹层和异恶唑体积截止位点；外表面接枝两性离子抗污链。 |
| 预期功能 | 依靠三点空间匹配区分 SMX、PABA、磺胺嘧啶、磺胺吡啶及乙酰-SMX，避免仅按电荷吸附。 |
| 测量 | 混合物 LC-MS/MS、位点滴定、竞争结合常数、不同 pH 下物种分布校正后的选择性系数。 |
| 因果对照 | 删除异恶唑截止位点；打乱氢键位点方向；等官能团无腔材料；只含附着底层的载体。 |

### 动态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | DHPS 催化口袋附近柔性环参与底物定位的假说；本证据集尚不足以将其记为硬事实。 |
| 材料实现 | 腔口连接两条短肽拟态链。SMX 完成三点定位后同时接触两链，使其靠近并降低口袋可达性；温和改变 pH 或加入 PABA 竞争物使封口重新开放。 |
| 预期功能 | 降低正确取向 SMX 的解离速率，并在再生阶段实现可逆释放。 |
| 测量 | 链端 FRET、结合/解离速率、PABA 置换动力学、闭合信号与 SMX 占据率的同步性。 |
| 因果对照 | 永久开放短链、单链、刚性交联封口、保留静态腔但删除动态链。 |

### 竞争物、可证伪预测与风险

- **竞争物：** 其他磺胺类、PABA、乙酰-SMX、芳香含氮药物、DOM 和无机离子。
- **可证伪预测：** 若三点位点有效，打乱空间方向应降低 SMX/其他磺胺选择性但不显著改变总官能团量。若动态封口有效，完整材料应比永久开放对照具有更慢的 SMX 解离和更强的 DOM 抗置换能力。若 FRET 只随 pH 变化而与占据率无关，动态主张不成立。
- **风险：** DHPS 动态环证据缺口较大；水中氢键方向性可能不足；聚多巴胺或壳聚糖若暴露过多会造成非特异吸附，并掩盖真正口袋贡献。

## B2. BPA：ERRγ式“双酚定位—占据诱导翼片重排”受体膜

### 事实层、受限线索与转译假说

- **事实层：** ERRγ 对 BPA 存在定义明确的受体相互作用（E011，`screened_abstract`）。该来源不独立证明可靠的动态对应，也没有为目标二级出水提供出现证据。
- **受限线索：** 壳聚糖/石墨烯、植物多酚和聚多巴胺匹配均源于通用 π-π、氢键或染料吸附机制，不能区分 BPA、BPS 和 BPF，也不能作为受体机制证据。
- **转译假说：** 构建 ERRγ 配体口袋式双酚腔体，在两个相对位置提供酚羟基氢键位点，中部提供疏水体积匹配；腔口弹性翼片只在两端同时定位后发生重排。

### 真实生物原型

人核受体 ERRγ 的 BPA 结合口袋。静态结合有直接研究线索；动态翼片对应属于从核受体构象调节抽象出的假说，不是 E011 已证实结论。

### 静态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | ERRγ 通过特定受体相互作用识别 BPA 的酚羟基和疏水骨架（E011）。 |
| 材料实现 | 在薄膜纳米腔的两端布置方向相反的氢键受体/供体，中部设置与 BPA 异丙叉连接的双芳环体积相符的疏水区。 |
| 预期功能 | 通过羟基间距和中央桥连体积区分 BPA、BPS、BPF、双酚 AF 及单酚。 |
| 测量 | BPA/BPS/BPF 竞争系数、等官能团腔体尺寸序列、固态光谱和分子探针位移。 |
| 因果对照 | 删除一端氢键位点；扩大中央桥空间；打乱两端距离；无腔平面芳香膜。 |

### 动态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | 核受体配体占据可伴随局部构象重排，但 E011 未证明 BPA-ERRγ 所需的具体动态路径。 |
| 材料实现 | 两片弹性翼片分别连接腔体两端。只有两个酚羟基同时就位时，翼片协同靠近并形成闭合 FRET 状态；改变溶剂组成或加入竞争双酚使翼片开放。 |
| 预期功能 | 以双端协同阈值降低单酚和 DOM 芳香组分的非特异占据。 |
| 测量 | 单分子/群体 FRET、QCM-D、结合与解离速率、不同双酚触发的闭合幅度。 |
| 因果对照 | 单翼片、永久开放翼片、两端距离错配、只保留静态口袋。 |

### 竞争物、可证伪预测与风险

- **竞争物：** BPS、BPF、双酚 AF、壬基酚、酚类 DOM 和其他芳香有机物。
- **可证伪预测：** 若双端定位有效，删除任一端位点应使 BPA/BPS/BPF 的差异收敛。若协同闭合有效，BPA 触发的闭合幅度与解离时间应不同于单酚；若所有芳香物都引起相同收缩，动态主张被否证。
- **风险：** 动态原型证据弱；双酚材料研究高度拥挤；通用多酚、石墨烯或聚多巴胺表面会放大非特异吸附并降低论文新意。

## B3. 磷酸盐：PBP-1式“十四氢键口袋—双叶闭合”多孔受体

### 事实层、受限线索与转译假说

- **事实层：** 磷酸盐结合态的 *C. perfringens* PBP-1 以闭合构象将磷酸盐完全包埋在两个结构域之间，并由九个残基形成十四个氢键（E015，`verified`）。PstS/PBP 家族可显著区分磷酸盐与硫酸盐/砷酸盐，但选择性倍数依蛋白和物种而变（E016，`verified_with_boundary`）。二级出水中的氮磷控制具有工程意义，代表性水样同时含硫酸盐、DOM、硝酸盐和磷（E013、E014）。
- **受限线索：** 牡蛎壳/CaO 对磷酸盐有直接沉淀线索；HAp、铁氧化矿物和壳聚糖提供常规配位/沉淀路线。上述机制可作工程对照，但不具备 PBP-1 式选择性和可逆闭合。硫酸盐还原过程与磷酸盐选择性识别无直接关系。
- **转译假说：** 以两个相对的短肽-聚合物叶片构建水相受体，叶内预组织多重酰胺/胍盐氢键位点；磷酸盐完成几何配位后驱动双叶闭合。材料不使用 Ca²⁺沉淀作为主机制。

### 真实生物原型

细菌 ABC 型磷酸盐转运系统的 PBP-1/PstS 结合蛋白，包含两个结构域夹缝中的预组织氢键网络和配体诱导闭合。

### 静态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | 九个残基、十四个氢键形成的完全包埋磷酸盐口袋（E015，仅对应所研究蛋白）。 |
| 材料实现 | 在两片刚性低聚肽叶面布置方向固定的脲、酰胺和胍盐位点，闭合后形成四面体互补的多点水相口袋；外层接枝中性抗污链。 |
| 预期功能 | 通过四面体几何和完全包埋区分磷酸盐、硫酸盐、碳酸氢根、硝酸盐和砷酸盐。 |
| 测量 | 五种阴离子的竞争结合常数、离子色谱/钼蓝法交叉定量、位点突变序列和选择性关系。 |
| 因果对照 | 逐级删除氢键位点；改变四面体角度；等电荷平面材料；CaO/牡蛎壳沉淀对照。 |

### 动态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | PBP-1 在磷酸盐结合态闭合并完全包埋目标物（E015）。 |
| 材料实现 | 两叶通过弹性铰链连接，叶端设置 FRET 对。磷酸盐同时满足两叶位点后稳定闭合态；降低磷酸盐活度并以温和竞争缓冲液解除多点作用后开放。 |
| 预期功能 | 未闭合材料保持快速传质，闭合后提高选择性和驻留时间，并允许非破坏性释放。 |
| 测量 | FRET/SAXS 闭合、开闭速率、静态与动态选择性、穿透次序、循环释放和位点稳定性。 |
| 因果对照 | 刚性开放铰链、刚性闭合铰链、单叶、铰链长度错配；所有对照保持总电荷和位点密度接近。 |

### 竞争物、可证伪预测与风险

- **竞争物：** 硫酸盐、碳酸氢根、氯离子、硝酸盐、砷酸盐、DOM 及天然有机磷。
- **可证伪预测：** 若多点四面体口袋有效，等电荷平面材料应表现出明显更差的磷酸盐/硫酸盐与磷酸盐/砷酸盐选择性。若闭合贡献成立，开放锁定对照应有相近初始接近速率但更快解离和更弱的复杂水体选择性。若闭合信号由总离子强度而非磷酸盐占据驱动，动态主张被否证。
- **风险：** 水中多重氢键可能被强水化削弱；合成位点难以复现蛋白的十四氢键几何；高亲和闭合可能导致温和再生困难。沉淀型匹配容易提供高去除率假象，必须用溶解磷质量平衡和 Ca/P 固相分析区分吸附与沉淀。

## B4. 硝酸盐：NrtA式“平面C-clamp—双结构域交替开闭”复合凝胶

### 事实层、受限线索与转译假说

- **事实层：** 硝酸盐结合态 NrtA 是双结构域 C-clamp，硝酸盐被完全包埋于结构域间裂隙；论文提出 C 端元件可能参与转运相关构象变化，但缺少成对 apo/open 与 bound/closed 结构，且硝酸盐/亚硝酸盐区分有限（E017，`verified_with_boundary`）。
- **受限线索：** 酸浸碳/壳聚糖/FeCl3 体系对硝酸盐有直接吸附线索，可提供成膜和离子交换支架思路；骨/HAp、牡蛎壳、植物多酚和聚多巴胺匹配均来自其他污染物机制，不能支撑硝酸盐口袋。
- **转译假说：** 使用低取代交联壳聚糖形成柔性支架，在其上安装两个刚性叶片和面向裂隙的平面氢键阵列。壳聚糖只提供机械连接和可加工性，硝酸盐选择性由 C-clamp 几何承担。

### 真实生物原型

蓝藻硝酸盐结合蛋白 NrtA 的双结构域 C-clamp 和完全包埋的平面硝酸盐口袋。动态开闭仅有结构推断，不是成对结构已证实事实。

### 静态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | NrtA 在结构域裂隙中完全包埋平面硝酸盐（E017）。 |
| 材料实现 | 两叶内侧布置共平面脲/酰胺供体和体积截止边缘，裂隙厚度适配 NO3− 平面；支架氨基密度保持低且统一。 |
| 预期功能 | 在硫酸盐、碳酸氢根和氯离子背景下优先容纳平面硝酸盐，并评估对亚硝酸盐的不可避免交叉。 |
| 测量 | 硝酸盐/硫酸盐、硝酸盐/碳酸氢根、硝酸盐/亚硝酸盐选择性；离子色谱；裂隙尺寸系列。 |
| 因果对照 | 加厚裂隙；改变供体共平面性；等电荷壳聚糖-Fe 对照；无叶片支架。 |

### 动态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | NrtA 的双结构域构型及 C 端元件提示转运相关构象变化，具体开闭幅度待补证（E017）。 |
| 材料实现 | 两叶由柔性但非 pH 主导的共价铰链连接；硝酸盐同时满足两叶位点后闭合。改变竞争阴离子活度使叶片开放并释放。 |
| 预期功能 | 将快速开放态传质与闭合态低解离结合，避免单纯阴离子交换。 |
| 测量 | FRET、SAXS、阴离子替换动力学、开闭与吸附同步性、二级出水连续流穿透。 |
| 因果对照 | 永久开放/闭合、单叶、删除 C 端模拟链、相同壳聚糖质子化度的无 C-clamp 对照。 |

### 竞争物、可证伪预测与风险

- **竞争物：** 亚硝酸盐、硫酸盐、碳酸氢根、氯离子、磷酸盐和 DOM。
- **可证伪预测：** 若平面裂隙是决定因素，增厚裂隙应优先损害硝酸盐相对硫酸盐的选择性。若双叶闭合有效，永久开放对照的解离应更快；若闭合只随壳聚糖质子化或总盐度变化，NrtA 动态对应被否证。预计硝酸盐/亚硝酸盐选择性有限，实验若证实这一点应被如实接受。

## B5. As(III)：ArsR式“AsS3配位—无序到有序”肽-丝素复合膜

### 事实层、受限线索与转译假说

- **事实层：** AfArsR 金属结合区模型肽在 As(III) 存在时由无规卷曲转为有序状态，并形成类似天然蛋白的三角锥 AsS3 位点（E018，`verified_with_boundary`）。该证据来自模型肽，不是完整蛋白构象实验。简单 L-半胱氨酸固定化砷吸附膜已有先例，单纯增加巯基不具备充分新颖性（E019）。
- **受限线索：** 铁氧化微生物生成的施氏矿物对 As(III) 有直接吸附线索，适合作为无机基准；丝素蛋白复合材料可提供 β-折叠机械骨架，但现有输入中的 As(III) 专属性不足。HAp、牡蛎壳和硫酸盐还原过程不提供 AsS3 选择性原型。
- **转译假说：** 将包含 ArsR 三半胱氨酸位点的短肽以单端、低密度方式接枝到再生丝素纳米纤维膜的无定形区，使肽保留构象自由度；丝素 β-折叠区只承担机械支撑和抗流失功能。

### 真实生物原型

AfArsR 金属结合区的三半胱氨酸 AsS3 口袋及 As(III) 诱导的无序到有序转变（E018）。

### 静态硬对应候选

| 字段 | 内容 |
|---|---|
| 生物特征 | 三个半胱氨酸围绕 As(III) 形成三角锥 AsS3 配位位点（E018，模型肽边界）。 |

### 竞争物、可证伪预测与风险

- **竞争物：** Sb(III)、Hg(II)、Cu(II)、Cd(II)、As(V)、硫醇类 DOM、磷酸盐及氧化剂。

### 总体判断

BMDL 对本批次的主要价值不是发现更好的生物识别原型，而是暴露常规材料路线、提供工程支架或基准对照。对 SMX 和 BPA，匹配会把设计压缩到壳聚糖、植物单宁和 PDA 等泛化家族，未带来可用增益；对磷酸盐、硝酸盐和 As(III)，帮助集中在识别“沉淀/离子交换/矿物吸附”对照和丝素/壳聚糖支架。配对 Reviewer 应把这些帮助与真正的选择性原型贡献分开评分。
