# Model-only 方案批次 A

## 隔离声明与使用边界

本文件是在未读取 BMDL 数据、审计结果或辅助设计输出的条件下生成的纯模型基线。设计输入仅限项目规约、污染物短名单和证据台账。文件中的“已核实事实”只复述证据台账所记录的支持范围；`pending_spot_check` 证据不能作为最终硬对应的充分依据。材料结构、响应机制和性能方向均为转译假说，不是实验事实，也不是合成 SOP。

共同应用场景为市政污水处理厂二级出水，超纯水用于本征机制和因果对照。五个概念均以目标物/主要竞争物的分配系数、选择性系数、动态开闭信号、解吸动力学和真实水样穿透次序作为判断核心，容量仅作为辅助指标。

## A1. PFOA：脂肪酸结合蛋白式“头基锁定—口袋闭合”吸附体

### 已核实事实

- 美国市政 WWTP 调查在全部九座设施中检出 PFOA、PFOS、PFBS 和 PFHxS；证据仍待全文定位核验（E001，`pending_spot_check`）。
- 人肝脂肪酸结合蛋白（hL-FABP）能够结合全氟烷基酸；证据台账指出 Arg122 对模型化 PFOA 相互作用重要，但该条目前来自摘要，且尚未证明可迁移的动态材料机制（E002，`pending_spot_check`）。
- 交联 β-环糊精材料已用于市政污水中的 PFAS 和药物去除，说明疏水空腔本身不是新颖点（E003，`screened_abstract`）。

### 真实生物原型

hL-FABP 的内部疏水腔体及带正电的头基定位区域。原型的事实层只覆盖 PFAA 结合和 PFOA/Arg122 模型化相互作用；“入口区开闭控制配体交换”在本证据集中尚未直接核实。

### 静态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | hL-FABP 以内部腔体容纳疏水链段，并以 Arg122 附近正电位点定位 PFOA 羧酸头基（E002，待全文核验）。 |
| 材料实现假说 | 构建窄分布疏水纳米腔，在入口布置平面双齿脒/胍盐位点，使羧酸盐头基定向，腔深匹配 PFOA 链长；外表面采用中性亲水层抑制 DOM 非特异覆盖。 |
| 预期功能 | 同时利用头基几何和链长匹配，区分 PFOA 与 PFOS、PFBA、HFPO-DA 及天然脂肪酸。 |
| 测量 | 水相结合常数、PFOA/PFOS 与 PFOA/PFBA 选择性系数、竞争条件下 LC-MS/MS 定量、入口位点滴定及腔体占据光谱。 |
| 因果对照 | 去除正电入口位点；扩大腔深；以随机分布胍基替代定向双齿位点；使用等官能团但无腔体的聚合物。 |

### 动态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | 假定 FABP 入口柔性区域允许配体进入后降低口袋可达性；本项目证据尚未直接支持这一动态细节。 |
| 材料实现假说 | 在腔口连接两条短两亲性聚合物“门户瓣”。PFOA 头基进入并被双齿位点固定后，链段去溶剂化驱动两瓣靠近，形成可逆闭合；改变 pH/离子组成解除头基配位并重新开放。 |
| 预期功能 | 闭合状态降低解离速率并屏蔽 DOM 对已占据口袋的置换；开放状态支持温和再生。 |
| 测量 | 门户两端 FRET、QCM-D 构象/质量耦合信号、吸附与解吸速率常数、不同溶液条件下的循环恢复率。 |
| 因果对照 | 永久开放的短门户；以刚性交联固定的门户；保留静态腔体但删除闭合链段。 |

### 选择性障碍与竞争物

主要竞争物为 PFOS、PFHxA/PFBA、HFPO-DA、天然脂肪酸、阴离子表面活性剂及 DOM。最大的化学风险是脒/胍盐对磺酸盐或天然羧酸也有高亲和力，导致所谓“PFOA 选择性”实质上由链长和疏水性控制。

### 可证伪预测

1. 若静态几何有效，定向双齿入口在相同官能团密度下应优于随机胍基对照，并表现出对 PFOS 四面体磺酸头基的排斥或显著较低亲和力。
2. 若动态闭合有效，完整材料与永久开放对照的初始结合量可接近，但完整材料应具有更慢的解离、更高的 DOM 抗置换能力和可检测 FRET 变化。
3. 若去除门户后选择性与动力学不变，则动态仿生主张被否证；方案退化为普通腔体吸附剂。

### 风险与待验证信息

- FABP 动态入口机制缺少当前证据编号，不能直接升级为硬事实。
- 水中羧酸盐的定向配位强度、门户单分子闭合幅度和闭合是否由目标物而非离子强度驱动均未知。
- β-环糊精类 PFAS 材料已有真实污水先例（E003），原创性取决于可观测的目标诱导闭合，而非空腔本身。

## A2. PFOS：两亲性蛋白口袋式“磺酸多点锚定—长度闸门”吸附体

### 已核实事实

- PFOS 在九座美国市政 WWTP 中均被检出，但证据仍待全文定位核验（E001）。
- hL-FABP 对 PFAA 类化合物具有结合能力；现有台账只明确记录了模型化 PFOA/Arg122 作用，不能据此声称已证实 PFOS 专属结合构象（E002）。
- 壳聚糖分子印迹聚合物早在 2008 年已用于 PFOS 选择性吸附，静态印迹路线不具备充分新颖性（E004，`screened_abstract`）。

### 真实生物原型

真实原型仍选用脂肪酸结合蛋白的两亲性内部口袋。原型本身真实，但 PFOS 的特异位点和动态门户目前没有直接证据，因此属于“蛋白机制启发”，不是已核实的 PFOS 仿生对应。

### 静态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | 两亲性蛋白口袋以带正电头基区域和疏水内部同时容纳阴离子头基与长链配体；PFOS 专属构象待验证。 |
| 材料实现假说 | 构建长度接近 C8 链段的狭长腔体，在入口设置三点氢键/静电阵列以匹配四面体磺酸根，在腔底设置体积截止位点。 |
| 预期功能 | 头基几何区分 PFOS 与 PFOA，腔深区分 PFOS 与 PFHxS/PFBS。 |
| 测量 | PFOS/PFOA、PFOS/PFHxS、PFOS/PFBS 选择性系数；同位点密度归一化结合常数；混合 PFAS 穿透曲线。 |
| 因果对照 | 双点而非三点头基阵列；延长/缩短腔体；无体积截止位点；等电荷非印迹材料。 |

### 动态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | 脂质结合口袋通过局部构象呼吸调节长链配体进入和释放；PFOS 对该过程的触发尚无当前证据。 |
| 材料实现假说 | 在腔口设置由三点磺酸结合共同拉拢的三臂分子闸门。只有头基完成三点配位且链段深入腔体时，三臂才闭合；竞争离子或温和碱性再生液解除配位并开放。 |
| 预期功能 | 通过协同阈值抑制短链 PFAS 和普通阴离子引发闭合，并延长 PFOS 驻留时间。 |
| 测量 | 三臂间 FRET/激基缔合信号、不同 PFAS 触发阈值、解离速率、循环闭合幅度。 |
| 因果对照 | 仅保留一臂或两臂；永久闭合和永久开放闸门；三点阵列空间错配对照。 |

### 选择性障碍与竞争物

PFHxS、PFBS、PFOA、阴离子表面活性剂、胆汁酸/脂肪酸及高浓度无机阴离子均可能竞争。主要风险是长链疏水性主导结合，使材料偏好所有长链表面活性物，而不是 PFOS。

### 可证伪预测

1. 若三点磺酸几何有效，PFOS 应比 PFOA 更强地触发闸门信号；若信号只随链长变化，头基仿生主张不成立。
2. 若长度截止有效，PFOS/PFHxS 选择性在腔深错配对照中应显著下降。
3. 若三臂删减不影响解离速率和 DOM 抗置换能力，则动态协同闭合被否证。

### 风险与待验证信息

- 当前没有 PFOS 与选定蛋白口袋的直接结构证据，也没有动态门户证据。
- 强磺酸盐结合位点可能导致再生困难；三点配位在水中能否维持未知。
- 与已有 PFOS-MIP（E004）的区分必须由目标诱导的协同开闭和因果对照建立。

|---:|---:|---:|---:|---:|---:|---:|---:|
| 10 | 15 | 9 | 12 | 12 | 4 | 9 | **71** |

未达到 85 分。直接生物机制证据不足是当前高风险问题。

## A3. HFPO-DA（GenX）：醚型短链 PFAS 的“分叉腔体—去溶剂化闸门”吸附体

### 已核实事实

- 当前证据台账没有 HFPO-DA 在目标二级出水中的直接出现证据，也没有 HFPO-DA 与某一蛋白的直接结合结构。
- 市政污水中 PFAS 的 β-环糊精吸附先例只说明通用空腔材料可行，不能证明对 HFPO-DA 的选择性（E003）。
- hL-FABP/PFAA 证据可作为寻找原型的线索，但现有记录只明确到模型化 PFOA 作用，不能外推为 HFPO-DA 已核实原型（E002）。

### 真实生物原型

候选原型为脂肪酸结合蛋白的“极性头基入口 + 柔性疏水腔体”。蛋白结构类别真实，但其对 HFPO-DA 的适用性完全属于待验证信息。本概念因此是五案中证据最弱、但结构转译差异最明确的一案。

### 静态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | 脂肪酸结合口袋将极性头基定位在入口，并允许弯曲/分叉链段占据内部空间；HFPO-DA 是否采取这种构象未知。 |
| 材料实现假说 | 构建非圆柱形分叉腔体：入口的双齿羧酸盐位点定位头基，侧向弱氢键供体面向醚氧，两个疏水子腔容纳支化全氟片段。 |
| 预期功能 | 通过“羧酸头基 + 醚氧位置 + 支化体积”三重约束，区分 HFPO-DA 与线性 PFOA/PFHxA 及其他短链 PFAS。 |
| 测量 | HFPO-DA/PFOA、HFPO-DA/PFHxA 选择性系数；二维核磁或竞争探针验证腔内取向；分叉腔体尺寸与亲和力关联。 |
| 因果对照 | 删除醚氧供体位点；将分叉腔改为等体积直腔；去除羧酸头基位点；随机共聚对照。 |

### 动态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | 柔性蛋白口袋可通过水排出和局部重排容纳支化配体；HFPO-DA 诱导该行为没有直接证据。 |
| 材料实现假说 | 分叉腔入口覆盖亲水门户；只有头基和醚氧先后就位时，门户中的两个弱相互作用点协同去溶剂化并收缩，使支化链段被包埋。改变溶液组成使门户重新水化并释放。 |
| 预期功能 | 以协同阈值补偿短链 PFAS 疏水驱动力不足，并避免单纯依赖长链效应。 |
| 测量 | FRET/溶胀信号、结合焓熵分解、溶剂同位素或水活度扰动、开闭与吸附动力学同步性。 |
| 因果对照 | 不响应亲水门户；仅保留头基识别；仅保留醚氧位点；线性腔体。 |

### 选择性障碍与竞争物

主要竞争物为 PFOA、PFHxA、PFBA、PFBS、其他醚型 PFAS、脂肪酸和 DOM。关键困难是全氟醚氧在水中的可用氢键碱性可能不足，所谓“醚氧识别”可能没有可测贡献。

### 可证伪预测

1. 若醚氧位点有效，删除该位点应优先降低 HFPO-DA 而非线性 PFCA 的亲和力；若所有化合物同步下降，则该位点仅改变总体亲水性。
2. 若分叉几何有效，等体积直腔体对 HFPO-DA 的亲和力和闭合幅度应降低。
3. 若门户信号只由盐度或 DOM 触发而不随 HFPO-DA 占据率变化，则动态仿生主张被否证。

### 风险与待验证信息

- 缺少目标水体出现证据、直接蛋白结合证据和可迁移动态原型，证据质量不足以支持硬对应。
- 醚氧定向作用可能过弱；复杂分叉腔的可合成性与批次一致性未知。
- HFPO-DA 标准品、痕量定量和混合 PFAS 方法成本较高。

## A4. 克拉霉素：核糖体出口隧道式“异质内壁—占据诱导夹持”纳米通道

### 已核实事实

- 克拉霉素等大环内酯可穿过常规污水处理并出现在最终出水；具体浓度范围尚待全文定位核验（E005，`pending_spot_check`）。
- 大环内酯在细菌大核糖体亚基出口隧道中占据确定位置，能够支持静态几何原型（E006，`screened_abstract`）。
- E006 不足以证明“药物结合后隧道主动闭合”；任何材料闭合机制都属于转译假说。

### 真实生物原型

细菌 50S 核糖体出口隧道的大环内酯结合区。静态原型是由 rRNA 主导的有限尺寸通道和异质极性内壁；动态原型不是整个核糖体“夹住药物”，而是配体占据改变通道可通行状态并影响后续分子通过。后者需要补充原始结构/动力学证据。

### 静态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | 核糖体出口隧道以有限空间和多点相互作用识别大环内酯的宏环及糖基排列（E006）。 |
| 材料实现假说 | 在刚性介孔颗粒中构建单分散短纳米通道，内壁按轴向顺序布置疏水区、氢键位点和一个酸性锚点，分别匹配内酯宏环、羟基图案和质子化去氧糖；外层为抗 DOM 的两性离子壳。 |
| 预期功能 | 利用通道直径、宏环取向和糖基锚定，区分克拉霉素与阿奇霉素、红霉素和罗红霉素。 |
| 测量 | 四种大环内酯的混合竞争 LC-MS/MS；通道尺寸系列的选择性；固态核磁/红外位点扰动；单通道电导或荧光占据。 |
| 因果对照 | 扩宽通道；打乱轴向位点顺序；删除酸性糖基锚点；等官能团无通道颗粒。 |

### 动态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | 大环内酯占据出口隧道后改变局部通行状态，并可导致特定底物背景下的停滞；本台账只核实占位，动态结果待补证。 |
| 材料实现假说 | 通道口设置两条相对的短肽拟态“唇瓣”。克拉霉素在静态位点正确取向后，同时接触两瓣并促使其靠近，形成可逆夹持；再生液破坏糖基锚定后门户开放。 |
| 预期功能 | 目标物正确取向才触发夹持，从而降低解离速率；结构相近但糖基位置不同的大环内酯应较少触发闭合。 |
| 测量 | 两瓣 FRET、单通道电导阻断、开闭时间分布、结合/解离速率及闭合幅度与占据率的对应关系。 |
| 因果对照 | 永久开放短瓣；单瓣；柔性长度错配；保留静态通道但删除门户。 |

### 选择性障碍与竞争物

关键竞争物为阿奇霉素、红霉素、罗红霉素、林可酰胺类、带正电药物、蛋白样 DOM 和阴离子表面。通道可能按分子大小而非核糖体式位点顺序筛分；酸性锚点也可能非特异吸附所有阳离子药物。

### 可证伪预测

1. 若轴向位点顺序重要，位点打乱对照在相同官能团密度和通道尺寸下应显著降低克拉霉素/阿奇霉素选择性。
2. 若占据诱导夹持成立，完整材料应出现与克拉霉素占据率相关的可逆 FRET/电导变化，并具有低于无门户对照的解离速率。
3. 若所有大环内酯产生相同闭合幅度，或删除门户不改变竞争选择性，则动态或糖基排列主张被否证。

### 风险与待验证信息

- 当前结构证据只支持大环内酯占位，不支持人工“夹持”机制。
- 纳米通道内壁轴向位点精确排布难度高，可能退化为普通尺寸筛分/静电吸附。
- 二级出水痕量大环内酯的定量、蛋白样 DOM 堵孔和传质速率可能限制实验辨识度。

## A5. O-去甲基文拉法辛（ODV）：SERT式“芳香胺口袋—Na+/Cl-耦合交替开闭”纳米凝胶

### 已核实事实

- ODV 在藻类二级处理研究中表现出低生物降解和有限去除；目标本地二级出水中的出现水平仍待测（E007，`screened_abstract`）。
- 人血清素转运体（SERT）具有中央配体位点以及 Na+/Cl- 耦合的交替构象（E008，`pending_spot_check`）。
- E008 不证明 ODV 的专属结合构象，因此 ODV 识别位点布局属于基于其芳香胺/酚羟基结构的材料假说。

### 真实生物原型

人 SERT 的中央芳香胺配体口袋和离子耦合交替开放机制。动态原型具有直接结构学方向，但 ODV 是否以假定取向占据该口袋、哪些残基决定 ODV/文拉法辛差异，当前证据未核实。

### 静态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | SERT 中央位点为芳香胺类配体提供有限空间、芳香相互作用和离子性锚定环境（E008 的适用边界不含 ODV 专属构象）。 |
| 材料实现假说 | 制备两叶式纳米凝胶口袋：一叶提供芳香/阳离子-π环境和羧酸盐锚点定位质子化胺，另一叶提供针对 ODV 酚羟基的定向氢键位点；外壳采用两性离子抗污层。 |
| 预期功能 | 通过胺头基、芳香核心和酚羟基三点约束区分 ODV 与母体文拉法辛及其他 SSRI/SNRI。 |
| 测量 | ODV/文拉法辛、ODV/去甲文拉法辛及 ODV/代表性 SSRI 选择性系数；竞争 ITC/光谱；二级出水混合穿透实验。 |
| 因果对照 | 删除酚羟基识别位点；将羧酸锚点甲酯化；单叶口袋；随机官能团纳米凝胶。 |

### 动态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | SERT 通过 Na+/Cl- 耦合在外向开放、闭合和内向开放状态间切换（E008）。 |
| 材料实现假说 | 在两叶铰链分别布置弱 Na+ 配位位点和水相阴离子结合位点。ODV 首先占据静态口袋，二级出水中的 Na+/Cl- 共同稳定闭合态；转入缺少相应离子或含竞争离子的再生液后两叶开放并释放。 |
| 预期功能 | 将真实水体离子背景从竞争因素转化为捕获协同因子，并使释放依赖离子组成而非强酸强碱。 |
| 测量 | 叶间 FRET、SAXS/DLS 尺寸变化、NaCl/NaNO3/KCl/KNO3 正交条件下的结合和解离、循环恢复率。 |
| 因果对照 | 删除 Na+ 配位位点；删除阴离子位点；刚性固定闭合或开放的凝胶；相同离子强度但不同离子种类。 |

### 选择性障碍与竞争物

主要竞争物为文拉法辛、N-去甲基文拉法辛、氟西汀/舍曲林等芳香胺药物、阳离子表面活性剂和 DOM 芳香位点。最大风险是材料只表现为普通盐响应水凝胶，或酚羟基位点在水中不足以区分 ODV 与其他含酚药物。

### 可证伪预测

1. 若酚羟基识别决定 ODV/文拉法辛差异，删除该位点应显著降低两者选择性比，但不应等比例消除全部芳香胺吸附。
2. 若 Na+/Cl- 协同闭合成立，NaCl 应比等离子强度的 NaNO3、KCl 和 KNO3 更强地稳定闭合态；任一离子位点删除后协同效应应消失。
3. 若 FRET/尺寸变化只随总离子强度变化而与 ODV 占据无关，则 SERT 式耦合主张被否证，方案退化为盐敏凝胶。

### 风险与待验证信息

- 缺少 ODV-SERT 专属结构证据和目标本地二级出水浓度证据。
- 阴离子受体在水中的结合常数可能过低，Na+/Cl- 正交效应可能被总体盐响应掩盖。
- 三点静态口袋若通过传统印迹获得，必须证明动态叶间开闭的独立贡献，避免与成熟药物 MIP 路线重合（E020）。
