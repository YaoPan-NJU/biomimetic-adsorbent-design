# Model-only 设计批次 B：五种污染物的仿生选择性吸附概念

## 0. 批次边界与判读规则

- 本批次仅依据 `SOUL.md`、`research_contract.yaml`、冻结的污染物短名单和证据台账生成，未联网，也未接触任何外部候选库输出。
- 下文将信息分为三类：
  - **已核实/已筛查事实**：证据台账状态为 `screened_abstract` 或 `screened_review`；支持强度限于台账所记载的摘要或综述层级。
  - **待点检事实线索**：证据台账状态为 `pending_spot_check`；在原文点检前不能作为最终硬对应的充分证据。
  - **转译假说或待验证信息**：由生物机制映射到材料结构、性能或实验结果的设计推断，不当作文献事实。
- “静态硬对应”和“动态硬对应”在本文件中表示**概念已经具备可测材料实现和因果对照**。其证据能否达到最终硬门槛，仍取决于对应原始文献点检。
- 五个概念都以市政污水二级出水为主矩阵，以超纯水为机理对照。容量仅为辅助指标，首要输出为竞争选择性、穿透次序、结构响应和可逆释放。

## 1. 磺胺甲噁唑（SMX）：DHPS 启发的柔性闭合双位点识别凝胶

### 1.1 科学链条

二级出水中的 SMX 与其他磺胺、对氨基苯甲酸（PABA）样分子和 DOM 竞争，且其水相形态随 pH 改变。普通分子印迹材料容易把选择性归因于非特异疏水作用或静电作用，因此本概念要求一个可被“锁开”对照破坏的配体诱导闭合过程。

### 1.2 真实生物原型与证据状态

- **生物原型**：二氢蝶酸合酶（DHPS）的 PABA/磺胺识别位点及催化相关柔性区域。
- **已筛查事实**：DHPS 结构研究解释了 PABA/磺胺识别与耐药机制；台账状态为 `screened_abstract`。[E010]
- **待点检边界**：E010 尚不足以确认哪一段柔性环、闭合幅度及其对 SMX 而非磺胺类共有骨架的贡献。动态对应在原始结构文献点检前只能算“硬对应候选”。
- **场景线索**：SMX 及乙酰化代谢物可穿过常规污水处理流程；该来源仍为 `pending_spot_check`。[E009]
- **先前技术边界**：药物分子印迹吸附已是成熟路线，真实水体基质仍是难点。[E020]

### 1.3 静态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | DHPS 以空间相邻的 PABA/磺胺识别区域识别磺胺药物共有骨架。[E010] |
| 材料实现（转译假说） | 在亲水多孔凝胶中构建“双区域口袋”：区域 A 提供针对苯胺/磺酰胺向量的定向氢键，区域 B 提供针对异噁唑端的尺寸和疏水互补。用 SMX 模板限定两区域的相对距离，去模板后保留口袋。 |
| 预期功能 | 相比单一带电位点，同时读取 SMX 的磺酰胺连接方式与异噁唑端，从而提高其相对 PABA、磺胺类和 DOM 芳香位点的分配选择性。 |
| 测量 | SMX/磺胺嘧啶、SMX/磺胺甲基嘧啶、SMX/PABA 的选择性系数；等温滴定或微量热；固态/原位红外的氢键变化；真实二级出水的动态穿透次序。 |
| 因果对照 | 随机分布相同官能团但不印迹的凝胶；仅保留区域 A 或区域 B 的半口袋；用尺寸相近非 SMX 模板制备的错配口袋。 |

### 1.4 动态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | DHPS 的底物识别与催化涉及柔性区域重排；具体闭合证据需点检 E010 后确认。 |
| 材料实现（转译假说） | 在口袋入口设置一条柔性聚合物“盖片”，盖片携带区域 B；SMX 同时结合区域 A 和 B 时拉近盖片与孔壁，形成局部闭合。荧光供受体分别布置在盖片和孔壁，直接报告距离变化。 |
| 预期功能 | 开态允许快速进入，闭态减少已结合 SMX 被 DOM 或竞争物置换；温和改变 pH 或离子强度使两区域协同减弱并释放 SMX。释放触发条件属于待验证信息。 |
| 测量 | 时间分辨 FRET、凝胶局部溶胀/散射、结合和解吸速率、开闭循环后的选择性保持率。 |
| 因果对照 | 共价锁开盖片；去除盖片上的区域 B；长度不匹配、无法跨越口袋的短盖片。 |

### 1.5 可证伪预测

1. 只有双区域且可闭合材料应同时表现出 SMX 优先结合和 SMX 诱导的 FRET 增强；随机凝胶不应出现两者的相关性。
2. 锁开材料若仍保持与可闭合材料相同的竞争选择性，则“动态闭合提高选择性”的主张被否定。
3. 若 SMX、其他磺胺和 PABA 均产生相同闭合信号，则该结构只能识别共有骨架，不能支撑 SMX 选择性论文主张。

### 1.6 主要风险

- 动态生物依据尚未完成原文点检，是当前最大证据缺口。
- 分子印迹可能主导全部选择性，使“DHPS 仿生”沦为事后命名。
- pH 同时改变 SMX 形态和凝胶电荷，可能把响应性误判为配体诱导闭合。
- 二级出水 DOM 可能堵塞口袋入口并淹没分子识别效应。

| 因果闭环 | 16/20 | 双区域、锁开和半口袋对照可区分静态与动态贡献。 |
| 仿生保真度 | 11/15 | 复制识别与柔性闭合，但材料盖片是较远距离转译。 |
| 选择性机制 | 12/15 | 有真实竞争物设计，SMX 对其他磺胺的区分仍不确定。 |
| 原创性 | 11/15 | 动态盖片较新，但静态印迹路线拥挤。 |
| 证据质量 | 6/10 | 关键动态证据和场景证据尚待点检。 |

| **总分** | **77/100** | 当前不达最终通过线，需先补动态结构证据。 |

## 2. 双酚 A（BPA）：ERRγ 启发的双酚夹持与受体式入口收缩口袋

### 2.1 科学链条

BPA 必须与 BPS、BPF、单酚化合物和 DOM 芳香位点区分。普通疏水吸附或 BPA 分子印迹已经拥挤，本概念只在能够证明“双酚几何夹持”和“结合诱导入口收缩”均有独立贡献时成立。

### 2.2 真实生物原型与证据状态

- **生物原型**：人雌激素相关受体 γ（ERRγ）的配体结合口袋。
- **已筛查事实**：ERRγ 通过确定的受体相互作用识别 BPA；台账状态为 `screened_abstract`。[E011]
- **证据边界**：E011 未在当前台账中直接核实 BPA 诱导的宏观构象闭合。短名单明确把受体构象变化列为弱线索，因此动态生物特征必须在后续原文点检后才能升级为最终硬对应。
- **先前技术边界**：药物/有机微污染物的普通印迹路线成熟且真实水体受限。[E020]

### 2.3 静态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | ERRγ 配体口袋以明确的受体相互作用识别 BPA。[E011] |
| 材料实现（转译假说） | 在高交联亲水壳层中布置两个定向酚羟基受体和一个尺寸受限疏水腔，要求两个酚羟基的间距与中心异丙叉骨架同时匹配。 |
| 预期功能 | BPF 的桥连几何、BPS 的极性桥以及单酚化合物无法同时满足三点约束，因此 BPA 获得相对优先结合。 |
| 测量 | BPA/BPS、BPA/BPF、BPA/苯酚选择性系数；竞争吸附；核磁或红外结合位移；孔径探针与吸附热比较。 |
| 因果对照 | 删除一个酚羟基受体；扩大疏水腔；保持元素组成但随机分布位点；BPF 模板错配口袋。 |

### 2.4 动态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | 核受体配体结合域可通过构象重排改变口袋状态；BPA-ERRγ 是否具有足以支撑本材料映射的特定入口变化，当前为待点检信息。 |
| 材料实现（转译假说） | 将两个半口袋分别安装在同一柔性纳米囊壁的相邻“叶片”上。BPA 双点结合跨接两叶，使入口收缩；弱竞争洗脱液解除跨接后恢复开态。 |
| 预期功能 | 开态提高传质，闭态增强几何筛分并降低 DOM 置换。 |
| 测量 | 双位点荧光距离、单颗粒或小角散射尺寸、结合前后孔可达性、锁开与可动材料的传质和选择性差异。 |
| 因果对照 | 刚性锁开囊壁；只保留一个半口袋；使用不能跨接两叶的单酚配体。 |

### 2.5 可证伪预测

1. BPA 应同时产生双点结合信号和入口收缩，而 BPS、BPF 与单酚的响应幅度或稳定性应显著不同。
2. 若随机位点材料与几何口袋具有相同 BPA/BPS/BPF 选择性，则静态仿生主张失败。
3. 若锁开材料不损失真实二级出水中的选择性，则动态收缩不是必要机制。

### 2.6 主要风险

- 当前没有足够证据把 ERRγ 的 BPA 识别直接等同于强烈、可逆的入口闭合。
- BPA、BPS 和 BPF 的疏水性与酚羟基共有性可能使几何选择不够大。
- 设计容易退化为普通 BPA 印迹纳米囊；若动态对照不产生差异，应淘汰而非重新命名。
- DOM 芳香组分可能跨接或非特异收缩材料，产生假阳性构象信号。

| 维度 | 分数 | 理由 |
|---|---:|---|
| 科学正确性 | 10/15 | 静态识别有依据，特定动态对应缺证。 |
| 因果闭环 | 14/20 | 对照完整，但受体到纳米囊的跨度较大。 |
| 仿生保真度 | 8/15 | 静态口袋较可信，动态主要是设计类比。 |
| 选择性机制 | 10/15 | 三点识别可测试，但近似双酚区分难。 |
| 原创性 | 10/15 | 双叶动态高于普通印迹，仍有包装风险。 |
| 证据质量 | 5/10 | 仅有一个摘要层级受体来源，动态未核实。 |
| 可证伪性 | 9/10 | 锁开、半口袋和三种双酚竞争可直接裁决。 |
| **总分** | **66/100** | 不宜进入下一轮，除非补到直接构象证据。 |

## 3. 磷酸盐：PstS/PBP-1 启发的双叶磷酸根分子夹

### 3.1 科学链条

二级出水中磷酸盐处于硫酸盐、碳酸氢根、氯离子和 DOM 的高背景中。常规金属位点或正电位点容易表现为广谱阴离子亲和，因此本概念将选择性归因于“预组织多点氢键几何 + 磷酸根诱导双叶闭合”，并用半口袋和锁开对照分别切断两部分。

### 3.2 真实生物原型与证据状态

- **生物原型**：细菌 ABC 磷酸盐转运系统的周质磷酸盐结合蛋白 PstS/PBP-1。
- **场景事实**：氮、磷是污水二级出水深度处理的中心目标；台账状态为 `screened_review`。[E013]
- **代表性矩阵事实**：一项真实二级出水研究记录了硫酸盐、DOM、硝酸盐和磷共存；该矩阵仅作代表，不视为普适组成。[E014]
- **待点检结构线索**：PBP-1 以广泛氢键网络包围磷酸根并呈闭合构象。[E015]
- **待点检选择性线索**：PstS 类蛋白能强烈区分磷酸根与硫酸根、砷酸根；具体倍数在点检前不使用。[E016]

### 3.3 静态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | PstS/PBP-1 通过预组织、多点氢键网络包围磷酸根，而非仅依靠表面正电吸附。[E015][E016] |
| 材料实现（转译假说） | 在两个相对的亲水聚合物叶片内侧分别固定两组定向氢键供体和局部正电辅助位点；两侧合拢后形成匹配四面体磷酸根的分子夹。外层采用中性亲水多孔骨架以降低 DOM 非特异覆盖。 |
| 预期功能 | 只有空间配准的两半口袋才能形成协同结合；硫酸根、碳酸氢根和氯离子即使带电，也难以同时满足口袋几何和氢键取向。砷酸根区分仍须实测。 |
| 测量 | 磷酸盐/硫酸盐、磷酸盐/碳酸氢根、磷酸盐/砷酸根选择性系数；结合热与化学计量；原位红外/拉曼；固定床穿透顺序；二级出水中单位位点工作容量。 |
| 因果对照 | 仅保留单叶的半口袋；相同官能团随机分布材料；扩大或压缩两叶间距的几何错配材料；普通季铵盐树脂基准。 |

### 3.4 动态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | PstS/PBP-1 由两个结构域构成结合裂隙，并在结合态呈闭合构象。[E015] |
| 材料实现（转译假说） | 两个口袋叶片通过柔性但长度受限的铰链连接。磷酸根与两半口袋协同结合时诱导叶片靠近；在降低协同作用的温和再生条件下恢复开态并释放磷酸根。具体再生条件需实验筛选，不预设为生物事实。 |
| 预期功能 | 开态保持通量，闭态屏蔽已结合磷酸根并提升对竞争阴离子的动力学保持；再生时开态降低多点协同。 |
| 测量 | 叶片两端 FRET、原位小角散射或动态光散射、QCM-D 构象/质量耦合信号、结合与解吸动力学、至少五次开闭循环。 |
| 因果对照 | 共价锁开铰链；共价锁闭但保持口袋化学的材料；删除一侧氢键位点；铰链长度错配。 |

### 3.5 可证伪预测

1. 磷酸盐应产生浓度依赖的叶片闭合信号，且闭合阈值与协同结合区间一致；硫酸盐和碳酸氢根不应产生同等响应。
2. 半口袋与随机位点材料若仍达到同等竞争选择性，则预组织几何不是性能来源。
3. 锁开材料若在真实二级出水中保持与可动材料相同的选择性和抗置换能力，则动态闭合主张失败。
4. 若闭合可以发生但无法在可接受的温和条件下逆转，则该材料不满足可逆门控目标，只能算高亲和捕获材料。

### 3.6 主要风险

- 水会削弱人工氢键，人工分子夹可能达不到蛋白口袋的协同性。
- 高交联有利于几何保持但降低通量，柔性增加又会损失预组织优势。
- 砷酸根与磷酸根几何接近，不能从蛋白综述直接推定人工口袋仍保持区分。
- 叶片运动可能只是离子强度引起的整体溶胀，必须用非结合盐和锁开对照排除。

### 3.7 自评

| 维度 | 分数 | 理由 |
|---|---:|---|
| 科学正确性 | 14/15 | 蛋白原型与场景高度匹配，材料水相氢键仍有风险。 |
| 因果闭环 | 19/20 | 半口袋、随机位点、锁开、锁闭可逐一切断因果链。 |
| 仿生保真度 | 14/15 | 同时转译预组织口袋和双叶闭合。 |
| 选择性机制 | 14/15 | 直接面向真实主要竞争阴离子，砷酸根区分待验证。 |
| 原创性 | 13/15 | 有别于普通金属位点和离子交换，但人工分子夹并非无先例。 |
| 证据质量 | 7/10 | 关键 PstS 结构和选择性来源仍为待点检状态。 |
| 可证伪性 | 10/10 | 每个核心主张均有结构信号和拆分对照。 |
| **总分** | **91/100** | 概念最强；进入下一轮前先完成 E015/E016 原文点检。 |

## 4. 硝酸盐：NrtA 启发的平面阴离子 C-clamp

### 4.1 科学链条

硝酸盐在二级出水中与硫酸盐、氯离子、碳酸氢根和亚硝酸盐共存。普通强碱阴离子交换难以证明形状选择性；本概念利用平面硝酸根在双叶夹中的多点匹配，并要求硝酸根结合与夹体闭合同时发生。

### 4.2 真实生物原型与证据状态

- **生物原型**：蓝藻硝酸盐转运系统的周质结合蛋白 NrtA。
- **场景事实**：氮是二级出水深度处理的中心目标。[E013]
- **代表性矩阵事实**：真实二级出水研究记录了硫酸盐、DOM、硝酸盐和磷的共存。[E014]
- **待点检结构线索**：NrtA 具有 C-clamp 型硝酸盐/亚硝酸盐结合架构，并与向膜转运组件递交底物相关。[E017]
- **明确边界**：NrtA 对硝酸盐与亚硝酸盐的区分有限，不能把其描述为绝对硝酸盐专一受体。[E017]

### 4.3 静态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | NrtA 在两个叶片之间形成针对硝酸盐/亚硝酸盐的 C-clamp 型结合裂隙。[E017] |
| 材料实现（转译假说） | 在两片亲水纳米薄层内侧布置相向的酰胺/带正电氢键位点，闭合后形成偏好平面三角形阴离子的浅夹层；叶片外表面保持中性亲水以减少 DOM 覆盖。具体单体需通过计算和小分子结合实验筛选。 |
| 预期功能 | 平面几何与分散多点相互作用共同抑制四面体硫酸根、单原子氯离子和形态不同的碳酸氢根；预计无法完全排斥亚硝酸根。 |
| 测量 | 硝酸盐/硫酸盐、硝酸盐/氯离子、硝酸盐/碳酸氢根、硝酸盐/亚硝酸盐选择性；原位光谱；结合热；固定床穿透和再生富集液组成。 |
| 因果对照 | 单叶材料；随机位点材料；为四面体阴离子放大的错配夹层；普通强碱阴离子树脂。 |

### 4.4 动态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | NrtA 的双叶 C-clamp 架构与结合后的底物递交相关；构象变化幅度和顺序需原文点检。[E017] |
| 材料实现（转译假说） | 两片纳米薄层由柔性铰链连接，硝酸根的多点配位降低叶片间距并产生闭合；电荷屏蔽或温和竞争再生解除协同结合并打开夹层。 |
| 预期功能 | 开态提供快速通道，闭态延缓硝酸盐被高浓度共离子置换；再生开态有利于获得浓缩硝酸盐液。 |
| 测量 | FRET/小角散射叶片距离、选择性与闭合信号的同步性、锁开材料的解吸速率、连续柱循环。 |
| 因果对照 | 共价锁开铰链；铰链过长的无协同材料；保持电荷密度但移除一个叶片的材料。 |

### 4.5 可证伪预测

1. 硝酸盐和亚硝酸盐可能都诱导闭合，但硫酸盐、氯离子、碳酸氢根不应产生相同的闭合—结合耦合曲线。
2. 若硫酸盐在代表性二级出水浓度下优先闭合或优先穿透占位，则几何设计失败。
3. 若锁开材料与可动材料具有相同选择性和解吸行为，则动态门控没有独立贡献。
4. 若 NrtA 启发材料不能优于普通阴离子树脂的单位电荷选择性，则仿生复杂度没有工程正当性。

### 4.6 主要风险

- 水相中硝酸根脱溶剂化代价高，浅人工口袋可能只表现为普通静电交换。
- NrtA 同时识别硝酸盐/亚硝酸盐，论文不能承诺两者完全区分。
- 背景硫酸盐浓度可能很高，任何弱几何优势都会被质量作用压倒。
- 深度脱氮已有成熟生物路线，论文必须强调选择性回收或低碳源抛光，而非仅报告去除率。

## 5. As(III)：ArsR 启发的三硫醇诱导折叠捕获层

### 5.1 科学链条

As(III) 在中性水中的静电捕获困难，含硫位点还会受到 DOM、硫化物及亲硫金属竞争，并可能氧化失活。普通固定半胱氨酸材料已有先例，因此本概念不把“有硫醇”视为仿生，而要求三个硫醇的空间协同与 As(III) 诱导无序到有序转变同时成立。

### 5.2 真实生物原型与证据状态

- **生物原型**：AfArsR 类 As(III) 感知转录抑制蛋白的金属结合区。
- **待点检结构线索**：As(III) 诱导 ArsR 金属结合区发生无序到有序转变。[E018]
- **边界事实**：市政二级出水相关性是条件性的，必须以本地监测或明确工业混排场景为前提。[E018]
- **已筛查先前技术**：固定 L-cysteine 的静电纺丝膜已被作为 bio-inspired 砷吸附剂报道；单纯固定硫醇不具新颖性。[E019]
- **代表性矩阵**：E014 只支持二级出水存在复杂共存背景，不支持 As(III) 在该水样中必然出现。[E014]

### 5.3 静态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | ArsR 金属结合区以三个半胱氨酸硫配位形成 As(III) 识别环境；具体几何参数须点检 E018 后锁定。 |
| 材料实现（转译假说） | 将一条含三个定距半胱氨酸的短肽刷接枝到中性抗污多孔支撑体。序列和锚定位点使三硫醇只有在局部折叠时形成协同口袋，而不是随机高密度硫醇表面。 |
| 预期功能 | 三点协同提高 As(III) 相对单硫醇和非亲硫背景物的结合稳定性；亲硫金属仍可能竞争，必须实测而不能预设选择性。 |
| 测量 | As(III) 与 Hg(II)、Cu(II)、Cd(II)、硫化物和 DOM 共存时的选择性；XPS/吸收谱或其他可用配位表征；结合化学计量；位点归一化容量。 |
| 因果对照 | 三个 Cys 逐个替换为 Ser 的单突变和多突变肽；相同 Cys 数量但序列打乱的肽；随机固定 L-cysteine 膜基准。[E019] |

### 5.4 动态硬对应

| 项目 | 内容 |
|---|---|
| 生物特征 | As(III) 诱导 AfArsR 金属结合区域从无序转变为有序。[E018] |
| 材料实现（转译假说） | 三半胱氨酸肽刷在无 As(III) 时保持松散可达；As(III) 同时配位三个位点后形成局部折叠。肽两端布置荧光供受体，使折叠信号与吸附同步测量。采用温和竞争配体或受控氧化还原条件解离并恢复开态；再生方案需先证明不会损伤硫醇。 |
| 预期功能 | 开态提高 As(III) 进入位点的速率，折叠态屏蔽配位中心并降低竞争置换；解折叠实现释放。 |
| 测量 | FRET、圆二色谱或局部构象光谱；As(III) 结合与构象变化的剂量响应；竞争置换；硫醇氧化状态；多次折叠—解折叠循环。 |
| 因果对照 | Cys-to-Ser 突变；共价锁开肽；预组织刚性三硫醇但不能折叠的材料；无 As(III) 的离子强度和 pH 对照。 |

### 5.5 可证伪预测

1. As(III) 吸附应与肽折叠信号共享剂量响应区间；Cys-to-Ser 突变应同时削弱结合和折叠。
2. 随机 L-cysteine 膜若在位点归一化后达到同等选择性，则三点几何口袋没有独立贡献。
3. 锁开肽若仍保持相同抗竞争置换能力，则无序到有序转变不是选择性来源。
4. 若 Hg(II)、Cu(II) 或硫化物诱导相同折叠并占据位点，则该概念不能宣称 As(III) 选择性。

### 5.6 主要风险

- 普通市政二级出水的 As(III) 载荷可能不足，导致工程场景不成立。
- 硫醇氧化、二硫键形成和金属竞争会同时降低容量和造成非 As(III) 构象响应。
- 肽材料成本、蛋白酶稳定性和循环再生可能弱于普通无机吸附剂。
- 需要证明折叠贡献，而非重复已有固定半胱氨酸吸附路线。[E019]
