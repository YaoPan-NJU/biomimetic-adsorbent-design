# 关键主张复核记录

复核日期：2026-07-16

## 结论

十条关键主张中，5条得到直接支持，5条需要缩窄表述，没有发现完全不受来源支持的主张。以下边界已同步回 `evidence_ledger.csv`，后续设计者只能使用收窄后的表述。

| 证据 | 判定 | 可安全使用的范围 | 禁止外推 |
|---|---|---|---|
| E001 | supported | 9座美国市政WWTP最终出水中PFOA、PFOS、PFBS、PFHxS检出率100%，HFPO-DA为67%；短链PFCA升高与前体转化一致 | 不外推为所有污水厂的固定检出率 |
| E002 | supported | 分子模拟提示Arg122对hL-FABP结合PFOA的重要性 | 不能写成突变结合实验或现成动态材料机制 |
| E005 | partial | 瑞士研究夏季处理后WWTP出水克拉霉素57–330 ng/L | 未核实工艺终点前不得写“二级出水” |
| E008 | partial | SERT为Na⁺/Cl⁻依赖转运体，2016年结构显示外向开放态和中央结合位点 | 单篇文献不能证明完整构象循环，也不证明ODV特异结合 |
| E009 | partial | 22座WWTP中SMX、SPY及乙酰代谢物高频检出；SMX对藻类HQ>10 | 不把SMX的风险结论转移给AcSMX |
| E015 | supported | 结合磷酸盐的PBP-1闭合并完全包埋磷酸盐，9个残基形成14个氢键 | 精确数值只适用于该蛋白 |
| E016 | partial | PstS/PBP对砷酸盐至少约500倍、对硫酸盐可达约10⁵量级的区分已有报道 | 不写成所有PstS固定常数；定量句需配原始研究 |
| E017 | partial | 结合态NrtA为双结构域C-clamp，硝酸盐完全包埋；作者提出C端可能参与运输相关构象变化 | 不声称已有apo/open与bound/closed成对结构 |
| E018 | supported | 覆盖AfArsR结合区的模型肽受As(III)诱导由无规卷曲转为有序AsS₃复合物 | 不外推为完整蛋白全构象实验 |
| E012 | supported | 特定4-乙烯基苯甲酸单体CBZ印迹材料容量28.40 mg/g、印迹因子2.72 | 不当作CBZ印迹材料类别的普遍容量 |

## 对设计的直接约束

1. PstS/PBP-1可以承担磷酸盐方案的静态硬对应和闭合态生物依据，但人工材料的可逆开闭仍是待证假说。
2. NrtA只足以支撑“结合态夹钳几何”和运输相关构象线索；若人工材料主张动态闭合，必须由材料自身的原位表征和锁闭对照证明。
3. SERT只作为离子依赖、外向开放和中央口袋的启发来源；不得声称ODV已被SERT结构直接验证。
4. hL-FABP的PFOA证据主要是结合与模拟，不足以独立支撑动态门控。PFOA方案若进入终审，必须补充更直接的口袋开闭依据或降低动态仿生表述。
5. 场景证据和材料机理证据分开记账。污染物在处理后出水检出，不等于已证明在本项目目标二级出水中达到同一浓度。
