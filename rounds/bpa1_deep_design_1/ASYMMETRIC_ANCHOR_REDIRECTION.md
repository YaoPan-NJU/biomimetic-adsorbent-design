# BPA-1 非对称主锚重定向

## 结论

`9.27 Å` 对称双端夹持不再是 BPA-1 的强制机制。`2E2R` 及突变证据更直接支持的是：BPA 的一个酚羟基由互补的供氢/受氢作用构成主锚，疏水口袋包围芳香骨架；另一端羟基可提供附加作用，但不是所有强结合类似物的必要条件。

因此，BPA-1 从“跨越两个羟基的刚性对称夹”改写为“单端互补极性主锚 + 疏水芳香微环境 + 可选第二端判别”。这仍是 ERRγ 启发，但只转译静态化学功能，不转译蛋白、肽、折叠或动态门。

## 为什么必须重定向

- ERRγ 突变研究把 Glu275/Arg316 对同一个酚羟基的作用列为 BPA 捕获的关键因素。
- 只有一个酚羟基的 4-α-cumylphenol 仍能与 ERRγ 强结合，说明第二个羟基不是高亲和作用的绝对前提。
- 原 R1 路线只匹配两个杂原子的距离，没有匹配氢键方向；继续寻找一个低分子、跨 `9.27 Å` 的完全对称夹会把未经证实的设计假设误当成生物原型要求。

## 精确低分子机制探针 P1-MAPy

- 名称：2-methacrylamidopyridine / N-(pyridin-2-yl)methacrylamide
- 简写：`P1-MAPy`
- 分子式：`C9H10N2O`
- 分子量：`162.19 g/mol`
- 结构表示：`CC(=C)C(=O)NC1=CC=CC=N1`
- 功能分工：邻位吡啶氮为氢键受体，酰胺 `N-H/C=O` 提供相邻供体–受体环境；甲基丙烯酰双键是聚合出口，不需要季铵化或消耗吡啶位点。

文献已证明 MAPy 可由 2-aminopyridine 与 methacryloyl chloride 制备，并可自由基均聚；另有工作把 MAPy 以 `6–40 mol%` 与 styrene 共聚。以上只证明合成和材料化入口，不证明 MAPy 能在水中选择性结合 BPA。

## 质量容量边界

若一个 MAPy 位点最多结合一个 BPA，则纯位点的理论质量上限为：

`q_site = 228.29 / 162.19 × 1000 = 1407.6 mg BPA/g MAPy`

对任意最终干材料，尚未扣除不可达位点的总材料上限为：

`q_total,theory = 1407.6 × w_MAPy mg BPA/g dry composite`

其中 `w_MAPy` 是最终干材料中 MAPy 衍生单元的质量分数。例如 20、30、40 wt% 分别只对应 282、422、563 mg/g 的理论上限，实际值必然更低。后续任何材料方案必须报告这个总干材料口径，不得用“每克受体”掩盖交联剂、孔壁或载体质量，也不得加入只增加重量而不提供孔结构、传质或吸附作用的惰性重载体。

这条约束不指定树脂、活性炭、碳材料或其他材料类别；它只要求所有组成质量进入分母，并说明每一部分的功能。

## 当前不能越过的风险

1. **水竞争：** 目前没有 BPA–MAPy 在中性水中的直接结合常数或化学计量证据。
2. **自缔合：** MAPy 是供体–受体单体，已有聚合物研究利用其自缔合；这可能预占 BPA 位点。
3. **选择性未闭合：** 单端主锚可能只提高一般酚类亲和力。若 BPA/BPF/BPS/4-cumylphenol 的顺序仅随疏水性变化，ERRγ 几何选择性主张失败。
4. **孔结构未设计：** MAPy 可聚合不等于含 MAPy 的珠粒一定具有可达微孔或床层强度。
5. **同系对照仍需冻结：** 受体全体、供体删除、受体删除、邻位关系破坏和纯疏水骨架必须尽量在相近组成与孔结构下比较。

## 下一道门

`P1-MAPy` 当前冻结为确切的**机制探针候选**，不是已验证受体，更不是最终材料。下一步只做以下纸面/计算与可溶验证合同设计：

1. 检查 MAPy 与 BPA 单端酚羟基能否形成方向合理的互补作用，而不是只形成单一 O-H···N 接触；
2. 冻结尽量等组成的供体删除、受体删除和位置异构对照；
3. 估算在水中可辨识的信号与自缔合竞争，给出停止阈值；
4. 只有该增量可被实验区分，才比较自支撑多孔聚合物、活性炭/碳孔壁接枝或其他轻质高有效质量架构。

当前不冻结配方、载体、SOP、采购或实验执行。
