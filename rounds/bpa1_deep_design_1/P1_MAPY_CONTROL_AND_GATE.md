# P1-MAPy 对照系列与可溶门合同

## 结论

`P1-MAPy` 可以进入机制验证设计，但尚不能进入材料配方设计。当前冻结四个层级的对照，把“吡啶受体、酰胺供体、二者邻位关系、一般芳香疏水作用”拆开。完整探针只有在无聚集的中性水中同时超过这些对照，BPA-1 才保留“互补单端主锚”主张。

## 冻结分子组

| 编号 | 确切结构 | 分子式 / MW | 删除或保留的功能 | 证据边界 |
|---|---|---:|---|---|
| `P1` | 2-methacrylamidopyridine；`CC(=C)C(=O)NC1=CC=CC=N1` | `C9H10N2O` / 162.19 | 完整的邻位吡啶–酰胺供受体组合 | 已有合成、均聚及 styrene 共聚；无 BPA 水相结合数据 |
| `C1-NHoff` | N-methyl-N-(2-pyridyl)acrylamide；`C=CC(=O)N(C)C1=CC=CC=N1` | `C9H10N2O` / 162.19 | 甲基封闭酰胺 `N-H`，保留吡啶氮、羰基、芳环和同分子式 | 确切化合物/CAS `1156926-11-2` 可追溯；聚合反应性与 P1 不视为相同 |
| `C2-Noff` | N-phenylmethacrylamide；`CC(=C)C(=O)NC1=CC=CC=C1` | `C10H11NO` / 161.20 | 用 `CH` 替代吡啶 `N`，保留酰胺 `N-H/C=O`、芳环和甲基丙烯酰基 | 已有自由基聚合证据；是近等质量受体删除对照 |
| `C3-meta` | N-(pyridin-3-yl)methacrylamide | `C9H10N2O` / 162.19 | 保留同样官能团和分子式，只破坏吡啶氮与酰胺的邻位关系 | 已有从 N-methacryloyl-3-aminopyridine 自由基制备线性/交联聚合物的证据 |
| `C4-H` | styrene | `C8H8` / 104.15 | 只保留一般芳香/疏水作用 | 只作疏水空白，不用于单独证明某一氢键因果 |

`C1-NHoff` 是可溶层最接近的供体删除体，但其 acrylamide 与 P1 的 methacrylamide 双键位置分配不同；因此它不能单独承担最终聚合物的等反应性对照。进入材料层时，必须测定实际共聚组成并按实测位点含量归一化，不能按投料比假定相同。

## 第一段：先排除假结合

条件以 `20–25 °C`、`pH 7.0 ± 0.1` 为中心，所有样品采用相同缓冲液和有机助溶剂比例；有机助溶剂优先为零，确需使用时上限为 `0.5 vol%`，且不得用提高助溶剂比例挽救失败。

1. 测 P1/C1/C2/C3 在 `0.1–5 mM` 的表观溶解度、pH/UV 滴定和浓度依赖谱图；先得到可用的质子化窗口。
2. 用浓度依赖 `1H NMR` 检查 P1 自缔合；拟合时不能把二聚导致的位移误作 BPA 结合。
3. 在最高工作浓度检查浊度、过滤前后浓度和扩散/粒径异常。出现可见沉淀或过滤损失超过 `10%` 即不进入结合拟合。
4. BPA 工作浓度不得超过实测水相溶解度；所有空白包含完全相同的瓶壁、时间和过滤步骤，排除 BPA 对容器的非特异损失。

## 第二段：正交结合合同

至少使用两种原理不同的方法：

- `1H NMR` 化学位移滴定，优先观察 BPA 芳香区和探针吡啶/酰胺区，并把探针自缔合写入全局模型；
- 差示 UV 或荧光滴定作为高灵敏度读出，但必须用 C1/C2/C3/C4 排除单纯折射率、内滤和疏水缔合；
- 若热信号足够，可用 ITC 作为第二或第三方法；ESI-MS 只能证明气相离子存在，不能作为水相结合常数证据。

预注册的通过条件：

1. 两种正交方法支持相同的主要化学计量，`Ka` 相差不超过 3 倍；
2. `Ka(P1–BPA) >= 100 M-1`，并且 P1 至少比 `C1-NHoff`、`C2-Noff` 和 `C3-meta` 中的最高值高 3 倍；
3. 全模型必须优于“探针自缔合 + BPA 非特异分配”的空模型，且残差不呈系统性；
4. 盲样中能区分 P1–BPA、P1 空白和最强对照–BPA；
5. 同一增量在低离子强度水成立后，再进入二级出水稀释系列；不能先靠疏水基质把弱单体信号放大后倒推主锚成立。

任一硬条件失败，`P1-MAPy` 终止为“ERRγ 主锚的确切化学实现”；可以保留为普通含氮功能单体对照，但不得继续用 BPA-1 的仿生选择性叙事。

## 第三段：同系物机制指纹

首轮不要求 P1 单体已经表现最终材料的全部选择性，但必须产生可解释的指纹：

- `phenol`：只保留单酚主锚、缺少 BPA 的第二芳环；
- `4-alpha-cumylphenol`：保留 ERRγ 已知的单羟基强结合轮廓；
- `BPF/BPS`：保留双酚而改变桥连基和整体疏水/构象；
- `BPA`：目标。

若 P1 与上述分子的表观作用只按疏水性单调变化，且不优于 C1/C2/C3，则只能声称一般分配，不能声称互补单端主锚。真正的 BPA/BPF/BPS 选择性仍需在后续孔壁几何和真实二级出水中建立。

## 材料化前的容量纪律

即使 P1 通过可溶门，也不自动选择聚合物树脂。后续比较自支撑多孔聚合物、碳孔壁接枝或其他架构时统一使用：

`q_total,theory = 1407.6 × w_P1 mg BPA/g dry material`

并进一步乘以实测共聚/接枝含量和可达率。任何声称的选择性收益必须与 `C1/C2/C3/C4` 的等孔容、等表面积或成对校正版本比较；不能让孔结构差异替代主锚因果。
