# 一、设计者 r2 修订（逐条闭合，仅记增量，未改部分沿用 r1）

## 闭合 a1（HIGH → resolved）：合成前 DFT-SAPT 预筛门（Stage −1 go/no-go）
在四套架构任何合成投入之前，新增 **Stage −1 计算预筛门**（低成本、单点否决）：
- 方法：对 D/N/A 三成员孔壁片段模型与 OC 核、EHMC 核，分别在 ωB97X-D/def2-TZVP 下优化并做 DFT-SAPT（或 ALMO-EDA）能量分解，取 CT+诱导分量。
- go 判据（须同时满足）：三成员对 OC 与 EHMC 的 CT+诱导分量差 |ΔΔ(CT+ind)| ≥ ~RT（298 K 约 2.5 kJ/mol），且符号序正确（D 端偏向 OC、A 端偏向 EHMC 或至少显著减弱）。
- no-go：若 |ΔΔ(CT+ind)| < ~RT 或符号序错乱，本角度在合成前即降优先级或终止，残余价值转交 A14（几何协同）与 A19（预浓缩）。
- 定位更新：DFT-SAPT 由 r1 的"假设精制、非独立证据"升为**合成前决策门 + 归属支柱之一**（与 d1 联动）。此门把 r1 最大失败风险（水相中性弱极性客体 CT 幅度不足）前移到不花钱的计算阶段，避免盲目合成四套框架。

## 闭合 a2（HIGH → resolved）：实测电子学描述符替代跨语境 σp，反转降为加分项
- 承认 EHMC 为**推-拉发色团**（对甲氧基为供体、丙烯酸酯端为受体），故"OC 受体/EHMC 供体"的干净二分不成立；r1 的 σp(CN)=+0.66 vs σp(OMe)=−0.27 属跨分子语境（氰基在烯键、甲氧基在苯环），不可直接相减。
- 判别量改为**核整体电子亲和/LUMO 能级**：OC 的 α-氰基-二苯基丙烯酸酯核因额外氰基，LUMO 更低、电子亲和更高，作为 π 受体强于 EHMC 的对甲氧基肉桂酸酯核。该量以**实测 + 计算双通道**给出：(i) 循环伏安还原电位/起始还原电位（受体强度）与氧化电位（供体强度）；(ii) DFT LUMO/EA、HOMO/IE。孔壁供体强度同样以框架薄膜 CV 起始氧化电位 + UPS 实测（沿用 r1 S3）。
- **必要信号重定义**（主判据）：α(OC/EHMC) 随实测孔壁供体强度单调升高（D>N），且该单调性与孔壁疏水性**解耦**——以逐成员水接触角/水蒸气吸附为协变量回归、并以 OC/HHCB logKow 相似对独立扣除疏水基线（见 b2/a2 联动）。
- **加分信号（降级，非必要）**：A 端方向反转 α(OC/EHMC)<1。r1 把反转当作"最强因果信号"，但其立于不成立的干净二分之上，故降为加分项；退路判据 α(D)>α(N)>α(A) 显式要求"扣除壁面极性/亲疏水单调趋势后仍成立"，否则只计为疏水协变量效应、不归 CT。
- S-PASS 重构：原 S-PASS-2（反转为必要）拆为 S-PASS-2a（供体强度单调、疏水解耦后成立，必要）与 S-PASS-2b（A 端反转，加分）。

## 闭合 d1（MEDIUM → resolved）：SAPT 升为与谱学并列的归属支柱
DFT-SAPT 的 CT/诱导分量分解由"非独立证据"升为**与谱学证据链并列的归属支柱**，用于拆开 CT（A01）与 CN 偶极（A02）：谱学链（DRS CT 带 / 荧光淬灭 / XPS N1s）证明"有电子性相互作用且随壁面电子密度变化"，SAPT 分解证明"该相互作用的 CT+诱导分量而非纯静电偶极占主导"。C9 归属规则更新为：谱学≥2 项 + SAPT 分解 CT/诱导分量显著，二者并列。概念上增设"保留 CN 偶极但抬高核 LUMO 的类似物"作为两维拆分的计算对照（合成困难，列为计算优先）。

## 闭合 b1（MEDIUM → resolved）：联苯胺类 CMR 标注 + 非联苯胺受体二胺升为首选
- 明确标注：联苯胺为 **IARC 1 类**确证人类致癌物（膀胱癌），3,3'-二硝基联苯胺、四氟联苯胺为同构型 CMR 危害物。
- A 成员（受体端壁面）**首选改为非联苯胺电子贫二胺**：4,7-二氨基-2,1,3-苯并噻二唑（苯并噻二唑受体）或三嗪二胺类（与 r1 已引 CTF-1 三嗪受体平台 E-F4 同族），均为成熟电子受体构筑块、无联苯胺骨架。联苯胺/硝基联苯胺仅作密闭手套箱条件下的备选对照。
- SOP S0 写入防护等级（密闭操作、废物流管理）。D 成员（供体端）2,5-二甲氧基联苯胺同理标注并优先以非联苯胺烷氧基供体二胺替代（如 2,5-二甲氧基对苯二胺——苯环单环、非联苯胺）。

## 闭合 b2 / W2（MEDIUM → resolved）：逐成员结构核验 + 第三独立格式 + 疏水协变量
- 逐成员报告 PXRD/BET/孔径；若 D/N/A 拓扑发散、孔径不可比，则**按带结构协变量的定性趋势处理，不作干净电子归因**，并在正文如实降级。
- 增设**第三独立验证格式**：同一刚性骨架上 D/N/A 取代的**离散分子笼/大环**（区别于 COF 周期性框架与柱芳烃小腔），提供孔径解耦的电子梯度对照。
- 疏水协变量：逐成员水接触角、水蒸气吸附等温线随电子取代基的变化一并报告，作为电子归因回归的协变量（与 a2 疏水解耦联动）。

## 闭合 g1（MEDIUM → resolved）：ng/L 床层显式依赖 A19 + 二次废液量化
- ng/L 床层性能显式标注为**依赖 A19 预浓缩协作**；本角度床层判据（S-PASS-6）限定为 µg/L 机理工况，ng/L 另设**短接触时间/动力学床层预筛**（EBCT 梯度）判断构型扩散限制程度。
- 再生二次废液量化：极性切换洗脱的溶剂/处理水体积比、洗脱液 OC 与二苯甲酮浓缩倍数、单位处理水的溶剂消耗与二次废液体积，纳入 S7 记录与工程判据。

## 闭合 c1（MEDIUM → resolved）：H2 降为一般刚性使能论证
仿生对应矩阵**硬对应缩为 2 条**（H1 电子互补供体面预组织 + H3 真黑素/柱芳烃富电子表面隔离电子亏缺客体）。原 H2（Moser-Dutton 距离标度律 → 刚性固定间距）**降级为"一般刚性预组织使能论证"**，不作硬对应宣称；正文讨论如实呈现 RC 功能为光驱电荷分离（非吸附）、其在本设计中主要承担叙事与"刚性预组织重要性"的一般论证，设计原则的可操作来源实为成熟超分子化学（Hunter-Sanders / CB[8] / 柱芳烃）。

## 闭合 a3（LOW-MEDIUM → 部分 resolved）：承重 kO3 全文外部核验
Hopkins 2017（10.1016/j.jhazmat.2017.05.016）经 PubMed（PMID 28535480）摘要外部核验，二级速率常数确证：**OMC（EHMC）5.25×10⁴、OC 1.58、OXY(中性) 3.80×10² M⁻¹s⁻¹**，"电子亏缺致亲电臭氧进攻钝化、OC 较 EHMC 低约 3.3×10⁴ 倍"的疼痛-机制同源论证定量成立，E-B2 就此三数值升为 verified。σp(CN)=+0.66/σp(OMe)=−0.27 为 Hansch-Leo-Taft（Chem Rev 1991, 91, 165, E-D3）标准列表值，按标准常数引用；因 a2 已改用实测电子学描述符作主判据，σp 仅作背景说明，其"跨语境"缺陷不再承重。TTF 起始氧化电位、Moser 距离常数仍为综述典型值，标注定性、入正文前需全文核定（残余，见 r3）。

## 闭合 a4 / a5（LOW → resolved）
- a4：在 100/500/2000 ng/L 柱工况（S6）**补报 α(OC/EHMC)**，与 µg/L 批次 α 并列，论证或证伪浓度迁移性；选择性主判据不再仅定义于 µg/L。
- a5：C5 增设 **DOM 富电子芳组分位点竞争对照**（腐殖酸/富里酸芳组分分级 + 竞争性占据 CT 位点的机制臂），补齐原"孔堵/增溶"二分之外的第三机制维度。

## 残余（转 r3，均非 critical/high）
- 证据层级：多数辅助引用仍为 abstract_read/metadata_verified，仅 Kroll 2025、Medici 2022、Hopkins 2017（本轮）为全文/摘要外部核验级；入正文前需完成 TTF 氧化电位、Moser 常数、Zhu 2004 EDA 参数全文核定。
- 原创性口径：一般 CT 捕获（碘/硝基芳烃）主题已占用，增量收窄为"首例中性高疏水水污染物/首例紫外过滤剂 + 电子梯度同拓扑因果系列"，论文措辞禁称"CT 水相捕获全球首例"。

## 总判定
r2 未发现致命打击，**两项 HIGH（a1、a2）均实质闭合、无新增 critical/high**。方案因果链更紧（预筛门 + 实测电子描述符 + kO3 verified）、归属更干净（SAPT 并列支柱）、安全与工程边界诚实。残余为证据层级（多数 abstract 级）、原创增量收窄、离散笼可得性等 LOW/LOW-MEDIUM 项。就"是否达 85 通过线"：主选择性主张（水相中性弱极性客体可测 CT 增量）仍为未经实验证实的科学赌注，方案以预筛门 + 可证伪判据诚实前置，但**在纸面设计层，其原创增量与证据成熟度尚不足以支撑 85**，建议 revise 进 r3。
