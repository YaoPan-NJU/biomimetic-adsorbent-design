## 监督者裁决（2026-07-22）

PCB-209（pcb209）Phase B 首个方案槽（本会话本地批次）。核心思路：PCB-209 十氯全取代、四邻位使双苯环锁定约 85–90° 螺旋桨扭转构象。自然界的 AhR PAS-B 平面疏水腔**几何排斥**非平面 PCB（非平面 PCB 非 AhR 配体），而 PXR/CAR 大柔性腔**容纳**非共平面 PCB（Al-Salman 2012 已核验：非共平面 PCB 153/180/194 直接激动 PXR/CAR，共平面 PCB-77 不激活）。把这一"平面腔判别扭转构象"的自然几何原理**反转**为人工刚性扭转互补腔：接纳扭转 PCB-209、几何排斥共平面竞争物（TCDD、PCB-77/126/169、平面 PAH）。选择性来自扭转构象互补，正交于 logKow。

r1 判 64/revise，三项 high：(1)**入场门禁/靶标合理性**——PCB-209 无实测水浓度、且为非平面故非二噁英样（毒性远低于共平面 PCB），"专门去除 PCB-209 而排斥更毒的共平面 PCB"方向存疑；(2)**扭转构象识别已有先例**（Weh 2023 Nat Commun 非平面轴手性联苯分子识别），原创性须收窄为水相 POP 判别；(3)**logKow 单调陷阱**——须证明扭转选择性超越疏水分配，且能区分 PCB-209 与九氯近异构体（体积/扭转近同）。三项均 high、非 critical；重建方向明确（把靶标重构为"高氯 PCB 类 + 共平面/非共平面双读出"或论证 PCB-209 为持久性示踪），故 revise。

## 一、科学问题与工程难题

- **科学问题（SP）**：PCB-209 中性、无氢键供受体、无解离基团、零 C-H，唯一独特维度是四邻位锁定的扭转螺旋桨构象。能否以**扭转二面角轮廓互补**作为不可被 logKow 还原的识别维度，在水相区分扭转 PCB-209 与共平面卤代芳烃？
- **工程难题（EC）**：水相 PCB 由低/中氯同系物主导，PCB-209 单体溶解相仅 pg/L–亚 ng/L 量级且无实测（只能由 ΣPCB 量级 + 单体份额推断，不得声称实测）。传统黑碳/GAC 对平面芳烃 π 面吸附强，恰对扭转 PCB-209 弱（Jonker & Koelmans 机制），且随 logKow 单调、无同系物/构象选择性。
- **闭环主张（收窄版）**：扭转互补腔在水相对 PCB-209 相对共平面竞争物（PCB-77/126/169、平面 PAH、TCDD）产生可测滞留差异，经刚性平面腔对照证伪；选择性来自几何而非 logKow。**注意：本主张不声称区分 PCB-209 与九氯近异构体**（二者扭转/体积近同）。

## 二、疼痛-机制同源

PCB-209 难以被现有平面 π 吸附剂捕获（扭转构象削弱平面堆叠），恰是其可被"扭转互补腔"选择性识别的根源：同一扭转性质既是常规吸附失效的原因，又是几何识别的把手。自然界 AhR 平面腔对非平面 PCB 的排斥（Safe 1994；Van den Berg 2006 WHO-TEF 体系将非平面 PCB 归零 TEF）是这一几何判别力的进化实证。

## 三、生物原型与硬对应矩阵

**硬对应 H1（承重，几何级）· 负面原型反转**
- biological_feature：AhR PAS-B 平面疏水腔以平面性为准入几何，**排斥**非平面/邻位取代 PCB；PXR/CAR 大柔性腔则**容纳**非共平面 PCB（构象-受体判别的双向实证）。
- source_support：Al-Salman & Plant 2012 *Toxicol Appl Pharmacol*（10.1016/j.taap.2012.05.016，被引 132，**verified**："non-coplanar PCBs directly activated PXR and CAR, whilst the coplanar PCB77 did not"）；Watkins et al. 2001 *Science* 人 PXR 腔约 1290 Å³ 柔性疏水腔（**tier: primary，简报级引，待复核体积**）；AhR 平面配体需求（Safe 1994，**tier: review-lead**）。
- material_realization：机制驱动（非模板）刚性扭转互补 cavitand/笼（间苯二酚芳烃 / HCP / COF），腔壁按邻四氯联苯约 85–90° 扭转轮廓预组织。
- expected_function：接纳扭转 PCB-209，几何排斥共平面竞争物。
- measurement：α(PCB-209 / PCB-77)、α(PCB-209 / TCDD)、α(PCB-209 / 芘)；配刚性**平面腔对照**材料。
- causal_control：刚性平面腔变体应丧失 PCB-209 摄取 / 反获共平面竞争物摄取（几何反转对照）。

**硬对应 H2（体积窗口，使能）**
- biological_feature：lipocalin/PXR 有限体积疏水腔（可饱和，区别于无限脂质/黑碳分配）。
- source_support：lipocalin 刚性疏水 calyx（Godovac-Zimmermann 1988，**tier: literature-lead**）；组织富集启发式对 PCB-209 判**不够格**（简报），故此维仅作体积上限约束，不作富集主张。
- material_realization：腔体体积受控刚性骨架（PCB-209 扭转轮廓约 1.0 nm 量级）。
- expected_function：可饱和 Langmuir 型高亲和，尺寸排斥更大 NOM 组分。
- measurement：饱和等温线 + **脂质/有机质归一化对照**（区分受限腔与无限分配）。
- causal_control：放宽腔体积变体应回落至非饱和分配。

## 四、材料架构与选择性机制

- **人工识别单元**（非蛋白）：机制驱动刚性扭转互补腔（间苯二酚芳烃 cavitand / 超交联 HCP / COF），非分子印迹（规避十氯巨分子模板残留）。
- **选择性分层**：L1 疏水预浓缩（前置、非选择性，logKow~8.2）；L2 扭转构象互补（选择性来源，主判据）；L3 体积窗口（使能，尺寸排斥）。
- **容量口径**：mg PCB-209/g 总干复合材料 + packed-bed mg/mL + 可及腔密度 + 空白骨架分配本底 + 脂质归一化对照。

## 五、r1 攻击与裁决

**攻击者 r1（隔离）· 7 维打击**
- A1（HIGH）：**靶标合理性与入场门禁**。PCB-209 无实测水浓度、非平面故非二噁英样、毒性远低于共平面 dl-PCB。设计一个"接纳非毒 PCB-209、排斥更毒共平面 PCB"的材料，处理价值方向存疑。须重构主张：或以 PCB-209 为"最高氯持久性示踪/最难捕获极端案例"论证科学价值，或把靶标扩为高氯 PCB 类并设共平面/非共平面**双读出**（同时能捕获毒性共平面者）。
- A2（HIGH）：**原创性被 Weh 2023 收窄**。非平面/轴手性联苯的分子识别已实证（Weh et al. 2023 Nat Commun 10.1038/s41467-023-35851-3，**verified**）；扭转构象作识别维度非零先例。增量须收窄为"水相 POP 的扭转-共平面判别吸附剂"，措辞禁称"扭转识别首例"。
- A3（HIGH）：**logKow 单调 + 近异构体不可分**。须证明扭转选择性超越疏水分配（平面腔对照必设），且诚实声明**不能**区分 PCB-209 与九氯近异构体（扭转/体积近同）——若环境目标是"最高氯代者"，此不可分性削弱主张。
- A4（MEDIUM）：Al-Salman 面板不含 PCB-209，容纳性属外推须显式声明；PCB-209 二面角具体值（Pedersen 1975）待全文核验。
- A5（MEDIUM）：Å 级扭转轮廓预组织的合成可达性；间苯二酚芳烃腔径与扭转轮廓匹配须建模前置。
- A6（MEDIUM）：pg/L–亚 ng/L 溶解相下达 Kd≥10⁴ 的可行性本身即科学问题。
- A7（LOW）：水相 PCB-209 捕集材料零先例（G1，简报），先例风险低，但为空白亦意味无正面可行性佐证。

## 六、r2 重建方向（带台账）

1. **重构靶标/主张（A1）**：三选一——(a) 论证 PCB-209 为"最高氯代最难捕获极端案例 + 持久性示踪"的科学价值；(b) 扩靶为高氯 PCB 类，设共平面/非共平面双读出腔（同时捕获毒性共平面 dl-PCB）；(c) 若二者均不成立，则本角度按靶标合理性终止。
2. **收窄原创口径（A2）**：定义增量为"水相 POP 扭转-共平面几何判别吸附剂 + 平面腔反转因果系列"，引 Weh 2023 界定边界。
3. **前置 logKow 解耦（A3）**：平面腔对照 + 脂质归一化 + 共平面/扭转竞争对为 make-or-break；诚实声明近异构体不可分。
4. **建模前置**：PCB-209 二面角（Pedersen 1975 全文核验）与腔壁扭转轮廓匹配的分子建模 go/no-go。

## 七、成功与失败判据（实验级，摘要）

- **通过（须全部）**：α(PCB-209/PCB-77)≥3 且 α(PCB-209/TCDD)≥3；刚性平面腔对照材料对 PCB-209 摄取回落 >60%（几何反转成立）；饱和等温线 + 脂质归一化证明可饱和受限腔而非无限分配；全谱 α 与 logKow 非线性（R²<0.8）；靶标合理性主张（§六.1）经论证成立。
- **失败（任一）**：平面腔对照与扭转腔对 PCB-209 摄取等价（几何非承重）；全谱 α 与 logKow 线性 R²>0.9（纯疏水，命中 T1）；无法区分共平面/扭转竞争物；靶标合理性三条均不成立。

## 八、残余与后续（不阻断本轮记录）

- PCB-209 二面角具体值、PXR 腔体积 1290 Å³ 待全文核验升 verified。
- 与 A02（最大体积受限腔）、A03（C-Cl σ 穴卤键阵列）区分与叠加：本 A01 承担扭转几何维，A03 承担卤键阵列维，二者正交乘积见 A23。
- 无实测水浓度为本污染物共性约束，全部 pcb209 方案主目标须统一以"ΣPCB 量级 + 单体份额推断 + 竞争谱含水相主导低氯同系物"表述。
