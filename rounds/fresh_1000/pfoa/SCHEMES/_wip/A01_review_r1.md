## 2. 创新性检查清单 A 至 E 逐项裁决

判定规则回顾：A 至 E 五维中至少一维须展示明确的、可被实验验证的创新性；纯载体替换、参数优化、已发表平台简单复用不算创新。本裁决逐维判定如下，结论为五维整体均不通过。

### A 机制创新（整体不通过）

- A1 选择性机制是否超越普通疏水分配、静电吸附或尺寸筛分：不通过。CD 腔对全氟链的疏水容纳属尺寸匹配加疏水效应，胍基-羧酸根离子对为已知超分子相互作用（E15），且 H1 表明胍基对硫酸根/磷酸根是经典强结合而非几何排斥。所谓"乘积"叙事的算式量纲错误（H2），不构成新机制。
- A2 是否提出可被因果对照推翻的新识别原理：不通过。形式可证伪成立（2×2 因子设计、F1 阈值明确，此为本方案优点），但所证伪的命题既不新（C1，胍基杯芳烃 2019 已覆盖识别基元）又先验存疑（H1，方向相反）。因 A2 要求"新识别原理"，新颖性不成立，判不通过。
- A3 机制在目标水质场景是否有独特适用性：不通过。乘积机制在超稀释摩尔比（10⁻⁶ 至 10⁻⁹）下的理论优势被 H1（硫酸根占有压倒 PFOA）与 H4（DOM 疏水羧酸根共享全部机制）否定，ng/L 床层热力学不自洽（C2）。
- 设计稿自评 A 维通过（A2、A3 明确通过），本裁决推翻。

### B 材料架构创新（整体不通过）

- B1 是否超越载体替换或孔径调参：不通过。载体沿用 StyDex 型 Sty-DVB 网络（E6），识别单元为成熟胍基化 CD，实质为载体复用加官能团替换。设计稿自认 B1 部分通过，本裁决判不通过。
- B2 是否引入新空间组织方式：不通过（实质）。声称的 CD 宽口缘胍基预组织在随机自由基共聚下邻接度不受控（H2；磁盘版 CD-乙烯基摩尔分数约 3%，rim-adjacency 先验更低），材料主体为两类独立位点的机械混合，宏观等温线为异质位点加和，不构成受控预组织。
- B3 合成路线是否有方法学贡献：不通过（设计稿自认）。TsCl 活化、叠氮置换、Staudinger 还原、GMA 乙烯基化、悬浮聚合均为成熟方法。

### C 仿生转译创新（整体不通过）

- C1 转译是否超越官能团类比：不通过。H3 示设计稿提取的是"胍基位于疏水腔口缘"的几何关系，却移植到完全水化、高介电（约 78）的 β-CD 宽口缘，即精氨酸-羧酸根方向性识别最弱的区间；原型中使方向性识别成立的低介电、部分去溶剂化口袋这一使能条件未被转译。实质为"胍基等于胍基"的官能团类比加一个未经证实的方向性假设。
- C2 是否提取了非常规化学直觉可得的设计原则：不通过。所引原型（E1 hL-FABP、E2/E3 hSA、E9 FABP4）均为链长/疏水驱动的 PFAS 结合，无任一条证据表明蛋白胍基区分羧酸根与磺酸根；FABP4 对 PFOS 同样结合，E9 明确氟代物亲和力差异归因疏水与体积。"胍基头基筛选"假设在原型证据中无对应物。
- C3 转译后材料特征是否保留原型核心功能逻辑：不通过（先验不利，只能实验回答）。核心功能逻辑（低介电预组织阳离子氢键供体的方向性识别）未转译，水相中先验最弱（设计稿 C3 已自认"待验证"，但未给出使能条件的转译路径）。
- OAT4 证据（E4）同时改变链长（C4 对 C8）与头基（磺酸根对羧酸根），两维混淆，不构成干净的链长窗口生物证据（H3 其四）。动态对应不声称是正确的（陷阱 14 意识）。

### D 选择性策略创新（整体不通过，仅 D3 通过）

- D1 选择性来源是否可由单一物化参数单调解释：不通过（条件性，依 H1）。形式上为头基几何乘链长两维，但 H1 表明头基维度先验无效或反向，净选择性退化为腔体链长/疏水单一维度，即成熟 CD 平台既有能力，落回"单一参数（疏水/链长）单调解释"的不通过条件。
- D2 是否设计至少两个正交识别维度的协同：实质不通过。两维度名义正交，但 H2 表明协同因子未定义、裸乘积量纲错误且预设两维独立（与 F2 超加和检验自相矛盾）、邻接度不受控，材料级协同在设计内不可辨识。
- D3 竞争物谱是否包含结构近似物：通过。PFOS（同链长 C8 异头基，最关键结构近似物）、PFNA（同头基异链）、PFBS/PFHxA/PFDA、GenX，覆盖同系物、同头基异链、同链异头基三类，并含辛酸根类比，满足陷阱 15 要求。此为 D 维唯一达标项。
- 设计稿自评 D 维通过（D1 至 D3 全部通过），本裁决推翻至"仅 D3 通过"。

### E 先例区分（整体不通过）

- E1 是否完成系统先例检索：不通过。漏检决定性先例胍基杯[5]芳烃 Nat Commun 2019（C1），E8/R10 机制误标（脲位点 → NDMB 叔胺）。检索受 WebSearch 故障限制虽已声明，漏检的恰是核心识别概念同构先例，致命。
- E2 与最近先例区别是否可量化：不通过。因 C1，最近先例变为胍基杯芳烃 2019 与 E8/R10 协同 POP，设计稿未设对该二者的头对头量化区分；原设的"对 StyDex、对脲位点 POP"区分基于误标。
- E3 增量是否足以独立发表：不通过。扣除已发表概念后增量收窄为载体替换、ng/L 床层、真实基质数据，恰为 C2/H1/H4 预测失败处，且面对胍基杯芳烃头对头比较，增量不足以支撑独立发表。

创新性门槛裁决：五维整体均不通过，不满足"至少一维展示明确可验证创新性"的准入条件。局部优点（D3 竞争物谱、A2 形式可证伪、报告口径严格）不足以构成维度级创新。

## 4. 第 2 轮修复要求与终止触发条件

第 2 轮须为真正重建（机制重定位或换角度），不得就地小修。攻击稿提供两条路径，裁决者采纳为第 2 轮的合法选项：

路径一（机制论文重定位）：保留"胍基对季铵的水相头基选择性量化"为核心命题，把 H1 的高先验风险本身转化为科学问题，载体降级为工具；删除或改写 S5 类工程判据为质量吞吐量基准（µg PFOA/g 或 µg/mL 床层、注明进水浓度）；机制叙事改写为"腔体主导、胍基贡献待测且可能为负"，写入绝对选择性因子与占有分析；仿生矩阵诚实降级为官能团类比加超分子假设；先例地图按 C1 重绘，给出相对胍基杯芳烃 2019 与 E8/R10 两个基准的可量化区分；合成按 M4 修补；判据按 M1、M2、M5 重构（首要差值判据、床层硫酸盐直接证伪、摩尔基准分离因子）。

路径二（换角度重建）：若第 2 轮判定 H1 先验风险过高（胍基装饰化概率高）且 C1 增量窗口不足，本槽带台账终止，记录终止原因为"先验化学不支持水相头基几何筛选，且识别概念已有胍基杯芳烃 2019 先例"。

终止触发条件（任一条第 2 轮仍不满足即终止该槽，按 SPEC 耗尽条款记录）：

1. C1 不能重绘先例地图并给出相对胍基杯芳烃 2019 与 E8/R10 的可量化、可发表的增量区分；
2. C2 不能将 2000 BV 目标与直接测得的 ng/L 等温线及明确运行工况对账、或改写为热力学自洽的质量吞吐量口径；
3. H1 不能写入绝对选择性因子与占有分析、并把机制叙事降级为腔体主导；
4. H3 不能将仿生矩阵诚实降级为官能团类比加超分子假设（或改译为有证据支撑的链长单调亲和力梯度原则）。

无论走哪条路径，第 2 轮均须闭合 H2（协同因子定义与 rim-adjacency 测定方案）、H4（DOM 共享机制的防线或诚实限缩适用范围、阴离子性 DOM 占据测量）、M1 至 M6、L1、L2，并由监督者消除 P1 版本错位。

## 5. 裁决边界与诚实声明

本裁决严格基于磁盘版 A01_design_r1.md（GMA/TsCl 路线）与 A01_attack_r1.md 的交集实质判定。攻击稿 C1、C2、H1、H2、H3 经逐条核验对磁盘版设计稿成立；H4 核心成立、个别判据编号按磁盘版 F6 重新归置；M4 的 VBC 专属子项标注为版本错位、其余子项成立。评分反映磁盘版设计稿当前状态：诚实度与报告口径、因果对照结构、竞争物谱为真实优点（已计入相应维度得分），但核心识别概念已发表（C1）、关键工程判据热力学不自洽（C2）、中心选择性假设先验反向（H1）、乘积模型量纲错误且不可辨识（H2）、仿生转译丢失使能条件（H3）共同构成不可就原叙事修复的缺陷合力。未降低 85 分通过线，未将任何未解决 critical/high 计为已解决。裁决者未读设计与攻击角色的隐藏推理，未读 `_r2` 系列产物，未对攻击稿引用的 DOI 重新联网核验（沿用攻击稿 Crossref 核验层级，其中胍基杯芳烃 2019 与 E8/R10 为摘要级核验、其余多为元数据级；C1 另获 _wip 既存裁决独立 Crossref 复核旁证）。P1 版本错位与并发产物问题提请监督者在组装正式方案文件前回溯核对。
