## 1. 评分细目（合同 final_scoring 量表，通过线 85 不降低）

### 1.1 因果闭环 15/20

总问题 SP-A01-0 分解为三个可独立测量子问题，每命题配测量、阈值与失败条件；G0 亲和力门槛把床层目标与协同假说脱钩为门控关系，规避了第一轮 2000 BV 判据与亲和力相差三至五个数量级的矛盾；第 4.5 节床层质量衡算 BV_stoich = ρ_b·q_max·K_eff 量纲自洽、数值正确（经攻击者独立复算与裁决者抽查）；硫酸盐预饱和置换与传质三联诊断直接列为证伪项而非辅助观察。扣分有二：其一，核心新意命题（胍基对季铵增量）先验薄且方向不确定，因果链的最弱环节恰是方案的存在理由；其二，硫酸盐稳健性完全押注腔体单一机制防线，SCI-3 所指二价硫酸根桥联占据口缘胍基的分支未进入批次层裁决结构，M1 对 M4 判据缺净负分支；H-A01-1 题名"腔内阳离子位锚定头基"表述与第 4.1 节自认的 6A 尾端侧几何内部张力明显（BIO-1），因果叙事在仿生环节不自洽。

### 1.2 选择性吸附机制 13/25

机制框架本身严谨：ΔG 加和分解含无量纲耦合因子 ω，PFOA/PFOS（同链长异头基）与 PFOA/PFNA（同头基异链长）构成单参数解释的反例构造，竞争物谱含 PFOS、PFHxS、PFNA、PFDA、GenX 等结构近似物加油酸、棕榈酸、胆汁酸最危险类别并满足 10⁻⁶ 至 10⁻⁹ 真实稀释比，全部因子改摩尔基准并以双向方差分析交互项为主诊断。扣分基于三条结构性约束：其一，头基项先验方向不利，E15（水相胍鎓-硫酸根强离子聚集体，本轮核验存在）与 E19（胍鎓缀合物直接以硫酸根为客体，本轮核验实体确认）构成方向反证，设计自判 α(PFOA/PFOS) 存在小于 1 的合理预期；其二，按设计自判最可能结果，头基维约等于一、PFCA 同系选择性退化为对链长即疏水性类参数的单调函数，命中 INNOVATION_CHECKLIST D 第一问"单一物化参数单调解释即不通过"规则（SEL-1），选择性机制的核心维度在最可能实现的材料中塌缩；其三，长链两亲羧酸根类别简并在本架构内无解，材料按自设链长单调性优先结合 C16 至 C18 羧酸根并置换 PFOA，阴离子性 DOM 多价占据胍基位点无正交防线。乘积协同在概念层级已由 E17 发表，机制层无可宣称新原理。

### 1.3 人工材料可转译性 15/20

人工识别单元为单胍基-β-CD 主客体结构，非蛋白、非折叠肽、非生物杂化；可溶模型与网络双形态分工清晰（严格双突变循环限于位点定义均一的可溶层，网络层显式标注群体加权表观值）；SOP 对区域化学（低当量 TsCl 加色谱纯化加定量核磁，双取代杂质放行限 5%）与胍基化完全度（元素 N%、电位滴定双 pKa、XPS N1s、茚三酮四法联立）控制到位；泄漏面板含苯乙烯单体（MCL 0.1 mg/L）、DVB、叠氮残留、N-氯胍基氯化副产物筛查；容量按总干复合体 mg/g 与床层 mg/mL/BV 双口径报告，附可及位点密度、空白载体 uptake 与受体质量分数，符合陷阱 11；溶胀-可及性-水力学关联与 50 循环疲劳预实验覆盖高阳离子网络工程风险。扣分有二：其一，CHEM-1 未承接，VB-CD 保留 20 个游离羟基而被要求溶于甲苯/十二醇疏水致孔油相共聚，无羟基保护或极性共溶剂条件下非均相风险高，SOP 缺溶解性预实验，直接危及位点定义与群体加权耦合的可解释性，并可系统性偏向 F1 表面主导分支；其二，6A 伯缘取代使胍基位于尾端侧而非头基侧，实现几何与所售仿生并置倒置（BIO-1），统计共固定化无预组织且共定位占比不受控，仲缘路线回避；单体总收率约 19% 量级、色谱通量限制放大，中试经济性未评估。

### 1.4 原创性 5/15

核心识别概念"胍鎓阳离子锚加疏水大环腔选择性结合并水相去除 PFOA 与 PFOS"已由胍基杯[5]芳烃系列发表（E11 核验不成立但 E12 本轮核验存在且独立占据该三元组合）；机制层先例拥挤：水相胍基-羧酸根定向（E13、E14）、CD 聚合物静电乘疏水协同（E17，概念层级已发表）、胍鎓-硫酸根强聚集（E15）、胍基-CD 缀合阴离子识别实体（E18）、大环胍鎓受体源头（E20）、胍基离子型 POP 水相阴离子捕获（E21、E22）。本轮独立核验进一步确认 PREC-1：设计台账漏检同课题组 styrenic CD 平台论文 acscentsci.2c00478（ACS Cent Sci 2022）与 acsami.4c01969（ACS AMI 2024），后者对本方案 styrenic CD 载体类别做系统阳离子共单体加 CD 尺寸结构-性能映射，直接预告 M2 季铵对照与 γ-CD 腔径变体两个核心实验；含氮三脚架交联 CD 聚合物 PFAS 吸附（acsmaterialslett.0c00240，2020）亦漏检。设计"系统先例检索通过"在该漏检下不成立。残余增量收窄为"胍基特异地置于 styrenic β-CD 的 ng/L 床层加真实基质加容量口径协议加类别边界定量表征"，属二阶增量，且完全系于设计自判最可能归零的胍基增量（阈值 2 倍本身落在已知化学区间内，达到亦不构成新识别原理）。设计自评 6 分写于 PREC-1 漏检被指出之前，裁决者据确认后的先例密度下调至 5。

### 1.5 实验可证伪与对照 9/10

证伪结构为全方案最强维度：G0 前置门槛门控床层目标层级；M1 至 M5 五重材料对照加 γ-CD 腔径变体；可溶层严格双突变循环四角物种承接小效应洁净测量，网络层表观耦合显式标注不能区分预组织与统计共现；F1 至 F9 九条失败判据覆盖腔体归属、胍基增量、亲和力门槛、硫酸盐床层置换、类别竞争、官能化完全度、合规泄漏、传质伪平衡与 DOM 阻断；硫酸盐预饱和 100 BV 后 ng/L PFOA 置换列为核心证伪项；分析质量控制含方法空白、基质加标回收、平行样与流程空白。扣分有二：其一，位点密度稀释系列对预组织存在性仅为启发式判据非严格证明；其二，SCI-3 所指"M1 低于 M4 逾 3σ 即胍基净有害"终止分支与位点模型双位点桥联项缺失，F2 仅覆盖增量近零不覆盖增量为负；G2 裁决门未预注册统计功效（重复数、材料间方差、最小可检效应），弱结合增量存在埋于批次误差而长期不可判定的风险。

分层标注纪律总体良好：31 条证据区分元数据核验、摘要级核验、台账转交与通行值待核验四层，已知、推断、设计假设分层清晰，E10 误标（非脲位点、NDMB 叔胺加疏水 PAF-1）更正正确，检索式与覆盖面声明齐备。但存在三处经本轮接口核验确认的台账缺陷：其一，EV-1（high），E11 DOI 裁决者 Crossref 复验 404，"PMC6789061 交叉印证"核验主张不成立，属本仓库 SOUL.md 禁止编造引用与核验硬边界所辖缺陷，无论是否出于无心，该条核验状态必须改写；其二，EV-2（medium），E19 实体经核验为胍鎓-苯甲酸酯缀合物非胍基-CD 缀合物，设计 L2 闭合依据所引两条胍基-CD 实体锚失其一，仅余 E18；其三，PREC-1（high），漏检三篇平台先例（acscentsci.2c00478、acsami.4c01969、acsmaterialslett.0c00240，本轮均核验存在），系统检索完整性主张不成立。设计自评 9 分写于上述缺陷被指出之前，据确认后缺陷下调至 5。全部条目未读全文、β-CD·PFOA 直接 K_a 未定位、E30 CMC 非接口核验，沿用设计自标待核验，不重复扣分。

## 2. 创新性检查清单 A 至 E 五维逐项

### A 机制创新：不通过

- 超越普通疏水分配、静电吸附或尺寸筛分：不通过。腔体疏水项是 β-CD 平台通行机制（E7），阳离子加疏水协同在 CD 聚合物上于概念层级已发表（E17），胍基-羧酸根定向有 Schmuck 受体先例（E13、E14，本轮核验 E13 存在），胍基-CD 缀合阴离子识别有实体先例（E18）。第一轮头基几何排硫宣称已被 E15、E19 反证并撤回，无新识别原理可宣称。
- 可被因果对照推翻的新识别原理：仅保留可证伪测量命题（胍基对季铵摩尔增量大于等于 2 倍、α(PFOA/PFOS) 大于等于 1.5），命题可证伪性不构成原理新颖性。
- 目标水质独特适用性：腔体项主导的硫酸盐稳健性若成立有实用价值，属平台共性延伸。
- 改进方向（备查）：几何可辨识的对照系列（脒鎓、N-甲基胍鎓、硫脲、中性氢键供体）加 DFT 几何分解加受限水微环境。

### B 材料架构创新：不通过

- 超越载体替换或孔径调参：不通过。StyDex 为成熟平台（E8），且该平台阳离子共单体加 CD 尺寸的结构-性能空间已由 Dichtel 组系统映射（acscentsci.2c00478、acsami.4c01969，本轮核验存在），胍基以苄基胍形式统计散布属已知阳离子 CD 树脂上的胍基接枝。
- 新空间组织方式：不通过。网络无腔口预组织，共定位位点占比未知且不受控（H2 不可辨识性已接受）；严格预组织仅在可溶模型层级，其几何为尾端侧而非头基侧。
- 方法学贡献：不通过。TsCl 活化、胺置换、胍基化、自由基共聚均为成熟方法，质量控制改进非方法学贡献。
- 改进方向（备查）：仲缘 C2/C3 定点单取代加刚性连接臂取向锁定加二维固态核磁取向表征。

### C 仿生转译创新：不通过（设计自评"条件通过"，裁决改判）

- 超越官能团类比：不通过。设计所设条件为可溶模型双突变循环耦合能非零且网络胍基增量可测；但 BIO-1 经核验成立，实现几何为 6A 窄口尾端侧取代，PFOA 自宽口入腔、羧酸头基留宽口外，胍基与头基分处空腔两端而不能并置，即便双突变循环测得非零耦合能，证明的也是尾端侧胍基-氟链作用而非其所售"腔口阳离子锚定头基"的仿生并置，操作判据的验证对象与宣称几何倒置，条件达成亦不支撑宣称。
- 提取常规化学直觉之外的设计原则：不通过。降级后实际转译内容仅为"疏水腔加腔口阳离子锚"，即面对阴离子靶标的标准 β-CD 阳离子聚合物设计（BIO-2），原型使能条件（低介电预组织口袋 ε 约 4 至 10）在全水化 CD 口缘（ε 约 78）丢失，原型证据（E1 至 E3）均为疏水与链长驱动（E4 明示），无一条支持头基区分。
- 保留原型核心功能逻辑：待验证且先验不利。水相胍鎓-羧酸根 K_a 约 10⁰ 至 10² M⁻¹ 背景（E13）下核心功能逻辑在水相保留的先验薄。
- 判定理由：条件通过的两个条件即便全部达成也不能验证其所售仿生宣称（几何倒置），诚实读数下 C 维为装饰性仿生框架，改判不通过。改进方向同 B 维仲缘路线，或放弃仿生框架纯以平台测量研究定位。

### D 选择性策略创新：条件通过，最可能结果下不通过

- 单一物化参数单调解释：形式通过，最可能不通过。反例构造（PFOA/PFOS 与 PFOA/PFNA 两组对照）在形式上排除单参数吸收，但设计自判头基维最可能零结果甚至反向、PFCA 同系序列选择性最可能退化为对链长（疏水性类参数）的单调函数（SP-A01-2、W4、第 5.5 节诚实预判），即在设计自判最可能实现的材料中命中 D 第一问不通过规则（SEL-1）。D 维通过系于设计自判将归零的维度。
- 两个正交识别维度协同：形式通过，警示保留。腔体链长维与头基锚定维正交为假设而非给定；网络层交互项为群体加权表观值；胍基对季铵的差异（电荷离域度、脱溶剂化罚分、供体数）彼此共变，属头基静电维度内扰动，设计未设区分方向性与电荷密度的对照（如电荷匹配几何不同的硫脲或中性氢键供体），即便 ΔΔG 非零亦可能被单一局部电荷密度参数吸收。
- 竞争物谱含结构近似物：通过并扩充，含同系结构近似物与最危险类别，满足真实稀释比，无保留。
- 判定：改判条件通过，条件与 C 维同一（头基/胍基维可测且非单参数吸收）；据设计自判先验，最可能结果下不通过。

### E 先例区分：条件通过，当前检索项不通过，最可能结果下不通过

- 完成系统先例检索：当前不通过。PREC-1 经本轮核验确认：台账漏检 acscentsci.2c00478（ACS Cent Sci 2022）、acsami.4c01969（ACS AMI 2024）与 acsmaterialslett.0c00240（ACS Mater Lett 2020）三篇平台先例，其中 acsami.4c01969 对本方案载体类别的阳离子共单体加 CD 尺寸空间已做系统映射。空白收窄表述（胍基-β-CD 网络的 ng/L 床层与真实基质未见先例）属实，但系统检索完整性不成立，须补检后重判。
- 与最近先例区别可被实验量化：通过。对胍基杯[5]先例（E12）的三指标增量量化（链长锐度、连续柱对批次形态的可操作性、硫酸盐床层置换寿命）与对 NDMB POP（E10）的 α(PFOA/PFOS) 差异结构清晰，惟 M2 对照与 γ-CD 尺寸实验已被 Dichtel 2024 先占。
- 增量足以支撑独立发表：条件通过，窗口窄且先验不利。条件为补检后胍基增量可测且对 Dichtel 2022/2024 平台与 2020 含氮三脚架 CD 先例给出至少一项稳健增量；据最可能结果（胍基增量归零、α 约 1、链长单调、K_eff 处 10³ 至 10⁴ 量级），增量为现有阳离子序列中又一个未测且自判归零成员的二阶增量，独立发表概率低。
- 判定：条件通过；当前检索项不通过，须补入三篇漏检先例并重写 L2 与 5.1 节；最可能结果下整体不通过。

### 清单总判定

判定规则要求至少一维展示明确的、可被实验验证的创新性。当前磁盘版本中 A、B、C 三维不通过，D、E 两维为条件通过且其条件均系于设计自判最可能归零的胍基增量；诚实读数下没有任何维度无条件通过，方案创新价值整体从属于低概率实验结果。清单不满足。

## 4. 裁决

### 4.1 得分与门槛

总分 62/100，低于 85 通过线；open_critical = 0，open_high = 5。两项门槛同时不满足。门槛不降低。

### 4.2 verdict：terminate（本方案槽 A01 终止）

裁决依据分两层。

其一，文档质量层面。第二轮修订异常严谨：第一轮全部致命风险（几何排硫、乘积模型量纲、单一预组织取向、2000 BV 判据）均以主张撤换加诚实降级分支承接，宣称未超出对照支撑，攻击者未发现设计者漏判的新 critical 级硬错误。若问题仅为文档缺陷，应判 revise。

其二，结构约束层面。未决五项 high 中，EV-1 与 PREC-1 属台账可修正缺陷，修正后亦不提升原创性实体；SEL-1、SCI-1、SEL-2 属角度内在约束，第三轮设计无法解决：不能取消 E12 与 Dichtel 2022/2024 平台对识别概念与载体空间的先例占据（原创性上限约 5 至 8/15），不能改变水相胍鎓-羧酸根结合与胍鎓-硫酸根强聚集的化学先验（头基维最可能归零，D 维最可能按自身规则不通过），不能使 CD 包结量级亲和力（10² 至 10⁵ M⁻¹）闭合 ng/L 床层所需的 10⁶ 至 10⁸ M⁻¹（命名交付在最可能世界不存在）。设计自评与攻击者独立判断方向一致：本槽最可能收敛为 styrenic β-CD 阳离子聚合物序列中又一个胍基零效应负结果，独立发表概率低。修订不改变结构，再一轮设计-攻击-裁决不产生通过路径，故判 terminate 而非 revise。

诚实的降级与终止优于凑数的宣称。设计第 11 节自述"是否触发 SPEC 耗尽条款由裁决者判定"，裁决者据此判定：A01 角度终止。
