# MAC-BI1隔离攻击

日期：2026-07-18

裁决：`concept_valid / commercial_chemistry_prescreen_required`；3项critical、5项high，不能进入`E1_ready`。

## Critical

1. **现有材料对照不能单独证明界面协同。** MAn同时改变聚合、孔结构、润湿、溶胀和交联；商业弱酸树脂与物理混合物也不能与一体微球完全匹配。必须增加同一合成体系的MAn投料梯度和部分水解梯度，测湿态孔容、溶胀、扩散和可接近酸容量，并用双因素交互判断协同。
2. **第一轮性能阈值缺少固定实验条件。** 去除率没有绑定进水浓度、材料剂量、液固比、接触时间、竞争物、重复数、空白损失和LC-MS/MS能力。E1前必须冻结两个浓度层级及完整分析条件，并同时报告单位质量吸附量、分配系数、类别偏好和置信区间。
3. **尚无符合成本边界的准确小试配方。** 1987年成珠配方使用甘油、甲苯和二氧六环，只证明颗粒可制；NaOH水解后材料的Na型、H型或部分中和状态也未定义。必须冻结配方、清洗、水解、转型、残余单体、溶剂残留、湿态抗压和粉化门；二氧六环未替代前不能称为低成本放大方案。

## High

1. **普通树脂包装风险。** 当前只证明同一颗粒含羧酸和苯环，没有证明二者位于同一可接近界面。若不能优于商业混合模式树脂，仿生论文主张失败。
2. **化学未必偏好大环内酯。** 普萘洛尔、文拉法辛和西酞普兰同样是疏水阳离子且尺寸更小。正式合成前应先用商业弱酸树脂、非离子大孔树脂、物理混合物、混合模式树脂和GAC做等摩尔竞争预筛。
3. **真实水条件定义不足。** 必须分别测DOM共吸附、Ca/Mg占位、过滤前后差异和至少三个日期水样，不能用一个模糊的保持率代表基质耐受。
4. **GAC比较方式不合理。** 不再要求绝对提高20个百分点；改为同一出水目标下比较所需剂量、床寿命、单位有效吸附量、DOM共吸附和处理成本。PAC批量投加与GAC固定床不得混为一个成本门。
5. **再生去向未闭合。** 每周期必须报告解吸和再吸附质量回收、再生液体积、浓缩倍数、清洗水量、残余苯乙烯、可浸出有机碳和细颗粒。

## 继续顺序

1. 商业材料等摩尔竞争预筛；
2. 只有出现大环内酯类别偏好，才冻结MAn–St–DVB小试配方；
3. 用同体系组成和水解梯度解析协同与孔结构；
4. 关闭安全致孔剂、湿态强度、残余单体和再生质量去向后，再申请E1。

如果商业物理混合物和混合模式树脂均没有大环内酯类别偏好，MAC-BI1停止主线，不使用仿生措辞继续包装。
