# 原方案复核：严格生物吸附原型口径

状态：监督复核完成，等待潘尧讨论；未授权采购、合成或实验。

## 1. 本轮问题

本轮不继承 `rounds/portfolio_100/` 的方案与分数，只复核 `main` 上已完成深审的非蛋白路线，并把潘尧提出的新要求加入判断：优先使用现实中可观察、可核验的生物吸附、富集或屏障过程，不把污染物偶然结合某个蛋白靶点自动写成“生物吸附污染物原型”。

每套方案必须同时回答：

1. 仿生故事是否诚实完整；
2. 人工识别单元、材料结构和实验因果链是否科学；
3. 是否有机会进入台式制备、二级出水竞争、床层、再生、浸出与稳定性验证。

`通过` 表示纸面设计已满足本项；`条件通过` 表示方向合理但尚有一个可明确关闭的关键门；`不通过` 表示当前方案不能靠补充配方或改变措辞关闭。

## 3. 现有原方案中真正值得保留的内容

### 3.1 第一优先：`S1-SYN`，但必须降低故事强度

这是现有方案中因果设计最完整的一条：羧酸头基定位、有限低极性链区、两区共定位、长度错配和亲水抗污界面均可形成明确的删除/错配对照。人工材料不含蛋白或肽，载体又保持材料类型中立，工程指标也已经包含总干材料质量、床层体积、可达位点、DOM、天然脂肪酸、表面活性剂、再生与浸出。

缺口有两个。第一，FABP 路线属于“生物机制参考＋天然脂肪酸功能类比”，不是现实生物体系对 PFOA 的工程吸附方案。第二，胍基、酰胺和有限链区仍只是功能清单，不是确定可制造的人工识别单元。因此本轮只能把它列为最值得继续设计的原路线，不能判为实验就绪。

### 3.2 第二优先：`BPA-1/P1-MAPy`，只保留为低成本机制门

P1-MAPy 的优点是结构确切、对照可合成、聚合出口清楚，实验能快速判定 ERRγ 式“单端互补极性环境”是否有任何人工转译价值。缺点是现有成熟化学支持羧酸两点识别，不支持对一个酚氧的同等几何；水相先例也更像一般酚类吸附。因此 P1-MAPy 是高信息量失败前置实验，不是可以直接写 SOP 的材料方案。

### 3.3 工程阳性对照：`BPA-2E`、`PFBS-1`、`ROX-1`

三者的材料可制造性和阳性吸附概率高，但分别受成熟 β-CD 先例、盐敏感离子交换和直接 COF–MIP 先例限制。三者适合建立最低工程基准，不能承担本项目的仿生创新主张。

## 4. 更符合新口径的两条重定向线索

以下不是已通过方案，只是比“从蛋白结合位点反推材料”更符合新要求的候选原型。两条线都需要独立深设计，不能继承 `S1-SYN`、ROX-2 或 GLM 分数。

### 4.1 `PFAS-M1`：生物膜界面富集启发的稳定两亲纳米域

直接实验表明 PFAS 能进入磷脂双层并在细菌中积累，强度随全氟链长度和头基改变；中子反射进一步显示 PFBS、PFHxA、PFHxS、PFNA、PFOS 和 FOSA 均可进入双层，且 PFNA 在温和水洗条件下表现出不同的保留行为。这里的生物参考是现实膜界面对 PFAS 的吸收/富集，不是某个蛋白口袋。

可讨论的人工材料是共价稳定的两亲纳米层或纳米孔：水相界面设置磷脂头基层级的两性离子/氢键环境，内部设置有序低极性链区，但不使用真实脂质体。硬对照包括头基层删除、低极性域删除、同组成随机共聚物、纳米域有序/无序体和链长/头基系列。

主要风险同样明确：现有膜数据总体随链长增强，PFSA 往往比 PFCA 更强，尚未证明 PFOA 或 PFNA 的非单调选择性；若材料只复制广谱长链 PFAS 分配，它只能成为工程吸附路线，不能成为选择性主方案。

### 4.2 `ROX-BF1`：生物膜/EPS 实际吸附启发的分级带电微域

真实生物膜研究在 `1 µg/L` 量级发现 ROX、红霉素、克拉霉素等带正电微污染物发生显著固液分配，厚度与可达孔隙会影响平衡分配和扩散；好氧颗粒污泥研究又显示 ROX 去除受污泥吸附能力影响，并与腐殖酸样 EPS 富集同向变化。这里至少存在可观察的生物材料吸附过程，而不是只引用核糖体靶点。

可讨论的人工材料应转译“水化带负电外域＋内部腐殖样芳香微域＋受控扩散长度”，并用相同总电荷但均匀分布、反向梯度、无芳香域和薄层体作因果对照。ROX 的 9-O-甲肟醚只能作为待验证的人工判别线索，不能冒充 EPS 的生物证据。

主要风险是生物膜数据首先支持“带正电药物类别吸附”，并不支持 ROX 相对其他大环内酯的选择性。若找不到对 9-O-甲肟醚的确切水相人工识别单元，该路线最多形成大环内酯类别富集材料。

## 5. 当前排名与实验许可

按“值得继续投入纸面设计”的优先级，而非旧分数：

1. `S1-SYN`：现有因果链最完整，但只能按 B 类机制启发叙事继续；
2. `PFAS-M1`：最符合实际生物富集原型的新重定向，选择性仍需证明；
3. `ROX-BF1`：具有实际生物膜吸附证据，ROX 专属性风险高；
4. `BPA-1/P1-MAPy`：仅作机制门，不进入材料；
5. `BPA-2E`：工程阳性对照，不承担论文仿生主线。

当前没有任何方案同时通过仿生故事、材料/实验科学性和工程验证三关。`primary_design` 与 `backup_design` 必须继续为 `null`，也不应启动完整材料合成。下一步应只在 `S1-SYN`、`PFAS-M1` 和 `ROX-BF1` 中选择不超过两条，冻结确切非蛋白人工识别单元/纳米域、匹配对照和材料质量账本，再进行一次纸面攻击。

## 6. 新增证据边界

- PFAS–磷脂双层中子反射：[DOI 10.1016/j.jcis.2017.09.102](https://doi.org/10.1016/j.jcis.2017.09.102)。支持真实膜掺入及链长/头基依赖，不支持 PFOA 专属性。
- PFAS 在模型膜和细菌中的分配：[DOI 10.1021/acs.est.8b02912](https://doi.org/10.1021/acs.est.8b02912)。支持细菌与膜积累，不支持可逆吸附床性能。
- 纳摩尔 PFOA 改变 DPPC 双层：[DOI 10.1021/acs.est.4c03652](https://doi.org/10.1021/acs.est.4c03652)。支持低浓度膜相互作用，不支持选择性材料结构。
- 微污染物在不同厚度生物膜中的扩散与吸附：[DOI 10.1016/j.watres.2017.06.027](https://doi.org/10.1016/j.watres.2017.06.027)。支持 ROX 等阳离子药物的实际生物膜分配，不支持 ROX 单分子选择性。
- 好氧颗粒污泥对 ROX 的吸附贡献：[DOI 10.1016/j.jhazmat.2020.122583](https://doi.org/10.1016/j.jhazmat.2020.122583)。支持 ROX 去除与污泥吸附/EPS 变化相关，不证明腐殖样组分是唯一因果位点。
- PFBS 耐受菌 EPS 生物吸附：[DOI 10.3390/toxics12110797](https://doi.org/10.3390/toxics12110797)。支持 EPS 参与 PFBS 去除，但目标浓度、选择性和硫酸盐竞争不足以把它升级为活动方案。
