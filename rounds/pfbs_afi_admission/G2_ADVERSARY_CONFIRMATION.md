# G2 裁定对抗核验(放行方镜头)

- 核验结论:维持原裁定(uphold_original=true)
- 最终裁定:杀死

## 理由

以放行方立场尽力核验后，两个窄命题均成立：(1) 膜抗污与吸附剂床寿命保护确属不同问题类（膜保护界面分离功能本身；吸附剂涂层须同时满足排阻大分子、透传小分子、保分配容量、耐床层水力的耦合约束），概念简报该处表述无误，原裁定亦未依赖膜文献作为杀死依据；(2) 增量 (i) 的完整现象级合取（水合界面 x DOM x 弱分配微污染物 x 床寿命 x 二级出水）经本人独立 Crossref/PubMed 补检索未见任何单篇先验直接占据，与原裁定记录一致。但两条窄胜不能合成规范 4.4 所要求的"可辩护的非组合式增量"：机制级主张"两性离子抗污层保护活性炭对小溶质的吸附免受大分子污堵"已被 Cai 2017(JCIS,10.1016/j.jcis.2017.04.024,PMID 28521219，摘要级核验）直接占据，且本次核验新确认同组延续工作 Zhang 2018(10.1007/s10856-018-6127-4,PMID 30019317）与 Li 2019(10.1016/j.jcis.2019.07.045,PMID 31377640)，存亡轴先验密度上升；问题级框架"排阻 NOM 保护微污染物吸附"已被 Karanfil 2006(10.1021/es051285o，题录核实）及 NOM 预载文献群占据；水处理方向亦存在两性离子接枝吸附剂（Wang 2020 IECR,10.1021/acs.iecr.9b06171)。增量 (i) 的全部元件已知，"未被占据的已知元件合取"按规范 4.3 定义为类目 (c) 组合式应用级增量；简报自身亦将其标注为"应用层级增量"。放行方唯一可争的非组合空间是边缘尺寸选择性（DOM 低分子馏分与 PFBS 尺寸差约 3-7 倍，远窄于 Cai 2017 中白蛋白/亚甲基蓝的约 200 倍），但该主张未经任何先验演示，成立性悬于方案自设证伪门，按规范仅属 (b) 条件性窄口径增量；增量 (ii) 测量构型同为 (b) 级，增量 (iii) 自认组合式。v2 回溯先例已裁定条件性窄口径增量不能挽救主增量为组合式的方案。依附性核查：增量价值依附"DOM 归因床寿命损失 ≥2x"未落账存亡变量，但该事件非可证低概率，依附性杀死条款不独立触发，亦不能将 (c) 升级。规范 4.5 杀死判据第一条（评级低：无可辩护的非组合式增量）触发；杀死判据第三条（掩盖清单缺口）不适用，因简报采用"待门 2 检索证实或证伪"的开放表述。
