# PFHxS设计关键引文核验

日期：2026-07-19。核验目标是限制主张强度，不以文献数量代替因果证据。

| 主张 | 裁决 | 可安全使用的表述 |
|---|---|---|
| PFHxS与其他终端PFAA可在市政污水过程中持续出现 | supported with boundary | 受调查污水厂中PFHxS高频检出；不能外推为所有污水厂，也不能把出水浓度变化全部归为吸附 |
| PFHxS/PFSA进入磷脂双层或细菌界面 | supported | 支持真实界面分配和富集；不支持天然系统专门清除PFAS，也不支持PFHxS同系物特异性 |
| OATP/NTCP/ASBT均直接运输PFBS、PFHxS和PFOS | partial | OATP1B1/1B3/2B1与NTCP对PFBS/PFHxS/PFOS有直接运输证据；ASBT直接证据主要是PFOS |
| 磷脂膜证明固定正电头与烃链尾是PFSA识别单元 | unsupported | 磷脂酰胆碱头基为两性离子；膜研究支持被动分配/扰动，不能给人工季铵树脂提供一一对应的功能仿生证明 |
| ionic fluorogel已全局优化含氟/季铵比例 | partial | 只能说在有限配方范围内筛选，并在真实沉积物相关水样中表现出广谱PFAS去除 |
| 阳离子β-CD已用严格匹配删除体分开电荷与空腔贡献 | partial | 研究通过组成、盐和链长趋势推断电荷、疏水/空腔贡献；不能说完成严格的一变量删除实验 |
| 含氟季铵COF已有PFHxS容量、离子强度、腐殖酸和真实废水结果 | supported | 可作为直接拥挤先例；不得把季铵＋低极性域重新表述为新机制 |
| EPA 1633A禁止实验室任何PTFE材料 | partial | 标准水样流程强调原瓶与瓶壁冲洗，特定条件允许受控分样；明确避免PTFE衬垫瓶盖及提取流程中的含氟聚合物耗材，不应夸大为整个实验室绝对禁用 |
