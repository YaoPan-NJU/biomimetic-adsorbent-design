# PFH-1 M0制造与空间结构否决方案

状态：`pre-lab protocol / not experiment-authorized / not E1`。

本方案只回答一个问题：能否用低复杂度树脂化学稳定制造“弱碱外层—强碱内核”，并做出可比较的均一、随机和反向空间对照？在M0通过前不使用PFAS、不优化吸附容量。

## 1. 起始材料与批次基准

优先使用同一批商品大孔氯甲基化PS-DVB珠，避免实验室氯甲基化和悬浮聚合同时引入粒径、孔结构与残单差异。

- 目标粒径：0.60–1.00 mm；筛分后记录D10/D50/D90。
- 厂商应提供Cl容量和交联度；实验室复核干态Cl、湿态孔体积、含水率和溶胀比。
- 每个候选工艺使用5.00 g干珠；三批独立重复必须来自同一原料大批次的随机分装。
- 所有胺化反应在通风橱和密闭耐压容器中进行；三甲胺尾气进入酸洗涤瓶。具体风险控制以试剂SDS和本单位审批为准。

以实测苄氯摩尔数`n_Cl`为计量基准，不按名义投料估计转化。

## 2. M0-A外层形成小型工艺筛选

目的不是假定某一条件必然形成30–80 μm外层，而是在有界矩阵中寻找可重复的扩散受限窗口。

### 因子

- N-甲基乙醇胺：`0.15`或`0.30 mol/mol n_Cl`。
- 反应时间：5或20 min。
- 溶剂状态：低溶胀水/乙醇`70/30 v/v`；对照为充分溶胀乙醇/甲苯`70/30 v/v`。在反应前用湿态体积测量确认两种体系确实产生不同溶胀，而不是仅依赖溶剂名称。

### 固定条件

1. 干珠用对应无胺溶剂预平衡30 min；沥去游离液但保持湿润。
2. 加入N-甲基乙醇胺和无水K2CO3。K2CO3用量为拟反应N-甲基乙醇胺摩尔数的`0.60 mol/mol`，提供略高于理论HCl中和量的碱容量。
3. 固液体积比`10 mL/g干珠`，25±1 ℃，端对端混合20 rpm；到时立即用20倍床体积冷溶剂淬洗。
4. 依次用乙醇、水、0.1 mol/L NaCl和水清洗，直至连续两份洗液的pH差<0.2、氯离子和TOC低于预先用空白洗液确定的定量终点。
5. 测总氮、残余Cl、弱碱容量、湿态孔体积和初步截面分布。只把外层厚度30–80 μm且径向富集比≥2的条件带入M0-B。

若8个条件均不能得到窗口，允许一次工艺修订：只改变温度至10 ℃或降低溶剂溶胀，不扩大化学体系。仍失败则终止PFH-1。

## 3. M0-B四种空间拓扑

每种拓扑做三批独立重复。

### COMPLETE：弱外/强内

1. 采用M0-A选定的外层条件处理N-甲基乙醇胺。
2. 洗净并测残余`n_Cl,res`。
3. 用25 wt%三甲胺水溶液，按`5 mol/mol n_Cl,res`加入；水/乙醇`70/30 v/v`，10 mL/g干珠，45 ℃、12 h。
4. 依次用乙醇、水、1 mol/L NaCl和水清洗，转为Cl型。

### UNIFORM-S：均一强碱

同批原珠充分溶胀后直接按上述三甲胺条件反应至残余Cl达到相同终点。它是商业强碱树脂化学的内部基准，不要求弱/强比例匹配。

### REVERSE：强外/弱内

1. 在M0-A同样的低溶胀状态下，以三甲胺限制投料形成强碱外层；限制投料摩尔数由COMPLETE实测外层位点量决定，而不是直接照搬0.15或0.30当量。
2. 清洗后充分溶胀，以N-甲基乙醇胺`3 mol/mol n_Cl,res`和K2CO3`1.6 mol/mol n_Cl,res`，45 ℃、12 h处理剩余位点。
3. 若截面结果不能形成强外/弱内的相反分布，不允许把它称为反向体；整条路线因缺乏关键因果对照而停止。

### RANDOM：同组成无梯度

1. 原珠充分溶胀，先用N-甲基乙醇胺处理至其总量匹配COMPLETE，但以低浓度、长时间条件追求均匀分布。
2. 再用过量三甲胺处理剩余位点。
3. 只有在截面无显著径向梯度，并且弱/强比例、总IEC与COMPLETE进入匹配窗口时，才能作为随机体。

这里没有假定REVERSE和RANDOM一定能被制造。能否制造就是M0的核心否决问题。

## 4. 结构与化学质量控制

### 必须完成

- 干态元素分析：C/H/N/Cl；残余苄氯用经验证的化学滴定或光谱方法定量。
- 湿态总IEC及弱/强碱分量：酸碱滴定加Cl交换容量交叉核对。
- 湿态孔体积、含水率、溶胀比和粒径分布；不以干态BET替代。
- 截面微区XPS：在碱性预处理后，利用均一叔胺和均一季铵标准校准N 1s组分。
- 截面ToF-SIMS或另一独立微区方法：必须用两种均一标准和已知混合比例标准建立响应，不接受未经校准的彩色元素图。
- 必要时把多颗珠包埋、逐层切片并汇集外层/中层/内层，做独立的总氮和滴定交叉验证。切片损失和层间拖带必须做质量平衡。
- 用非反应性阴离子示踪物和零长床/搅拌颗粒实验估计湿态有效扩散系数。

### 匹配与通过窗口

| 指标 | COMPLETE相对RANDOM/REVERSE |
|---|---|
| 湿态总IEC | ±10% |
| 弱/强碱位点比例 | ±10% |
| 湿态孔体积 | ±15% |
| D50 | ±10% |
| 外层厚度 | 30–80 μm，三批CV≤20% |
| 径向富集 | COMPLETE外弱/内强各≥2；REVERSE相反；RANDOM无显著梯度 |

有效扩散系数不强制匹配，但必须实测并作为协变量；若拓扑与扩散完全共线、无法统计拆分，则M0失败。

## 5. M0统计与决定

- 每个批次至少随机选30颗珠测外层厚度；报告批内和批间变异。
- 微区化学至少分析三个批次、每批不少于10颗代表性珠；分析人员对拓扑标签盲法。
- 预先拟合`chemical_state ~ radial_position * topology + batch`；COMPLETE和REVERSE的交互方向必须相反，RANDOM交互不显著。
- 所有原始数据、谱图拟合参数、排除珠和洗液终点进入批次记录。

M0通过只表示“材料与因果对照可制造”，不表示PFAS吸附有效，也不自动获得E1。通过后另行冻结痕量分析、PFSA竞争和小柱方案；失败则终止PFH-1，不以继续调温度、溶剂或投料无限延长路线。
