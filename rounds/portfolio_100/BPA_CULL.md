## BPA 攻击/排序淘汰记录

角色：隔离攻击者/排序者（GLM 分支）。输入：BPA.md 中 20 个方案、research_contract.yaml（通过线 85/0-critical；选择性吸附为硬门；硬对应规则要求至少 1 个静态对应，且 biological_feature、source_support、material_realization、expected_function、measurement、causal_control 六字段齐备）。本次攻击只用 Read/WebFetch/WebSearch/Write，未用 bash；不杜撰 DOI、数值或机制，源无法独立核验者标 tier none。

独立核验状态：PDB 2E2R 已由 RCSB 直接读取核验为 ERRγ 配体结合域与 BPA 复合物，配体 2OH 即 4,4'-亚异丙基双酚，关键残基 Glu275/Arg316/Asn346/Tyr326，KD 5.5 nM（IC50 4.9-13.1 nM），与方案描述一致，tier verified 成立。该结构是 BPA-1、BPA-3、BPA-10 三个 ERRγ 衍生方案的承重生物学源，核验通过。其余证据沿用方案内 tier 标注；Elsevier 重定向未给出正文，Zr-MOF 孔径门先验（BPA-11、BPA-18 引用）维持方案内 tier，不升级也不杜撰。

### 攻击主轴

把硬对应规则的 biological_feature + source_support 作为首要淘汰判据。多数方案名为"生物原型"实为材料机制（表面印迹、尺寸门、阴离子交换、N 掺杂碳、CTF 三嗪 N、卤键等），没有生物学源，source_support 也不是生物结构或酶受体证据。这些方案即便材料分高，也通不过硬对应规则。真正以生物学源立论的只有 ERRγ-BPA 复合物（PDB 2E2R，BPA-1、BPA-3、BPA-10）和 β-CD 主客体（β-CD 为生物寡糖，BPA-2E、BPA-2R）。以此为基线做对抗淘汰。

### 保留方案（5，含 1 个最弱标记）

#### 保留 1｜BPA-3 非印迹双端极性 COF 通道（78，partial）

存活理由：总分最高，且 ERRγ 静态夹持（PDB 2E2R 核验通过）转译为水稳 β-酮烯胺 COF 相对两壁定距双极性站加疏水面，六字段齐备，是portfolio 里对应与可制造性平衡最好的一例。水稳 COF 有 class 级先例（含氟 COF 气凝胶双酚固相萃取、离子液体修饰 COF 纳米球双酚萃取），非杜撰。单壁/错位/无极性站/孔径匹配化学随机 COF 四组对照直接证伪夹持主张，可证伪性 9。容量按整颗粒干质量与床层 mg/mL 报告，符合 carrier_mass_efficiency 规则。

关键风险：限量后修饰位点非均一（须消解 NMR/元素分析定量真实比例）；水中氢键被屏蔽或 DOM 堵孔则降级为普通芳香吸附体；粉体成型与共价骨架规模化困难。

深化问题：在水稳 COF 通道两壁量化（消解 NMR/元素分析）双极性站的真实位点比例下，整通道对 BPA 的中性水选择性是否超过 BPF/BPS 且在 DOM 竞争中保留，并在删除任一极性站或错位时增益崩塌？若增益只随疏水或总芳香量变化，则双站夹持主张失败。

#### 保留 2｜BPA-1 ERRγ 非对称单酚锚定超交联多孔聚合物（70，partial，experiment-gated）

存活理由：全 portfolio 仿生保真度最高。直接转译经核验的 ERRγ-BPA 复合物（PDB 2E2R，KD 5.5 nM，突变证据支持非对称单酚锚），显式放弃已被否的对称双端夹持，六字段齐备且 source_support 为 hard 级。P1-MAPy 可溶水相门（无聚集、含自缔合模型、正交两法一致、非疏水指纹）是把"单酚锚在水中是否真有增量"压到可证伪单点的正确关卡，门未过即不冻结材料，纪律性强。

关键风险：P1 水相门尚未通过，人工单酚锚在水中对 BPA 的选择性增量 tier none；自缔合与水竞争为主不确定；定制刚性受体合成步骤多。

深化问题：是否存在任一单酚极性锚（脲/酰胺或替换体）在中性无聚集水相通过 P1 门（自缔合模型含、正交两法一致、非疏水指纹）？若不存在，BPA-1 应降级为机制探针而非材料方案。

#### 保留 3｜BPA-18 Zr-MOF 悬挂酚结合链接剂与孔径门（75，partial）

存活理由：经核验最弱的"生物学源"但经验先验最强。Zr-MOF 孔径调控增强 BPA 去除为方案内 tier verified 直接先验，水稳性好，UiO-66-NH2 功能化 BPA 吸附与 Zr-MOF 雌激素类选择性吸附为 class 级，可制造性最稳。孔径门加链接剂氢键加锆簇疏水面三重机制可证伪（无修饰 UiO-66、孔径错配、无酚结合链接剂、等孔容 COF 对照）。

关键风险：本方案 biological_prototype 字段实为材料机制（Zr-MOF 孔道、链接剂、锆簇），没有生物学受体或酶源，source_support 也不是生物结构证据。按硬对应规则 biological_feature + source_support 两字段，这是 portfolio 内最大的对应缺口，必须在深化中正视：要么找到可审计的生物学对应（如酶/受体孔道尺寸加氢键的类比并标明类比边界），要么诚实改写为非仿生但经验扎实的方案并记为合约偏离。粉体成型与成本为次级风险。

深化问题：能否为"孔径门加悬挂酚结合链接剂氢键"给出可审计的生物学对应（受体/酶孔道加 H 键站）并标明类比边界，否则本方案须诚实改写为非仿生经验路线并登记合约偏离；且链接剂氢键在真实二级出水中的增量是否超过孔容贡献？

#### 保留 4｜BPA-10 深腔主体单脲缘锚定（70，partial）

存活理由：用可溶深腔主体（octaacid 类）疏水腔容纳芳环、缘口单脲锚定单酚氧，是 ERRγ 单锚加疏水包围的直接转译（PDB 2E2R 核验通过），六字段齐备。水溶性 cavitand 合成与缘羧酸 cavitand 客体诱导构象切换有 class 级先例。开口类似物/无脲腔体/缘位错配体/β-CD 对照四组直接证伪单锚增量，可证伪性 8。先以可溶完整体比较 BPA/BPF/BPS 与酚再决定聚合，符合"可溶门先于材料冻结"纪律。

关键风险：深腔合成步骤多，固载化与床层传质困难；BPA 专属增量 tier none（深腔可能只产生广谱疏水包合）。

深化问题：缘脲深腔完整体对 BPA 的中性水增量是否超过开口类似物、无脲腔体与 β-CD，且对 BPF/BPS/酚不退化？若深腔只产生广谱疏水包合而无单锚增量，停止单锚主张。

#### 保留 5（最弱标记）｜BPA-2R β-CD 次级口定位单极性受体（63，partial，exact-rim-receptor-not-frozen）

存活理由：保留方案里总分最低、最不成熟，但唯一带 β-CD 生物主客体对应（β-CD 为生物寡糖，BPA 从次级宽口进入的结构与 D2O 中 1:1 Ka 约 3.62-4.10e3 M-1 均为方案内 verified）。同一次级口确定位单一极性受体是可证伪的共定位主张（完整体增量须超过未改性 β-CD 加单独极性单元统计加和），六字段在原理上齐备。保留它为的是保住 β-CD 生物学脉络并留一个尖锐的共定位证伪问题。

关键风险：确切次级口受体衍生物未冻结（冻结前不选交联剂、不写 SOP）；共定位增量 tier none；总分 63 远低于其余四保留方案。

深化问题：能否合成取代位点可审计的单一次级口极性受体 β-CD 衍生物，且其整 BPA 增量在水中超过未改性 β-CD 与单独极性单元的统计加和，并在 BPF/BPS/DOM 竞争中保留？若不能冻结确切衍生物或增量不超加和，BPA-2R 退出深化。

### 淘汰方案（15）与理由

按主导淘汰原因分组。控工程基线 BPA-2E 单列：不进入深化，但作为工程阳性对照保留。

- **BPA-2E 高 β-CD 自支撑网络（72，control-only）**：方案自定为工程下限与阳性对照，不主张仿生选择性，原创 6。不进入深化，保留为对照。
- **BPA-4 活性炭承载超薄 BPA 表面印迹层（75，partial）**：biological_prototype 字段为"水相表面印迹"，是材料技术不是生物学源，硬对应 biological_feature + source_support 不齐；原创 8，MIP-炭为成熟先例；模板泄漏超定量限与涂层炭孔容损失为真实风险。高可制造性不足以补偿对应缺口与低原创。
- **BPA-5 双功能化杯芳烃主客体网络（71，partial）**：杯[4]芳烃为合成大环，非生物源；BPA 专属水相包合 tier none；高成本与受体负载上限致总容量风险（capacity ceiling）。共定位增量未证实。
- **BPA-6 半共价硼酸酯印迹网络（61，control-only）**：化学窗口窄，中性水硼酸位点贡献未证实（tier none）；control-only，低优先级风险路线。
- **BPA-7 假模板双酚类别印迹整体柱（72，partial）**：假模板印迹是材料技术非生物源，硬对应 biological_feature 不齐；方案自限为双酚类别捕获，BPA 相对同类无额外优势，不满足"对 BPA 的选择性吸附"硬门到 BPA 专属层面。
- **BPA-8 缺电子芳环电荷转移/π-π 多孔有机聚合物（63，loose）**：NDI 缺电子性为 class 级（蛋白结合旁证），酚环 CT 在水中 tier none；loose 对应；选择性偏弱。
- **BPA-9 共价三嗪框架三嗪 N 酚氧氢键受体（65，loose）**：三嗪 N 氢键在水中 tier none，仅 g-C3N4 与氯代苯醌 DFT 类级旁证（不同污染物）；离子热合成能耗高；loose 对应。
- **BPA-11 本征微孔聚合物尺寸形状门（74，partial）**：分数高但 biological_prototype 字段为"扭曲骨架亚纳米微孔"，是材料尺寸门概念非生物源；PIM 本身对 BPA 尺寸选择性为 class 级推断，verified 先验是 Zr-MOF（不同材料）。硬对应 biological_feature + source_support 不齐，分数主要来自可制造性与可证伪性而非仿生对应。
- **BPA-12 柱芳烃主客体网络（69，partial）**：柱[6]芳烃为合成大环非生物源；BPA 专属包合 tier none；可聚合变体与固载化为风险。
- **BPA-13 脲/硫脲酚氧氢键多孔聚合物（62，loose）**：Hamilton 型脲受体为 class 级综述，水中对酚选择性 tier none；脲自缔合与水竞争为主不确定；loose 对应。
- **BPA-14 多孔有机笼定尺寸腔（64，loose）**：有机笼水相客体行为为 class 级（非污染物吸附），POC 对 BPA 尺寸包合 tier none；笼规模化与床层装填困难；loose 对应。
- **BPA-15 弱碱阴离子交换/疏水混合模式床（64，loose）**：原创 6 最低，无仿生对应；pH 窗口为操作约束非机制优势；选择性与普通阴离子交换难分。
- **BPA-16 N 掺杂层级多孔碳吡啶 N 氢键（66，loose）**：N 掺杂生物炭 BPA 传感吸附为 class 级，吡啶 N 氢键选择性 tier none；氮形态稳定性与 DOM 竞争；loose 对应。
- **BPA-17 石墨相氮化碳七嗪腔酚氢键（63，loose）**：仅 g-C3N4 与氯代苯醌 DFT 类级旁证（不同污染物），七嗪 N 对 BPA 酚氢键 tier none；粉体成型与水相传质困难；loose 对应。
- **BPA-19 碘代全氟芳烃卤键受体聚合物（59，loose，high-risk）**：水中卤键保留性 tier none，无已检验先例；总分最低；碘代单体成本与稳定性为主要风险。

### 淘汰总结

15 淘汰、5 保留。首要淘汰判据是硬对应规则的 biological_feature + source_support：BPA-4、BPA-7、BPA-11、BPA-15 等高分组因"生物原型"实为材料机制（印迹、尺寸门、阴离子交换）而通不过仿生对应，分数来自可制造性而非仿生保真度。BPA-5、BPA-12、BPA-13、BPA-14 为合成大环/笼受体，BPA 专属水相增量均为 tier none 且伴容量或固载化上限。BPA-8、BPA-9、BPA-16、BPA-17、BPA-19 的中心水相机制（CT、三嗪 N H 键、吡啶 N H 键、七嗪 H 键、卤键）均无水中已检验先验，loose 对应。BPA-6 化学窗口窄、control-only。BPA-2E 自定为工程阳性对照，不进入深化但保留为对照。

保留的 5 个里，BPA-3、BPA-1、BPA-10 共享 ERRγ-BPA 复合物这一经核验的承重生物学源（PDB 2E2R），构成 portfolio 内对应最强的一簇；BPA-18 经验先验最强但仿生对应缺口最大，须在深化中正视或改写；BPA-2R 为最弱标记，带 β-CD 生物主客体对应但衍生物未冻结、增量 tier none。无方案达 85 通过线，5 个保留方案均需深化补对应缺口或补水相证据后方可晋级。
