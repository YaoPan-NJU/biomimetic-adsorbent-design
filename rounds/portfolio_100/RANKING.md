# 全局排序与讨论集（portfolio_100 综合轮）

> **状态警告：** 本文件是 `GLM` 分支的原始探索排序，已由 `main` 的 `SUPERVISOR_AUDIT.md` 审计。分数、晋级身份和下一动作均不得直接继承；新的用户交付须使用“人工识别单元”等当前术语。

监督综合角色输出。跨五污染物（BPA、ROX、PFOA、PFBS、DCP）各 20 方案共 100 条，经各污染物轮淘汰至各 5 条保留方案后，在此做全局排序与讨论集切片。通过线 85，未决 critical 允许 0，硬对应规则要求 biological_feature 加 source_support 等六字段。本轮独立复核三处承重来源：PDB 2E2R（ERRγ 配体结合域与双酚 A，1.60 Å，Matsushima 等 2007 J Biochem，经 RCSB 核验为真）、PDB 1SBP（硫酸结合蛋白，1.70 Å，Sack 与 Quiocho 存库，经 RCSB 核验为真）、Zeppuhar 等 ACS AMI 2023（10.1021/acsami.3c12826，标题经 Crossref 核验为"Linkage Transformations in a Three-Dimensional Covalent Organic Framework for High-Capacity Adsorption of Perfluoroalkyl Substances"，为真）。Zeppuhar 先例所引容量值（PFOA 约 250 mg/g）与二甲铵阳离子静电机制归属来自上一轮全文查验，本轮出版商页返回 403，无法独立再确认，下文如实标注。

逐条机制、前景与待闭问题如下。

1. PFOA-FABP4（83）。机制：非肽双位点口袋复现 FABP4 共晶 9MIW 的 Arg126/Tyr128 头基锚定与双位点占位，把 PFOA 直接写入结构对应。前景：全表最高分，选择性 20/25 与因果 17/20 全表最高，共晶 9MIW 双位点占位有 tier-1 结构证据，apo 9OB7 对照存在，非肽受体路线落在默认合成材料范围内。待闭问题：非肽受体能否冻结并复现 Arg126/Tyr128 头基几何与双位点浓度依赖，使 9MIW 静态双位点对应从结构证据转译为材料层因果。当前受体几何未冻结，方案暂停。

2. PFOA-CD1（80）。机制：β-CD 阳离子缘多孔共聚物以 β-CD/PFOA 主客体加 DFB-β-CD 聚合物 tier-1 先例为对应基础。前景：转译 18/20 全表最高，离散 β-CD 自由基共聚直接成珠装柱工艺成熟，对照集可分离腔体与阳离子贡献。待闭问题：β-CD 腔体是否贡献超出阳离子缘离子对的 PFOA 相对 PFBA/PFBS 增量，且在混合 PFAS 加 DOM 中保留而不退化为普通疏水树脂。阳离子与交联剂可能是真正机制而非腔体主客体。

3. DCP-101（79）。机制：全氟碘代-脲双识别 COF 以脲氢键加全氟碘代卤键构成正交第二识别维度，区分 2,6 与 2,4/3,5-DCP。前景：本轮总分与选择性双最高（19），少数机制上可区分 2,6 与 2,4/3,5 而非仅随疏水性，亚胺/β-酮烯胺 COF 自支撑成型成熟，后修饰位点比例可由消解 NMR 与元素分析验证。待闭问题：全氟碘代给体对中性 2,6-DCP 邻位 Cl 的卤键是否真给出超过 2,4 与 3,5-DCP 的选择性，还是只随氯数与疏水性变化。水中阴离子卤键已证、中性芳烃邻位 Cl 卤键未证，二者不可等价，高概率触停门。

4. BPA-3（78）。机制：非印迹双端极性 COF 通道把 ERRγ 静态夹持（PDB 2E2R 经 RCSB 本轮复核为 ERRγ 配体结合域与双酚 A，1.60 Å）转译为水稳 β-酮烯胺 COF 两壁定距双极性站加疏水面。前景：总分最高且硬对应六字段齐备，水稳 COF 有 class 级先例，单壁/错位/无极性站/孔径匹配化学随机 COF 四组对照直接证伪夹持。待闭问题：量化双极性站真实位点比例下，整通道对 BPA 的中性水选择性是否超过 BPF/BPS 且在 DOM 竞争中保留，并在删除任一极性站或错位时增益崩塌。若增益只随疏水或总芳香量变化，双站夹持主张失败。

5. ROX-7（78）。机制：肟氮铜配位咪唑 MOF 以 ROX 独有 9-O-甲肟氮配位铜为专一落点，静态对应金属酶开放金属位配位识别。前景：证据分最高（8/10），Ozdemir 2006（Acta Crystallogr C）与 Buvailo 2010（Dalton Trans）经 PubMed 独立确认 O-甲肟氮配位铜，红霉素 9-酮无肟氮给体，配位差异最硬。待闭问题：大体积 ROX 9-O-甲肟醚在水稳定铜 MOF 孔道内能否几何到达开放铜位并配位，且在二级出水 EDTA、Cl⁻、pH 摆动下铜不溶出至消除选择性。HKUST-1 型铜 MOF 水稳定性差、铜溶出风险。

6. ROX-6（78）。机制：肟醚氢键受体超交联聚合物以 ROX 独有 9-O-甲肟醚为落点，定距双脲/酰胺对肟氮与醚氧给定向氢键。前景：因果 16、选择 18 为本轮并列最高，可溶完整体加肟位删除体加距离错配体的增量证明路径清晰，可证伪 9。待闭问题：在 pH 7 至 8 缓冲水、10 mM Ca/Mg、5 mg/L DOM 中，可溶双脲受体对 ROX 相对红霉素 9-酮是否给出非单调、随肟位几何变化的氢键增量，而非随总疏水/总氢键数线性变化。脲/酰胺对肟氮的中性水氢键强度未量化，tier none。

7. PFOA-COF（77）。机制：氢键共价有机骨架通道以定向脲/酰胺氢键站加受限低极性通道区别于主客体与印迹机制。前景：非生物合成骨架中因果闭合 16/20 全表最高之一，全合成、自支撑、无惰性重核，对照集可分离氢键定向与孔容。待闭问题：水稳定 COF 通道内定向氢键站相对等孔径化学随机 COF，能否在真实二级出水中保留可测 PFOA 头基读取增量，而不被水屏蔽或 DOM 堵孔降级为普通芳香吸附体。COF 用于 PFAS 与脲/酰胺氢键识别羧酸两条证据线索均 tier none。

8. PFOA-MIP1（76）。机制：PFOA 伪模板分子印迹聚合物以印迹腔形状按链长区分 PFOA/PFBA/PFBS，为混合 PFAS 提供独立选择性轴。前景：PFOS 表面印迹 tier-1 先例为邻近转译基础，伪模板规避 PFOA 本体泄漏假阳性，整体柱成型成熟，区别于只读头基的氢键路线。待闭问题：非含氟伪模板印迹腔在痕量浓度下是否对 PFOA 相对 PFBA/PFBS 产生可重复链长/形状增量，且盲测下识别窗口不因伪模板几何差漂离 PFOA。PFOA 伪模板专属研究 tier none，只能声称类别捕获。

9. DCP-109（75）。机制：硼酸酯可逆共价预组织 COF 在合成期以硼酸酯可逆共价锁定 2,6-DCP 几何，使用期水解去模板后依赖残留几何腔与非共价氢键。前景：机制独立于脲/酰胺氢键方案，把模板预组织与印迹泄漏在化学上分开，化学依据为硼酸酯可逆共价而非生物原型，可证伪 9，停止门锋利。待闭问题：模板彻底水解后、在中性二级出水中，残留几何腔是否仍保留 2,6 优于 2,4 的选择性，还是只在碱性或不现实 pH 出现，且硼浸出是否在出水限值以下。2,6-DCP 预组织专属性 tier none。

10. DCP-114（72）。机制：β-CD 二聚协同夹持以定距 β-CD 二聚加缘脲提供协同增量，收编 DCP-3 单 β-CD 专属性有限的死路。前景：证据基础为本轮最厚（CD-氯酚包结有 NMR lead，10.1023/A:1008150908997；CD 聚合物选择性吸附有 verified 来源，10.1039/d0cc04784h），合成概要强制先在可溶模型证明二聚体相对单体的协同增量再聚合。待闭问题：定距 β-CD 二聚相对单 β-CD 是否产生协同增量足以区分 2,6 与 2,4 及 3,5-DCP，而非仅提升总结合，且在该协同下受体负载是否仍给出可工程化的总容量。2,6-DCP 二聚协同 tier none。

11. BPA-1（70）。机制：ERRγ 非对称单酚锚定超交联多孔聚合物直接转译经核验的 ERRγ-BPA 复合物（PDB 2E2R，本轮复核结构为真），显式放弃对称双端夹持。前景：全 portfolio 仿生保真度最高，硬对应六字段齐备且 source_support 为 hard 级，P1-MAPy 可溶水相门把"单酚锚在水中是否真有增量"压到可证伪单点，门未过即不冻结材料。待闭问题：是否存在任一单酚极性锚在中性无聚集水相通过 P1 门。P1 水相门尚未通过，人工单酚锚在水中对 BPA 的选择性增量 tier none，受体未冻结，experiment-gated。

12. PFBS-11（69）。机制：尺寸门 β-酮烯胺 COF 以约 7 Å 一维通道窗口几何排除匹配 PFBS 全长、对 PFOS C8 构象惩罚。前景：总分 69 为 PFBS 组第一，3D COF-PFAS 直接先例经 Crossref 本轮复核标题为真（Zeppuhar 等 ACS AMI 2023），β-酮烯胺 COF 合成可行、无氟、可颗粒化装柱，几何非电荷原则上最能绕开已核验的 PFBS-1 盐反例。待闭问题：在 1 mM Na2SO4 与二级出水下，约 7 Å 窗口能否保留一个独立于壁面电荷贡献的 PFBS 优于 PFOS 且优于硫酸根的几何增量，还是坍缩为先例式静电增强。先例高容量（PFOA 约 250 mg/g）实为二甲铵阳离子静电作用的归属来自上一轮全文查验，本轮出版商 403 无法独立再确认，如实标注。

## 二、各污染物最佳候选

- BPA：BPA-3 非印迹双端极性 COF 通道（78）。ERRγ 静态夹持经 RCSB 本轮直接复核（PDB 2E2R，ERRγ 配体结合域与双酚 A，1.60 Å），硬对应六字段齐备且水稳 COF 有 class 级先例，对应与可制造性平衡最好。须以消解 NMR 与元素分析量化双极性站真实位点比例，并验证 DOM 不堵孔、删除任一极性站增益崩塌。
- ROX：ROX-7 肟氮铜配位咪唑 MOF（78）。Ozdemir 2006 与 Buvailo 2010 经 PubMed 独立确认 O-甲肟氮配位铜，证据分最高，红霉素 9-酮无肟氮给体使配位差异最硬。须冻结大体积 ROX 在水稳定铜 MOF 孔道内的几何可达性，并验证铜在 EDTA、Cl⁻、pH 摆动下不溶出。
- PFOA：PFOA-FABP4 非肽双位点口袋（83）。唯一直接把 PFOA 写入结构对应的主张，共晶 9MIW 双位点占位与 Arg126/Tyr128 头基锚定有 tier-1 结构证据，apo 9OB7 对照，选择性 20/25 与因果 17/20 全表最高。当前因非肽受体几何未冻结而暂停，须冻结受体几何并验证双占位浓度依赖，冻结后为唯一可信主方案候选。
- PFBS：PFBS-11 尺寸门 β-酮烯胺 COF（69）。3D COF-PFAS 直接先例经 Crossref 复核标题为真，约 7 Å 几何窗口原则上绕开已核验的 PFBS-1 盐反例，无氟可颗粒化。须证明几何增量独立于壁面电荷贡献；先例高容量的静电归属本轮无法独立再确认，如实标注。
- DCP：DCP-101 全氟碘代-脲双识别 COF（79）。脲氢键加全氟碘代卤键正交第二识别维度是少数机制上可区分 2,6 与 2,4/3,5 的方案，本轮总分与选择性双最高。水中阴离子卤键已证但中性 2,6-DCP 邻位 Cl 卤键未证且不可等价，为核心机制赌注，高概率触停门。

## 三、讨论集（建议优先讨论的 10 个）

选取对应、转译、选择性、可证伪四维组合最佳者，并附诚实告诫。其中先例拥挤、受体未冻结、对照邻近各已标注。

1. PFOA-FABP4（83）。告诫：唯一接近 85 的方案但受体几何未冻结而暂停；FABP4 对多种 PFAS 有可测结合，支持的是 PFAS 识别窗口而非 PFOA 单分子专属。冻结受体几何是晋级硬门。
2. PFOA-CD1（80）。告诫：β-CD/PFOA 主客体与 DFB-β-CD 聚合物先例均 tier 1，对应强但原创 10/15 偏低、先例拥挤。阳离子与交联剂可能是真正机制而非腔体主客体，须以等电荷等孔容随机体分离。
3. BPA-3（78）。告诫：ERRγ-BPA 复合物经 RCSB 直接复核，硬对应六字段齐备。限量后修饰位点非均一，须消解 NMR 与元素分析量化真实比例，否则增益可能只随疏水变化。
4. ROX-7（78）。告诫：铜肟氮配位经 PubMed 独立确认，证据最强且红霉素无肟氮给体。HKUST-1 型铜 MOF 水稳定性差、铜溶出风险，大体积 ROX 几何可达开放铜位未冻结。
5. DCP-101（79）。告诫：选择性最高且正交机制可区分异构体。水中阴离子卤键已证但中性 2,6-DCP 邻位 Cl 卤键未证为核心机制赌注，高概率触停门。全氟碘代芳烃后修饰合成负载重。
6. ROX-6（78）。告诫：ROX 专一肟醚落点且可证伪路径清晰。脲/酰胺对肟氮的中性水氢键强度未量化（tier none），与 ROX-13 同机制，深设计须二选一。
7. PFOA-COF（77）。告诫：非生物骨架中因果闭合最高且机制独立、自支撑无惰性重核。COF 用于 PFAS 与脲/酰胺氢键识别羧酸两条证据线索均 tier none，水中氢键被屏蔽或 DOM 堵孔为停止门。
8. BPA-1（70）。告诫：仿生保真度全 portfolio 最高且 source_support 为 hard 级。P1 水相门尚未通过，单酚锚水中 BPA 选择性增量 tier none，受体未冻结，experiment-gated，门不过即降级为机制探针。
9. PFBS-11（69）。告诫：已核验 COF-PFAS 先例且几何非电荷原则上绕开盐反例。先例高容量实为二甲铵阳离子静电的归属本轮无法独立再确认，窗口不排除小硫酸根须壁面化学协同。PFBS 组整体最弱（67 至 69）。
10. DCP-114（72）。告诫：证据基础本轮最厚（NMR lead 加 verified CD 聚合物来源）。二聚协同是否真区分 2,6 与 2,4/3,5 而非仅提升总结合未证，受体负载使总容量过低为工程天花板，2,6-DCP 二聚协同 tier none。

## 四、总体评估

无方案达 85 通过线，此为首要事实。

最高分 PFOA-FABP4（83）因非肽受体几何未冻结而暂停，同时是全 portfolio 唯一把目标污染物直接写入结构对应的主张，有 tier-1 共晶证据（9MIW）与 apo 对照（9OB7），冻结受体几何后存在可信路径达线，判为唯一可信主方案候选。PFOA-CD1（80）对应硬、转译最高、工艺最成熟，为先例拥挤但可工程化的可信备选或替代主方案，前提是能用等电荷等孔容随机体证伪"阳离子才是真正机制"。BPA-3（78）有经 RCSB 本轮直接复核的 ERRγ-BPA 复合物与齐备硬对应对，为 BPA 最强候选，仍低于线，须量化后修饰位点比例与 DOM 堵孔。ROX-7（78）证据分最高且配位差异最硬，铜 MOF 水稳定性为可寻址工程风险，须冻结大体积 ROX 几何可达性。

其余多为修订或对照邻近。DCP-101 与 DCP-109 为高概率触停门的机制赌注，核心机制（中性芳烃邻位 Cl 卤键、硼酸酯水解残留几何）均 tier none。BPA-1 为 experiment-gated 探针，门不过即不冻结材料。BPA-18（75，未入前 12）为 portfolio 内最大对应缺口，其 biological_prototype 字段实为材料机制（Zr-MOF 孔道、链接剂、锆簇），无生物学受体或酶源，按硬对应规则须诚实改写为非仿生经验路线并登记合约偏离，或找到可审计生物学对应并标明类比边界。PFBS 组整体最弱（67 至 69），五条保留均带某非电静电机制在水相与硫酸根竞争下是否足够强的待闭条件，且风险坍缩为已核验的 PFBS-1 盐反例。ROX-6 与 ROX-13 机制冗余，深设计须二选一。

与 portfolio_20 相比，本轮实质改进在于 PFOA 首次出现把污染物写入结构的方案（FABP4 的 9MIW 双位点），给了 PFOA 一条具体的达线路径。然而 85 门槛仍全线未达，绝大多数方案为修订或对照邻近，整体结构与上一轮相似。可信主方案仅 PFOA-FABP4 一枚（暂停待冻结），可信备选 PFOA-CD1 一枚，BPA-3 与 ROX-7 为各污染物内的最强修订候选。深设计须各自关闭命名硬门才有路径达线，失败即终止，不换原理挽救。
