# 20 方案前五名攻击

## 隔离说明

受当前执行环境限制，本文件由主执行者在方案冻结并提交后进行角色分离式攻击，不声称为独立代理盲审。原始 `RANKING.yaml` 不修改，所有扣分另存于 `POST_ATTACK_RANKING.yaml`。

## BPA-1｜ERRγ 刚性双端几何夹

### 高风险问题

1. `2E2R` 给出的是完整蛋白口袋内的多点环境，不等于两个脲/酰胺和芳香壁在水中就能恢复同等几何约束。
2. 超交联可能改变受体间距、埋藏位点或产生大量随机疏水微孔，使表观 BPA 优势不能归因于预组织受体。
3. 高分建立在“可聚合刚性受体能够合成”这一尚未冻结的假设上；目前没有确切二维结构、收率或可达位点预测。
4. BPA/BPF/BPS 的尺寸和双酚端点接近，水中氢键削弱后，中心桥连差异可能不足以形成工程选择性。

### 必须关闭

- 先完成可溶完整体、单端删除体和距离错配体；
- 完整体必须在水相混合竞争中显示超出疏水性和总氢键数的几何增量；
- 聚合前后用位点滴定证明受体没有被超交联埋藏；
- 总材料容量和床层容量不能只按受体摩尔数归一化。

**处置：** `revise`，88 降至 84；仍为组合首位，但未通过 85 分门槛。

## PFBS-1｜阳离子浅腔 β-CD 共聚物

### 高风险问题

1. 直接先例表明短链 PFAS 捕获主要依赖静电，复杂盐基质会显著削弱；β-CD 对 PFBS 的独立腔体贡献没有被证明。
2. “浅腔 β-CD”表述过度，β-CD 固有腔深不能通过普通共聚任意缩短；需要用 α/β/γ-CD 系列实证，而不是命名。
3. 永久阳离子位点可能被硫酸盐、DOM 和阴离子表面活性剂占据，方案可能退化为普通阴离子交换。
4. 即使 PFAS 总去除率高，也可能仍是 PFOS/PFHxS 晚穿透而 PFBS 最早失效。

### 必须关闭

- 在真实二级出水中同时做 PFBS/PFPeS/PFHxS/PFOS 突破；
- 以无腔等电荷聚合物拆分 CD 腔体贡献；
- 报告盐和 DOM 对 PFBS 选择性而不是总去除率的影响。

**处置：** `revise`，82 降至 74。

## PFBS-2｜胍基–双脲头基笼/C4 链槽

### 高风险问题

1. 胍基/脲对磺酸盐在水中的结合可能被强水化和普通阴离子竞争压制。
2. 非氟化低极性槽对短 C4 全氟链的增益可能太小；加入含氟域又会带来材料生命周期和广谱 PFAS 分配问题。
3. 刚性受体分子量可能很高，降低 mmol 位点/g 总材料并重现“识别漂亮、容量很低”的问题。
4. 当前没有直接人工受体先例证明 C4 槽长可对 PFBS 产生非单调优选。

### 必须关闭

- 可溶 C2/C4/C6 系列必须先通过盐水和二级出水竞争；
- 完整体相对等电荷无序体产生交互增量；
- 在冻结结构时计算每克总材料的理论位点上限。

**处置：** `revise`，80 降至 73。

## BPA-2｜β-CD–双氢键协同多孔珠

### 高风险问题

1. β-CD 和 BPA MIP 均属于拥挤先例，新增双氢键单体可能只是普通共聚配方优化。
2. 羧酸/酰胺位点在中性水中的电离和水竞争可能使“双氢键”无法保持设计几何。
3. 随机共聚不能保证极性站位于 β-CD 孔口；没有空间共定位证据时不能声称协同口袋。
4. BPA、BPF 和 BPS 均可能进入 β-CD 并使用相似极性位点。

### 必须关闭

- 使用离散的“β-CD–定距极性站”单体，或将主张降级为统计协同；
- 证明完整材料存在超过两个单功能材料加和的交互增量；
- 真实水中直接比较 BPA/BPF/BPS。

**处置：** `revise`，79 降至 74。

## ROX-1｜COF 薄层表面印迹颗粒

### 高风险问题

1. 2026 年已有同类 DES–COF 表面印迹 ROX 材料，本方案若只增加二级出水实验，原创材料贡献不足。
2. 已报道容量和印迹因子来自近期单项研究，不能视为可重复工程基线。
3. ROX 真模板泄漏会在痕量 LC-MS 分析中制造假去除或假回收；大分子模板完全洗脱困难。
4. COF 加印迹层可能增加制造复杂度和成本，但并未证明比自支撑普通 MIP 或活性炭更优。
5. 与多种大环内酯的真实竞争选择性尚未建立。

### 必须关闭

- 给出超越 2026 论文的结构创新，而不是仅更换成型方式；
- 使用同位素内标和全过程模板质量平衡；
- 与普通 MIP、非印迹 COF 和 GAC 在同一二级出水中比较。

**处置：** `control_only`，78 降至 67；可作高成功率工程基准，不宜作为原创主方案。

## 总攻击结论

- 前五中没有一个达到“85 分且无高风险问题”的终审条件。
- `BPA-1` 是最值得继续的设计，但首先需要确切非肽受体和可溶几何验证。
- `PFBS-1` 是最有直接材料证据的 PFBS 路线，但盐/DOM 击穿风险高。
- `PFBS-2` 是原创性最高的 PFBS 路线，但受体化学和比重容量风险也最高。
- `BPA-2` 可作为较稳妥备份，但必须证明共定位协同而非普通配方效应。
- `ROX-1` 更适合作为工程/先例基准；原创路线应回看 `ROX-2`，但后者尚未完成攻击。
