# S11 独立攻击：FABP4 双位点、Phe57 局部门与定向固定化

## 攻击边界

本攻击只审查当前设计是否已形成可制造、可因果裁决的实验方案，不评价预期结果，也不沿用旧 S1 的分数或状态。允许证据支持 `9MIW` 晶体中两个 PFOA、主/次位点残基以及 Phe57 相对 apo `9OB7` 的局部侧链状态差异。晶体观察不能自动外推为溶液中可分辨的两阶段结合、固定态口袋朝向、PFOA 特异性或工程选择性。

当前方案比人工 portal 路线更接近直接生物原型，但仍有阻断性问题。序列和载体前体化学存在可立即修正的错误；溶液双占位、Phe57 因果状态以及真实竞争选择性则必须由实验决定，不能在设计文件中关闭。

## Critical

### C1. 冻结 FABP4 序列与 `9MIW` 不一致，构建体身份尚未成立

设计列出的核心序列在第 126/127 个 FABP4 字符处为 `ST`，而 `9MIW` 实际顺序为 `TS`。这不是格式问题。方案同时把 Arg126、Tyr128 作为主位点因果残基，并要求沿用 PDB 编号；一处相邻残基转置会改变后续编号对应、突变引物、理论质量和结构映射。当前“核心序列冻结”与“正式下单前再次核对”互相矛盾，不能把错误序列视为已冻结实体。

**纯设计可修复：** 以 `9MIW/9OB7` 的实际聚合物序列为唯一来源，逐位校正，明确结构中缺失或未建模残基、克隆残基与天然 FABP4 编号的映射；随后重新生成所有 WT/突变体 FASTA、理论质量、引物表和哈希。修订完成前不得订购或表达。

**实验要求：** 校正后仍须以完整质量质谱和肽段覆盖确认表达产物身份，但该实验不能替代设计时的序列纠错。

### C2. 晶体中的两个 PFOA 不等于溶液中存在可分辨的两阶段占位

`9MIW` 支持一个晶体状态中的主、次两个配体位置。该观察没有直接给出溶液中的两个解离常数、逐级化学计量、第二阶段浓度窗口、占位顺序或协同性。晶体浸泡/结晶浓度和晶格环境还可能稳定低占有率状态。设计把“低占位 PFOA”“高占位 PFOA”“第二位点占据区间”写入 HDX、突变和动态通过门，却没有先定义如何从溶液数据识别两个位点，也没有说明一位点、两独立位点、序贯位点和非特异分配模型如何竞争。

若溶液中只能得到一个表观结合过程，T29A、F57A/W 引起的变化无法被指定为“第二位点”效应，静态双位点与动态门的共同因果链均失去主要终点。

**纯设计可修复：** 在看数据前冻结逐级结合模型集合、活性蛋白浓度测定、直接化学计量方法、浓度范围选择规则、模型选择标准、参数可辨识性门和全局拟合方案。不得在发现单阶段曲线后把任一高浓度偏差事后命名为第二位点。

**必须实验：** 用至少一种直接计数占位的方法和一种正交平衡方法证明溶液中第二配体可重复、可饱和且可由 T29A 或相应位点扰动区分。没有该结果时，S11 只能保留单口袋 FABP4 吸附假说。

### C3. F57A/F57W 不是锁态，现有突变不能单独承担动态因果对照

设计正确承认 F57A 不是锁开、F57W 不是锁闭，但随后仍用两者对“Phe57 空间门”作主要因果裁决。F57A 同时删除芳香表面和大量疏水体积，F57W 同时增加体积、极化性和潜在局部堆积；两者都可能改变腔体自由能、配体接触、水网络和全蛋白稳定性。即便主位点表观结合仍在，第二配体变化也不能唯一归因于 apo/配体态之间的 Phe57 侧链状态转换。

此外，T29A 与 F57 突变“效应落在同一因果链”没有预注册的数学定义。相似方向、非加和或上位性分别对应不同结论，不能在结果出现后选择解释。

**纯设计可修复：** 将 F57A/W 明确降为扰动筛选，而非锁态因果对照；预先定义 F57 侧链反应坐标、状态占比、保守突变候选筛选规则和 `T29 × F57` 双突变循环/上位性统计式。若没有能够保持口袋化学而改变状态分布的构建体，设计中不得声称已有匹配锁态。

**必须实验：** 对每个可用 F57 构建体直接测定 apo 与不同占位下的侧链构象群体，并证明主位点几何、折叠与活性浓度可比。若所有突变只表现为腔体体积或疏水性变化，动态硬对应缺少因果控制，按合同退出主备竞争。

### C4. HDX-MS 的局部肽变化不能证明 Phe57 侧链 open/closed

HDX-MS通常报告肽段尺度的主链酰胺交换，受溶剂暴露、氢键、呼吸运动、聚集和配体保护共同影响。即使获得包含 Phe57 的肽段，局部 HDX 差异也不等于 Phe57 芳香侧链采用 `9OB7` 或 `9MIW` 的转子状态。设计表格允许“NMR/结构读出”，但阶段 A 和通过门实际把“Phe57 区域 HDX 变化”作为进入固相的充分条件，测量与所声称动态坐标不匹配。限量蛋白酶和 QCM 更不能补足侧链分辨率。

**纯设计可修复：** 冻结一个能分辨 Phe57 侧链坐标或经过结构标定的主方法，并给出残基级读出、状态判定区间、占比和重复性门。HDX-MS可保留为支持局部环境变化的辅助方法，不能独立通过动态硬对应。

**必须实验：** 取得 WT apo、第一占位区间、候选第二占位区间及关键突变体的残基级结构/谱学证据。若技术上只能得到肽段 HDX 或总体荧光，Phe57 动态主张不能成立。

### C5. `H2N-GGG-PEG-NH2` 与 NHS 琼脂糖的反应不能保证留下可用的 N 端 GGG

Sortase 受体需要自由的寡甘氨酸 N 端。设计用两端均为胺的 `H2N-GGG-PEG-NH2` 直接处理 NHS 琼脂糖，两个胺都可与 NHS 酯反应，可能消耗 GGG 的 N 端、通过另一方向偶联或造成桥连。乙醇胺封闭只能终止剩余 NHS，不能纠正已形成的错误拓扑。因此当前步骤不能制造定义明确的 `GGG-PEG-agarose`。

**纯设计可修复：** 改用具有正交保护或正交偶联端的寡甘氨酸间隔臂，明确哪一端连接载体、何时脱保护以及如何定量自由 GGG N 端；同时列出表面 GGG 密度、残余活化基团和 Sortase 工艺空白的验收方法。

**实验要求：** 修订后必须测定可用 GGG 密度、实际连接化学、Sortase 特异连接比例和未共价吸附蛋白。当前路线在化学上修正前不能进入固定化筛选。

## High

### H1. C 端连接方向确定，不等于口袋朝外或孔内可达

Sortase 可把每个蛋白的 C 端连接到载体，但柔性 `GSG-LPET-GGG-PEG` 允许蛋白旋转；琼脂糖孔壁没有统一法向，C 端相对 FABP4 入口的空间关系也没有在设计中量化。因而“定向固定避免口袋遮挡”只能作为待测假说。定向组优于随机 NHS 组也不能证明口袋朝外，因为两组的多点交联、构象刚性和蛋白损伤不同。

**纯设计可修复：** 删除“口袋朝外”的预设；根据结构坐标冻结 C 端到口袋入口的距离、PEG 长度范围、载体孔径与蛋白尺寸要求，并增加相同 C 端连接但不同间隔臂长度的可达性对照。随机偶联只作为工程参照，不作为方向因果对照。

**必须实验：** 分别测总固定蛋白、可折叠活性蛋白、位点可达比例和孔内扩散，并验证这些量随 PEG 长度和载量的关系。只有连接位点已知，不能证明固定后口袋可用。

### H2. FABP4 的广谱脂质/PFAS结合可能使选择性目标在二级出水中根本不成立

允许证据本身说明 FABP4 广谱结合 PFAS，天然功能又使脂肪酸成为强竞争风险。设计提出链长窗口或头基窗口是可检验假说，但没有证据保证 C8–C10 PFCA 相对更长链 PFCA、PFSA、油酸、棕榈酸、胆汁酸和表面活性剂形成可用窗口。`PFOA不优于PFNA/PFDA不判失败`进一步扩大了事后改口空间；如果比较对象可在 A/B 两种口径和多个组分中选择，容易把普通疏水链长偏好包装为选择性。

**纯设计可修复：** 在解盲前为口径 A、B 各指定一个主分析物、固定比较集合和联合检验，明确 PFDA、PFOS、油酸及 LAS/SDS 不能被排除；将单调随链长增强的结合预注册为普通广谱分配，而非窗口选择性。

**必须实验：** 在实测自由活度和环境相关比例下完成 PFAS、天然脂质、表面活性剂及 DOM 同时竞争。若窗口只在等摩尔纯水、没有天然配体或高于环境占位浓度时出现，S11 不能作为二级出水选择性吸附 finalist。

### H3. “可用 FABP4 位点数”未定义，归一化可能循环使用目标结果

`E_i/j` 以“每 nmol 可用 FABP4”归一化，固定态比较又要求按可用位点数归一化，但设计没有给出与 PFOA 结果独立的活性位点测量。若用 PFOA 或同一置换探针估计活性比例，再用其校正 PFOA/PFAS 结合，突变体和固定化组的效应可能被校正过程放大或抹除。双位点体系还不能简单把“一个蛋白”当作统一可用位点。

**纯设计可修复：** 冻结总蛋白量、正确折叠蛋白比例、主位点活性浓度和第二位点化学计量的独立测量与误差传播；明确容量比较按哪一个分母，选择性比值是否需要该分母，以及突变体缺失位点时如何归一化。

**必须实验：** 证明活性浓度测定不依赖待检选择性结果，并在各构建体、载量和基质中重复测量。

### H4. 预实验冻结点覆盖了决定可行性的核心参数，当前尚非实验就绪 SOP

表达温度和诱导时间作为工艺优化项可以后置；但当前未冻结的内容还包括正确核心序列、脱脂条件、缓冲液、载体孔径、PEG 结构与密度、GGG 表面化学、Sortase 条件、固定化密度、直接占位方法、Phe57 主结构方法、PFAS浓度、环境比例、自由浓度分离方法、统计模型、最小效应来源、柱流速和再生方案。若这些参数由预实验逐项选择，样品和比较数量会持续变化，也可能在看到目标结果后选择条件。

**纯设计可修复：** 把冻结点分为制造开发、方法资格和正式假说检验三张表。为每项给出允许候选范围、与目标 PFAS 结果隔离的选择指标、最大迭代次数、停止规则、锁定时间和变更审计。正式实验之前必须产生一份具体批记录与预注册分析计划。

**必须实验：** 蛋白折叠、脱脂后活性、双位点可辨识、固定化活性和侧链读出均属于可行性门。未通过这些预实验前，当前文件是开发计划，不是可直接交给正式比较的实验 SOP。

### H5. 预注册选择性门仍允许事后胜者和任意阈值

`口径A或口径B至少一个`、`R126A/Y128F/R106A至少一个`以及多个竞争物组合形成较大的结果选择空间。`E≥3`、突变后下降 50%、真实水活性保留 50%等阈值没有与方法重复性、空白分布、环境需求或功效分析连接。三批材料不等于独立生物重复，也没有多重比较规则和置信区间要求。

**纯设计可修复：** 冻结首要口径、首要突变、首要竞争集合、统计单位、批次层级模型、置信区间、多重比较、最小有意义效应来源及功效规则。其余结果明确列为次要或探索性终点。不能仅凭点估计跨门。

## Medium

### M1. 主位点突变不都是静态几何的干净删除

R126A改变电荷与水化，Y128F改变氢键但保留芳环，R106A可能重排水网络；任一变化都可能影响多个 PFAS 和天然脂质。方案承认这一点，但“至少一个显著削弱”仍不足以证明原结构中的特定锚定关系。

**纯设计可修复：** 为每个突变冻结局部结构保持判据，并预注册互补保守突变或双突变循环。将突变结果解释为残基贡献，而非自动等同于单一盐桥或氢键删除。

**必须实验：** 需要结构/谱学与热力学共同确认局部扰动范围。

### M2. 脱脂步骤可能重写后续比较的起始状态

设计拟复现 butanol 脱脂，但条件未给出，且用“脱脂前结合活性”作为比较基准会受未知内源脂质占位影响。不同突变体对内源脂质和 butanol 的耐受可能不同，造成表观位点差异。

**纯设计可修复：** 冻结各构建体残余脂质的测量、统一放行标准和溶剂工艺空白；不得仅凭总质量或 CD 判定脱脂等效。

**必须实验：** 用脂质/PFAS 靶向分析确认残余占位，并证明脱脂批次不造成构建体特异失活。

### M3. 固态 Phe57 动态读出缺乏可执行路径

“固相用 HDX-MS兼容条件或限量蛋白酶/QCM辅助”没有说明如何从琼脂糖上回收、淬灭、消化并保持时间分辨，也没有说明 QCM 如何区分构象与质量变化。固相动态主张目前无法被这些方法直接验证。

**纯设计可修复：** 若固相侧链分辨方法不可行，应把动态硬对应的证明严格限定在可溶构建体，并把固相突变效应写为外部功能验证；不要把 QCM 或蛋白酶敏感性描述为 Phe57 状态测量。

### M4. 再生筛选引入与选择性机制相冲突的变量

脂肪酸竞争洗脱可能残留并占据 FABP4，高盐或有机溶剂可能改变蛋白及载体。设计虽设结构保持门，但没有规定再生后残留竞争剂、残余 PFAS、活性位点和空白漂移如何共同放行。

**纯设计可修复：** 定义每轮质量平衡、残留脂肪酸/PFAS、蛋白构象、活性浓度和空白载体的联合放行标准。若不能再生，应按既定规则降为一次性富集介质，不能以更强洗脱继续优化至失活。

## 问题关闭顺序

1. **先修正文档即可关闭的问题：** 校正 WT 与全部突变体序列和编号；用正交保护化学重写 GGG–PEG 载体制造；冻结占位模型、统计主终点和 Phe57 残基级主测量；删除“C端连接即口袋朝外”的预设。
2. **再完成可溶态实验门：** 证明校正构建体单分散且脱脂等效，溶液中存在可分辨的第二配体占位，WT Phe57 在相关占位区间发生残基级状态变化，并找到不以总体积/疏水性解释的动态因果扰动。
3. **随后完成固定化可行性门：** 证明 C端连接化学、活性位点可达、孔内传质、蛋白泄漏与基质稳定性；定向连接本身不能替代这些测量。
4. **最后裁决选择性：** 在环境相关自由活度、天然脂质、表面活性剂、混合 PFAS 与真实二级出水中检验预注册窗口。广谱结合、单调链长偏好或天然配体压倒目标占位均应终止选择性材料主张。

在 C1 与 C5 完成设计修正，且 C2 至 C4 获得实验支持以前，S11 不具备实验就绪的静态加动态硬对应。上述实验若失败，不能把同一 FABP4 证据改写为合成门或仅靠调整载体继续维持 finalist。
