# S11 Gate 1 文档与制造修订

## 1. 修订状态

本文件关闭可由设计本身关闭的序列、编号、表面化学、模型选择和结果选择自由度问题。它不关闭溶液双占位、Phe58 构象群体、固定化活性、真实水选择性或再生性能；这些仍属于预实验和正式实验门。

## 2. 序列与编号冻结

### 2.1 唯一序列源

- 天然序列：UniProt `P15090`，132 aa，N 端 `M1` 至 C 端 `A132`。
- PDB `9MIW/9OB7` 聚合物：`GH` 两个克隆残基 + `P15090 M1–A132`，共 134 aa。
- PDB author numbering 把天然起始 Met 标为 `0`。因此论文/PDB author 编号相对 UniProt 编号统一加 1。

| 论文/PDB author 编号 | UniProt/构建体编号 | 本项目构建体名 |
|---|---:|---|
| Thr29 | Thr30 | T30A |
| Phe57 | Phe58 | F58A、F58W |
| Arg106 | Arg107 | R107A |
| Arg126 | Arg127 | R127A |
| Tyr128 | Tyr129 | Y129F |

旧设计稿中的 `R126A/Y128F/R106A/T29A/F57A/F57W` 仅作为论文编号标签保留，禁止直接用于引物或基因下单。旧稿序列末端的 `...KGVSTT...` 为转录错误，正确序列为 `...KGVTST...`。

### 2.2 成熟 Sortase 底物

成熟 WT 蛋白冻结为：

```text
GH + P15090(M1-A132) + GSG-LPETG-His6
```

N 端表达纯化标签必须通过 TEV 完全切除，使成熟产物以 `GHM...` 起始。Sortase A 在 `LPET|G` 处切割，固定化产物保留 `FABP4-GSG-LPET-GGGGC-PEG-carrier`。八个成熟底物的逐条 FASTA 与按“纯氨基酸字符串、不含换行”计算的 SHA-256 见 `S11_CONSTRUCTS.fasta`。基因供应商文件必须通过仓库校验脚本逐位比对；理论质量和引物表在选定表达载体后生成并作为采购放行记录，不能从错误旧稿复制。

## 3. 正交 GGG 表面化学

旧稿的 `H2N-GGG-PEG-NH2 + NHS-agarose` 路线删除，因为两个胺均可能消耗 NHS，不能保证 Sortase 所需的自由 N 端。

冻结替代路线：

1. 使用氨基化大孔交联琼脂糖，先与异双功能 `NHS-PEG24-maleimide` 反应；NHS 端连接载体氨基，maleimide 端保留。
2. 加入 `H2N-GGGGC-CONH2`。末端 Cys 的硫醇与 maleimide 加成，N 端 `GGGG` 保持自由，作为 Sortase 受体。
3. 以 cysteine 封闭残余 maleimide；制备没有 `GGGGC` 的 PEG/maleimide 工艺空白和仅加 cysteine 的封闭空白。
4. 用荧光或 LC-MS 可追踪的 Sortase 小分子供体测定可用寡甘氨酸密度；投料量不得代替表面可用密度。
5. 正式批次放行要求：自由寡甘氨酸密度批间 CV ≤15%；未反应 maleimide 低于方法定量限；无 FABP4 的完整工艺空白对主分析物吸附不超过 WT 材料的 20%。

具体 PEG 面密度、反应当量和时间仍由不接触 PFAS 选择性结果的表面分析预实验选择。选择指标只允许使用自由 GGG 密度、非特异蛋白吸附、孔内可达性和批间重复性。

## 4. 双位点模型预注册

可溶态 WT、T30A 和 R127A 的 PFOA 数据统一比较四个模型：单一可饱和位点、两个独立位点、序贯两位点、一个可饱和位点加线性非特异分配。直接化学计量主方法为 native MS 或平衡透析后 LC-MS/MS 的蛋白结合计数；正交平衡方法为 ITC 或经活性浓度校正的荧光竞争。

模型选择在解盲前冻结为：

- 同一批数据全局拟合 WT/T30A/R127A；
- 参数必须有有限 95% 置信区间，第二位点占位数不得靠近参数边界；
- 两位点模型相对最佳一位点模型需满足 `ΔAICc ≥ 10`，且留一浓度层级预测误差下降至少 25%；
- 第二占位须在两种独立方法中出现，并被 T30A 优先削弱；
- 高浓度线性上升、聚集或容器/DOM 分配不得计作第二位点。

任一条件失败，双位点主张终止。不得用更换浓度范围或删除高浓度点恢复两位点模型，除非删除规则在解盲前已由质量控制触发。

## 5. Phe58 动态坐标与因果控制

F58A/F58W 降级为腔体/门区域扰动体，不再承担锁开或锁闭对照。动态主链冻结为：

`独立确认第二占位 -> WT Phe58 残基级状态变化 -> T30A 消除第二占位并同步消除该状态变化`。

主测量采用芳香残基选择性标记的溶液 NMR。方法资格必须先完成 F58 峰归属，并以 apo `9OB7` 和 PFOA-bound `9MIW` 的结构坐标设计可区分两种 Phe58 局部接触的 NOE/化学位移组合；单一峰位移动或肽段 HDX 不得独立通过。F58A/F58W 用于峰归属和功能扰动，T30A 是第二占位因果对照，R127A 是主位点上游对照。

若 NMR 不能给出残基级、可重复且与占位相关的状态分布，则允许转用同一构建体的 apo/分级占位晶体或其他残基级结构方法，但必须在看选择性结果前完成方法资格。HDX-MS、限量蛋白酶和 QCM 仅为辅助证据。

动态通过须同时满足：WT 在独立确认的第二占位区间出现 Phe58 状态群体改变；T30A 在折叠和主位点可比时同时削弱第二占位与该状态改变；F58 扰动改变第二占位功能，但其效应不得仅由总体稳定性或主位点损失解释。未满足时，S11 只能保留静态 FABP4 生物杂化假说。

## 6. 固定唯一主选择性终点

首要污染物固定为 PFOA。首要竞争选择性终点固定为相同羧酸头基、相同碳数的氟化效应：

```text
E_primary = (q_PFOA / q_octanoate) / (Cfree_PFOA / Cfree_octanoate)
```

主比较只使用 WT、R127A、T30A 和空白载体；主机制判据为 WT 的 `E_primary` 及其在 R127A 或 T30A 中的预注册下降。PFNA/PFDA、短链 PFCA、PFSA、GenX、油酸/棕榈酸、胆汁酸和 LAS/SDS 全部保留在固定竞争集合，但作为关键次要终点，不允许从中事后挑选“最佳窗口”。单调链长增强只报告为广谱疏水/腔体适配，不称为选择性窗口。

正式统计单元为独立表达批次与独立固定化批次的嵌套批次，不是同批技术复孔。至少三批材料；报告批次层级模型的效应量和 95% 置信区间。通过阈值仍为 `E_primary ≥ 3`，且相对 WT 的突变效应下降至少 50%，但只有当 95% 置信区间下限分别高于 1 和 0 时通过。关键次要集合采用预先冻结的多重比较校正。

环境相关验证必须同时保留 PFDA、PFOS、油酸、棕榈酸、胆汁酸、LAS/SDS 和 DOM，不得因干扰强而删除。自由浓度由无蛋白平衡和目标体系分相/超滤资格实验后 LC-MS/MS 或相应定量方法实测，名义投料不代替自由活度。

## 7. Gate 1 与 Gate 2 边界

### Gate 1 文档与制造放行

- 本文件、`S11_CONSTRUCTS.fasta`、载体正交化学和模型/统计预注册必须经独立复审；
- 选定载体和表达质粒后补齐供应商、货号、理论质量、引物、反应当量和批记录模板；
- 任何序列或主终点变更必须新增修订记录，不覆盖旧稿。

### Gate 2 可溶态预实验

- 正确构建体通过身份、单体、折叠和脱脂等效性；
- 两种方法确认可分辨第二占位并由 T30A 优先削弱；
- 残基级方法确认 Phe58 状态变化与第二占位一致；
- WT 在固定竞争集合中显示预注册的 PFOA/辛酸氟化效应，并且不是蛋白外表面或聚集造成。

Gate 2 未通过不得开展固相材料优化。项目本轮不实际执行这些实验，因此 S11 在 Gate 1 修订后仍应标记为 `pause_for_preexperiment`，不得列为已通过主方案或备选方案。

### 8.2 双位点唯一方法与浓度设计

- **直接占位主方法：** native MS。蛋白总浓度固定为 `5.0 µM`，PFOA:蛋白摩尔比固定为 `0、0.1、0.25、0.5、0.75、1、1.5、2、3、5、10、20、50、100、200`。每个层级使用三份独立表达批次，每批两次技术进样。
- **正交平衡方法：** 平衡透析后同位素稀释 LC-MS/MS。蛋白总浓度固定为 `2.0 µM`，目标自由 PFOA 浓度层级固定为 `0、0.01、0.03、0.1、0.3、1、3、10、30、100、300、1000 µM`。三份独立表达批次，每层级每批两个独立透析单元。
- **平衡与质量平衡 QC：** 无蛋白单元、膜吸附空白、容器空白和双侧质量平衡必须同时通过；回收率 `80–120%`，两个连续时间点自由浓度变化 `<5%`。不通过的整个层级按盲化 QC 删除，不得只删除不利于两位点模型的重复。
- **活性浓度：** 总单体由氨基酸分析校准的 `A280 + SEC-MALS` 获得；正确折叠比例由预先资格化的 ANS 1:1 饱和滴定估计。ANS 数据只用于折叠/主口袋活性分母，不参与 PFOA 模型选择。native MS 报告 `0/1/2+` 配体物种分数；平衡透析报告每个总折叠单体的结合量。
- **脚本接口：** 输入长表列固定为 `batch,construct,method,total_protein_uM,target_free_uM,molar_ratio,replicate,occupancy_0,occupancy_1,occupancy_2plus,bound_uM,qc_status`；输出必须同时给出四模型参数、95% CI、AICc、留一浓度层级预测误差、残差和参数边界标志。模型规则沿用第4节。

任何 PFOA 浓度点若出现无蛋白损失、聚集或离子化抑制，只能按上述盲化 QC 处理。不得新增浓度层级或改换 ITC/荧光法来挽救两位点结论。

### 8.3 Phe58 唯一主方法与状态门

唯一主方法冻结为芳香 Phe 选择性 `13C/1H` 标记的溶液 NMR，不设开放式备选。F58 峰通过 WT 与 F58A 的谱差归属。基于 `9OB7/9MIW` 坐标，在正式滴定前冻结至少两组区分 Phe58 inward/outward 局部接触的芳香 NOE 或等价残基级距离约束；只观察化学位移变化不通过。

状态分析固定为两构象群体拟合，报告 open-like 群体分数及 95% CI。三份独立表达批次分别测 apo、主位点区间、已由占位实验确认的第二位点区间；WT、T30A 和 R127A 使用相同条件。方法资格要求两组独立距离约束给出一致方向，批间群体分数 CV `<15%`，留一约束预测不改变结论。

动态通过阈值冻结为：WT 从 apo 到第二位点区间的 open-like 群体增加至少 `0.30`，95% CI 下限大于 `0.15`；T30A 保留至少 `70%` 的 WT 主位点活性时，其群体变化不超过 WT 的三分之一；T30A 对第二占位的削弱与群体效应方向一致。F58A/F58W 仅作峰归属和功能扰动，不计作锁态。NMR 方法资格失败即 Gate 2 失败，不得临时换晶体学、HDX、QCM 或总体荧光维持动态主张。

### 8.4 唯一主因果链和统计功效

唯一首要选择性终点仍为 `E_primary(PFOA/octanoate)`；唯一首要因果突变固定为 **T30A**。首要通过要求：WT 的 `E_primary ≥3` 且 95% CI 下限 `>1`；T30A 相对 WT 的 `E_primary` 下降至少 `50%`，其下降比例的 95% CI 下限 `>0`。R127A 是预注册的关键次要静态对照，必须降低 PFOA 主位点占位，但不参与“任一突变成功即通过”的选择。

统计单元为独立表达并独立固定化的材料批次。初始 `n=4` 批，每批技术三复孔。方法资格阶段只用合并盲码方差、不得查看组均值，检验 log 尺度 `ln(3)` 的 WT 富集效应和 `ln(2)` 的 T30A 下降效应能否在双侧 `α=0.05` 达到 `80%` 功效；不足时最多增至 `n=6`。若 `n=6` 仍不足，判为不可稳健验证，不继续扩大样本追逐显著性。

### 8.5 独立分母与误差传播

总蛋白、折叠主口袋活性、第二占位化学计量分别报告，不合并为一个“可用位点数”：

1. `nmol total monomer/g`：氨基酸分析校准 A280，并以 SEC-MALS 单体分数修正；
2. `nmol folded primary pocket/g`：ANS 1:1 饱和滴定，在无 PFOA 条件下测定；
3. `second occupancy per folded monomer`：仅由通过 QC 的 native MS 和平衡透析模型给出，不用于重新缩放同一 PFOA 数据；
4. `E_primary` 为同一材料内两分析物比值，不再按 PFOA 推导的活性分数二次校正。

批次层级模型同时输入蛋白定量、自由浓度和 LC-MS/MS 测量误差，使用参数自举传播 95% CI。若 ANS 与结构/SEC 对折叠比例的判断冲突，该批不进入正式选择性比较，而不是选择对 PFOA 更有利的分母。

### 8.6 固定载体开发和真实竞争条件

PEG 间隔臂只比较 `PEG8、PEG24、PEG48` 三档，蛋白表面投料只比较 `0.2、0.5、1.0 µmol/g湿载体` 三档。选择完全隔离于 PFOA 结果：以自由 GGG 密度、Sortase 连接率、ANS 主口袋可达率、SEC/泄漏和无关蛋白非特异吸附排序；最多一轮九组合筛选，锁定后不得因 PFOA 结果更换。

机制竞争液固定为自由浓度：PFOA 与 octanoate 各 `100 nM`；PFDA、PFOS 各 `100 nM`；油酸、棕榈酸、胆汁酸、LAS、SDS 各 `1.0 µM`；标准 DOM `5 mg C/L`。真实二级出水保留原生 DOM 和共存物，PFOA、octanoate、PFDA、PFOS 标准加入至各 `100 ng/L`；原生脂肪酸、胆汁酸和表面活性剂只测定不调平。若原水本底高于加标值，以实测总浓度为准，不稀释到有利水平。
