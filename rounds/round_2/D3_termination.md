# Round 2 D3 终止审计：PFOA / hL-FABP

## 1. 终止结论

D3 终止终选资格并转入静态可行性归档。OT11/OT13 支持 PFOA/PFNA 与 hL-FABP 结合、Asn111/Arg122 参与及 PFCA 链长谱；OT12 的动态只属于油酸体系的 portal/gap 局部迁移率，并明确没有整体显著构象变化。当前没有 PFOA 或 PFNA 结合引起 hL-FABP 门户状态变化的直接证据。人工柔性门户即使可运动，也只能证明通用材料入口响应，不能关闭项目要求的生物动态硬对应。因此 ORG-R1-D3-01 仍为未关闭 critical，D3 不得继续占用终选候选名额。

## 2. 静态双位点裁决

“外酰胺初定位、内胍基驻留”在设计层面可形成可判别实验，但尚未形成证据。归档实验固定四种受体：完整双位点、去外酰胺、去内胍基、外内位置互换，并增加等总胍基/酰胺随机层。所有版本须配平工作 pH 电荷、湿态孔容、亲水性、可接近位点和无目标扩散。

采用浓度跃迁与短脉冲并预先比较单/双相模型。仅当完整材料出现双相，去外酰胺改变第一相、去内胍基改变驻留/解吸，且互换破坏该表型，才支持两阶段；若效应只随总电荷变化则否决。静态成立也不能恢复动态硬对应。

## 3. 链长谱裁决

归档竞争集固定为 PFHxA、PFOA、PFNA、PFDA，并加入 PFOS、脂肪酸和阴离子表面活性剂；在活度匹配混合物中测分配、突破和腔深扰动。

hL-FABP 对 C4–C11 PFCA 的亲和总体随链长增加。若限长腔使 PFOA 优于 PFNA/PFDA，只能称 FABP 启发的人工链长选择性；若仍按链长或疏水性排序，PFOA 专属静态路线终止。固定化蛋白只按活性位点比较链长趋势。

## 5. 保留内容与重开条件

仅保留双位点静态受体、PFAA 链长谱、模板空白和真实水竞争方法；可形成普通 PFAS 材料研究，不得以 hL-FABP 动态仿生进入 finalist。

恢复 D3 完整资格必须新增直接证据：在一致条件下获得 apo/PFOA/PFNA/油酸 hL-FABP 门户构象分布或动力学，证明 PFOA 相关变化具有位点依赖性；材料端还须出现同方向、单位占位归一且由化学可比锁定门户消除的距离—选择性/抗置换链条。仅新增 FRET、QCM-D、刷层响应或更好的静态选择性不能重开。

**最终状态：terminated as finalist / archived static feasibility。**
