# S1 人工材料路线恢复简报

## 1. 决定

本轮恢复最初的“生物机制提取—人工材料转译”路线。hL-FABP、人体组织分布和其他生物证据用于发现并约束识别机制，不作为默认材料构件。

第三轮双螺旋肽 portal 方案和 S11 FABP4 生物杂化方案完整保留为历史审计记录，但不再是当前活动路线。它们没有成为主方案或备选，也不得因已有序列、结构或 Gate 记录而自动恢复。

当前活动候选记为 **S1-SYN**：hL-FABP 启发的非蛋白人工 PFOA/长链 PFCA 选择性吸附材料。S1-SYN 尚未通过设计审查，当前文件只冻结方向、材料边界和下一轮必须交付的合成设计内容。

## 2. 要解决的问题

主目标不是提高纯水最大吸附容量，而是在市政二级出水中保留可解释的竞争选择性：

- 优先检验 PFOA 相对短链 PFCA、PFSA、天然脂肪酸、阴离子表面活性剂和 DOM 的选择性；
- PFOA 与 PFNA 若不能分离，允许把主张收窄为 C8 及邻近长链 PFCA 的非单调选择性窗口；
- 若结果只随链长、总正电荷或疏水性单调增加，判为普通吸附，不保留仿生主张；
- 实际二级出水中的抗竞争能力是主性能，容量、再生和稳定性是工程约束。

## 3. 可以转译的生物信息

### 3.1 组织层

肝组织或其他组织中的污染物检出、富集和毒性可以用于发现候选结合环境，但不能单独指定责任蛋白。血清蛋白结合、膜脂分配、转运体、肾脏重吸收和缓慢消除均可能影响组织分布。

### 3.2 分子层

hL-FABP 对 PFOA/PFNA 的直接结合、PFAA 链长依赖以及 Asn111/Arg122 的贡献，支持以下人工材料约束：

1. 羧酸头基需要空间邻近的胍基和酰胺型相互作用；
2. 全氟链需要有限长度的低极性容纳区域；
3. 头基站与链段容纳区域必须共定位，不能退化为随机正电荷与随机疏水基团；
4. 材料外界面需要亲水抗污背景，以降低 DOM 和天然两亲物的非特异占位。

这些信息不支持复制完整蛋白口袋、引入蛋白序列、制造锁开/锁闭肽门户，或声称 PFOA 诱导 hL-FABP 闭合。

## 4. 第一代材料边界

### 4.1 载体

第一代载体固定为 **OEG 钝化的贯通介孔硅微球**。选择该载体是为了获得可测孔结构、成熟表面硅烷化路线和可控低密度位点，而不是模拟蛋白外形。

### 4.2 人工受体

受体固定为 **非蛋白、非折叠肽的小分子或短非肽寡聚配体**，并通过单一锚定位点接枝。受体必须同时包含：

- 一个可定量的胍基；
- 一个与胍基保持冻结间距的酰胺型位点；
- 一个有限长度、可制备长度错配对照的低极性链段容纳单元；
- 一个不参与结合的载体锚定端。

下一轮必须选择一个确切化学结构并给出完整制造表。在结构冻结前，不允许把“胍基/酰胺/低极性槽”写成已经可合成的最终受体。

### 4.3 明确排除

- hL-FABP、FABP4 或其他完整蛋白；
- 表达质粒、蛋白突变体和蛋白固定化；
- 依赖折叠形成口袋的长肽、双螺旋肽和 β-发夹；
- 为满足评分而加入的通用柔性铰链、锁开/锁闭桥或响应聚合物；
- 可浸出的含氟模板或游离含氟功能单体；
- 以投料量代替可接近位点数。

## 5. 合成路线恢复要求

下一轮只推进一条人工材料路线，并按以下顺序形成可审查的实验前 SOP。

### 阶段 A：非肽受体冻结

在任何 PFOA 结果产生前提交：

- 确切二维结构、立体化学和理论分子量；
- 胍基、酰胺、链段容纳单元和锚定端的合成来源；
- 完整体、头基删除体、链槽删除体、空间错配体、短槽体和长槽体的对应结构；
- 每一步反应物、当量、溶剂、温度、时间、纯化和身份确认方法；
- 水相溶解、聚集和化学稳定性验收门。

如果完整体与对照无法做到总电荷、低极性基团数量和锚定方式可比，停止该受体，不用蛋白或肽替代。

### 阶段 B：可溶模型验证

先比较完整体与空间错配体、头基删除体、链槽删除体和长度错配体。采用 LC-MS 竞争分析，并使用至少一种独立热力学或光谱方法确认结合。首批分析物固定为 PFOA、PFNA、PFHxA、PFBA、PFOS、PFBS、辛酸、油酸和一个阴离子表面活性剂。

只有在完整体表现出不能由总电荷或疏水性单调解释的几何增量时，才进入载体固定化。可溶模型失败时不允许通过提高表面密度、降低盐度或改变载体补救。

### 阶段 C：载体表面构建

实验前 SOP 必须给出：

1. 介孔硅清洗、干燥和批次验收；
2. 低密度反应性硅烷与 OEG 硅烷的共接枝比例；
3. 剩余硅醇和未反应基团的封闭方法；
4. 人工受体单点偶联反应及洗涤终点；
5. 元素分析、TGA 或等效方法测得的真实负载量；
6. 小分子探针或滴定得到的可接近头基站数；
7. BET/孔径、ζ 电位、接触角、浸出和水相稳定性。

所有核心材料必须使用同一载体批次，并匹配孔结构、表面电位和有效位点数。

### 阶段 D：真实水竞争验证

纯水只用于机制控制。主验证必须在同一批市政二级出水中完成，保留原生 DOM 和共存物，并加入冻结浓度的 PFCA、PFSA、脂肪酸和阴离子表面活性剂。自由浓度由 LC-MS/MS 实测，不能用名义投料浓度替代。

主结果报告：

- PFOA/预指定竞争物的竞争选择性系数及置信区间；
- 完整体相对空间错配体的选择性增量；
- 头基删除、链槽删除和长度错配造成的效应损失；
- 选择性从超纯水到二级出水的保留比例；
- 竞争条件下的有效容量、穿透、再生、浸出和结构稳定性。

## 6. 最小因果对照

| 对照 | 唯一改变 | 排除的替代解释 |
|---|---|---|
| 完整体 | 正确头基站和链槽几何 | 候选材料 |
| 头基删除体 | 去除胍基/酰胺协同 | 普通羧酸静电吸附 |
| 链槽删除体 | 打断有限低极性区域 | 普通疏水分配 |
| 空间错配体 | 官能团数量不变、位置错开 | 总组成效应 |
| 短槽体/长槽体 | 只改变容纳长度 | 链长几何窗口 |
| OEG 载体空白 | 不含受体 | 载体和 DOM 分配 |
| 普通阳离子疏水材料 | 匹配电荷和疏水量 | 常规吸附基准 |

不再设置动态体、锁开体和锁闭体作为通过门。以后若提出材料动态，必须先证明静态人工材料不能解决的选择性问题，并由 Pan Yao单独批准扩展。

## 7. 通过、降级和停止

### 可进入下一轮审查

- 已冻结一个确切非肽受体和完整 SOP；
- 完整体相对匹配对照具有几何选择性增量；
- 真实二级出水中仍保留方向一致的竞争选择性；
- 蛋白、肽折叠和动态门户未被重新引入。

### 主张降级

- PFOA 与 PFNA 不可区分但相对短链 PFCA 和非 PFCA 两亲物形成非单调窗口：改为“长链 PFCA 窗口选择性”；
- 真实水中选择性降低但仍优于常规基准：报告基质敏感的人工识别材料，不声称强抗干扰；
- 仅容量增加：报告普通吸附性能，退出仿生候选。

### 立即停止

- 结果只随链长、logKow、总正电荷或位点密度单调变化；
- 空间错配体或普通阳离子疏水材料复制完整效果；
- 只有低盐、低 DOM 或有机助溶剂条件下存在选择性；
- 受体结构无法冻结或对照不可制造；
- 为挽救结果重新引入蛋白、折叠肽或无直接必要性的动态单元。

## 8. 唯一下一动作

由独立设计者依据本简报提交一个 **确切的非肽人工受体＋OEG 介孔硅固定化 SOP**，包括全部结构、试剂、当量、步骤、对照和身份确认方法。随后由独立攻击者只审查化学可制造性、对照可比性、真实水选择性和规模化边界。

在该设计通过前，不订购、不合成、不开展 S11 Gate 1b 或 Gate 2，也不把任何历史方案恢复为主方案或备选。
