# S1 Round 3：hL-FABP静态识别＋iLBP portal动态的双原型PFOA门户受体

## 1. 允许主张与禁止主张

本方案明确使用两个生物来源：

1. **静态来源：hL-FABP。** PFOA/PFNA与hL-FABP结合、PFAA链长依赖以及Asn111/Arg122相关作用，用于约束“外部羧酸定位＋受限链段容纳”的静态识别布局。
2. **动态来源：iLBP家族portal。** iLBP家族的helix-II、CD/EF loop或portal区域在apo/holo状态表现出局部动力学差异，用于约束“局部柔性入口在配体占位后稳定预存构象”的动态设计原则。

禁止写成：PFOA已被证明诱导native hL-FABP门户闭合；hL-FABP的毫秒运动已被证明是PFOA进入的必要步骤；材料FRET或选择性结果可以反向证明蛋白发生同样运动。

## 2. 冻结论文问题

在相同静态位点化学和可接近位点数下，iLBP式局部门户的构象自由度是否与hL-FABP式头基—链段几何产生交互，使PFOA/PFAA亚群在脂肪酸、胆汁酸、阴离子表面活性剂、其他PFAS和DOM竞争中获得不能由总电荷、疏水性、位点数或传质解释的选择性增量？

动态单元只提高总容量、平衡速率或位点可达性时，双原型主张失败。

## 3. 材料构件

### 3.1 可溶母体受体

先以SPPS制备单链双螺旋—环肽门户受体，不直接从固相材料起步。母体由四段构成：

- **H1头基螺旋：** 一个Asn样酰胺和一个Arg胍基位于同一面，模拟外部羧酸定位；相邻位点用Ala/Ser控制水相暴露。
- **H2链槽螺旋：** Leu、Phe和环己基丙氨酸样低极性侧链组成有限长度槽，不使用可浸出的含氟残基。
- **portal铰链：** Gly-Ser-Ala-Ser-Gly类亲水短环连接H1/H2，使两螺旋在不解聚的范围内采样开放和靠近状态。
- **固定化尾：** 远离入口的C端azidolysine或同等单点点击位点，用于后续固定到DBCO-OEG多孔琼脂糖/硅胶。固定化尾不得参与结合。

确切序列、螺旋长度和桥跨度必须由Round-3计算/实验预检查冻结；本文件不预写未经验证的唯一氨基酸序列。

### 3.2 同一母体的portal四元组

母体在H1/H2非结合面设置一对不参与PFOA接触的Cys。四个版本尽量保持序列、净电荷和疏水残基数一致：

| 版本 | 实现 | 用途 |
|---|---|---|
| `P-Flex` | 两Cys以不成桥的惰性硫醇帽封端，portal保持可动 | 完整双原型候选 |
| `P-Open` | 使用冻结的长跨度双马来酰亚胺桥，把无配体主分布固定在开放距离 | 动态必要性主对照 |
| `P-Closed` | 使用短跨度桥固定为靠近状态，但必须保留与P-Flex可比的静态位点可达性 | 区分动态自由度与单纯窄口袋 |
| `P-NoDoor` | 删除H2移动门，但以等长度、等组成的惰性螺旋固定在远离入口的位置 | 检验门户本体相对静态位点的贡献 |

长/短桥不是按预期结果挑选。先用无配体结构分布冻结桥长；任一桥造成分子间交联、聚集、静态站破坏或可接近位点不可比，整个动态硬对应停止，不另换桥补救。

### 3.3 静态几何系列

每个portal状态至少配对以下静态版本：

- `G-Correct`：Asn样酰胺/Arg头基站与冻结链槽几何完整；
- `G-HeadΔ`：删除或中性化头基站，净电荷通过非结合面补偿；
- `G-SlotΔ`：链槽低极性面打断，疏水残基数在非结合面补偿；
- `G-Scramble`：元素组成、净电荷和疏水残基数相同，空间方向扰乱；
- `G-LengthMismatch`：保持官能团但缩短或延长链槽，检验长度错配。

核心最小析因组为`P-Flex/P-Open/P-Closed × G-Correct/G-Scramble`；删除体用于确认静态头基与链槽分工。

## 4. 合成与固定化路线

1. Fmoc-SPPS合成母体及静态变体；以RP-HPLC、HRMS、氨基酸分析和Ellman法确认纯度、分子量和自由巯基。
2. 在稀释条件下分别完成长桥和短桥的分子内双加成；用SEC-MALS、DOSY-NMR、非还原LC-MS和分析HPLC排除分子间交联。`P-Flex`使用同批母体进行惰性封端。
3. 先在可溶态完成结合、构象和析因门；通过后再用C端单点点击固定到DBCO-OEG大孔载体。随机多点固定禁止进入主路线。
4. 固定化后测实际受体负载、可接近位点、孔结构、表面电位和残余反应基团；不以投料量代表有效位点。

任何未通过可溶态门的序列不得通过提高固相密度、降低盐度或加入有机助溶剂补救。

## 5. 无标签构象与等效性检查

### 5.1 无标签构象

- 可溶态用圆二色谱、DOSY/SEC-MALS和同位素标记NMR/ROESY或等价距离约束确认单分子折叠、开放/靠近分布和桥状态。
- 固定态用固态NMR距离约束、SAXS/SANS或经验证的无标签结构方法至少一种确认portal状态；寿命FRET只作辅助读出，且必须进行标签互换。

### 5.2 进入主比较的等效性标准

各核心版本必须预先满足：

- 净表面电位和离子交换容量处于预注册等效区间；
- BET/汞压入或适配载体的孔结构指标处于等效区间；
- 总受体负载和以小分子探针/滴定得到的可接近头基站数处于等效区间；
- SEC-MALS/DOSY或固相散射不显示差异聚集；
- 以非结合示踪物测得的外部膜传质和孔内有效扩散处于等效区间。

具体等效界值在看到PFOA结果前由方法学重复性冻结。任一关键等效性失败时，对应版本不能用于动态因果裁决。

## 6. 竞争体系与实验顺序

### 阶段A：可溶受体静态门

用同位素内标LC-MS/MS与ITC/SPR测PFOA、PFBA、PFHxA、PFHpA、PFNA、PFOS、PFHxS、PFBS、油酸和代表性胆汁酸。`G-Correct`必须相对`G-Scramble`呈现不能由链长/logKow单调解释的差异；头基或链槽删除必须给出方向一致的损失。

### 阶段B：可溶受体动态析因门

在相同静态几何内比较P-Flex、P-Open、P-Closed；同步测无标签构象、FRET和结合/置换动力学。材料发生运动但没有竞争选择性交互，不算通过。

### 阶段C：固定态混合竞争

混合液包含：

- PFCA：PFBA、PFHxA、PFHpA、PFOA、PFNA；
- PFSA/醚型：PFBS、PFHxS、PFOS、HFPO-DA；
- 生物/工程干扰物：油酸、棕榈酸、代表性胆汁酸、SDS或LAS；
- 基质：实际二级出水DOM、主要无机阴离子及pH/离子强度范围。

先做批量竞争，再做等床停留时间的动态穿透。纯水容量不参与主要通过判据。

## 7. 预注册主终点

对每个竞争物`j`定义混合竞争选择性`α(PFOA/j)`。以对数选择性构造交互量：

`I_j = [log α(Flex, Correct) - log α(Open, Correct)] - [log α(Flex, Scramble) - log α(Open, Scramble)]`

主假说是动态状态与正确静态几何存在正交互。设计级通过条件在数据解盲前冻结为：

1. 至少一个短链PFCA、一个非PFCA PFAS和一个脂肪酸/表面活性剂类别的`I_j`方向一致，预注册置信区间下限高于零；
2. 效应在匹配可接近位点数、孔结构和传质后仍存在；
3. `P-Closed`不能仅以更低可达位点复制结果；
4. `G-HeadΔ`和`G-SlotΔ`必须削弱交互，证明动态依赖正确静态几何；
5. 置换/穿透结果与批量平衡选择性方向一致。

为避免用任意微小显著性包装机制，正式实验前根据空白和技术重复冻结最小有意义效应；若无法在不看目标结果时冻结，则不进行通过裁决。

## 8. 成功、降级和终止

### 完整通过候选

portal四元组、静态几何系列和等效性检查全部可制备；交互主终点通过；无标签构象与辅助动态读出一致；真实二级出水中仍保留方向一致的竞争选择性。

### 主张降级

- PFOA与PFNA无法区分，但对短链PFCA和非氟两亲分子存在非单调选择性：目标降为“C8及邻近长链PFCA亚群”，同步修改题目和论文主张。
- 正确静态几何通过但交互量失败：保留hL-FABP启发的静态工程对照，退出完整仿生finalist。

### 立即终止

- 无法制备单分子、静态可比的P-Open/P-Closed；
- 动态差异由位点数、孔结构、聚集或传质解释；
- 选择性只随PFAS链长/logKow单调变化；
- G-Scramble或普通等电荷疏水材料复制完整效果；
- 油酸、胆汁酸、表面活性剂或DOM产生同等portal信号且无法区分；
- 只有降低盐度、提高受体密度或加入有机助溶剂才能观察机制。
