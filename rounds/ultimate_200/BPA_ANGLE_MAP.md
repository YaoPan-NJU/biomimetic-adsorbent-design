# BPA 角度地图（设计空间 → 100 方案生成矩阵）

> 目的：在框架约束下系统铺满 BPA 设计空间，使 100 个方案机制可区分、非重复、可漏斗。本文件是 `BPA_PORTFOLIO_*.md` 的推理脚手架。

## 1. 污染物锚定事实（用于倒推痛点）

- BPA（4,4'-异丙基双酚），中性双酚，`pKa₁≈9.6–10.2`，两个对位酚羟基 + 异丙基螺中心，`MW 228.29`，`logKow≈3.3`。
- 结构近似竞争物：BPF（亚甲基桥）、BPS（砜桥）、BPAF（六氟异丙基）、4-枯基酚（单酚）、壬基酚、EE2、天然酚、腐殖酸/DOM。
- 关键几何（已核，`E023/E033`）：ERRγ-BPA 复合物 PDB 2E2R（1.60 Å），BPA 两酚氧 `O–O 9.274 Å`、双环平面锐角 `~87.4°`；突变证据支持"同一酚羟基由 Glu275/Arg316 互补夹持"为主锚，`4-α-cumylphenol` 单羟基仍强结合。
- 场景量级（`V2`）：厦门 7 WWTP 中位进水 1318 ng/L、出水 177 ng/L；主目标场景为**市政二级出水（ng/L–低 µg/L）**。

## 2. 工程难题清单（思路二，痛点从仿生能力倒推）

| 编号 | 工程难题 | 传统方法为何原理上做不到 |
|---|---|---|
| E1 | 中性水中判别 BPA 与 BPF/BPS/BPAF | 常规疏水分配按 logKow 单调排序，双酚同系物疏水性接近，无判别维度 |
| E2 | ng/L 痕量传质驱动力不足、床层早穿透 | 微孔活性炭在分钟级空床接触时间下远未平衡，颗粒内扩散是瓶颈 |
| E3 | DOM 污堵与小级分位点竞争 | 疏水位点被 NOM 占据后容量骤降，无尺寸/化学选择性界面 |
| E4 | 位点可逆再生（温和、不破坏识别几何） | 强再生条件破坏预组织识别单元 |
| E5 | 惰性载体质量惩罚 | 低负载接枝把 mg/g 总材料容量压到位点密度支配 |
| E6 | 分子级识别几何的可制造性（不复制蛋白折叠） | 溶液自由基聚合无法保证 Å 级位点空间排布 |

## 3. 生物原型清单（思路一；须真实存在且进化解决类似问题）

### 3.1 蛋白原型（识别/催化）

| 原型 | 进化功能 | 可转译设计原理 | 证据状态 |
|---|---|---|---|
| ERRγ LBD (2E2R) | 核受体配体识别 | 静态非对称单酚互补主锚 + 疏水包围（**仅取静态几何，不取毒性语义/不取动态闭合**） | verified（`E023/E033/E039`） |
| 漆酶 laccase（多铜氧化酶 T1 位） | 氧化酚类木质素（**自然界处理酚的真实方案**） | T1 Cu(II) 对酚羟基的配位/单电子识别 + 疏水底物槽 | 机制 class 级已知；具体 BPA 复合物待核（tier lead） |
| 锰过氧化物酶 MnP / 木质素过氧化物酶 | 氧化降解木质素与酚 | Mn(III)/卟啉边缘对酚的氧化识别 | class 级（tier lead） |
| 细菌 BPA 单加氧酶 / CYP450（Sphingomonas BisdB） | 微生物 BPA 降解首步 | 血红素远端疏水口袋对 BPA 芳环定位 | 存在性 verified、几何待核（tier lead/none） |
| 酪氨酸酶 / 儿茶酚氧化酶（Type-3 双铜） | 氧化单/邻酚 | 双铜中心对酚羟基定向配位 | class 级（tier lead） |

> 注：ERRγ 为毒性受体，用它有思路三气味；本轮把**降解酶底物口袋**列为承重原型的首选，ERRγ 仅作静态几何对照与已验证锚。

## 6. 家族级痛点-仿生因果链（写作锚点）

- **A**：BPA 难判别根因是缺乏对"酚羟基电子环境"的专属读出；自然界用多铜氧化酶 T1 Cu 以配位+单电子转移识别酚，我们提取"Cu(II)-酚定向配位 + 疏水底物槽"原理，在活性炭/聚合物上固定单核 Cu(II)-N 螯合位点，获得对酚（尤其可氧化双酚）优于惰性竞争物的配位判别，且可耦合氧化预浓缩。
- **B**：BPA 与 BPF/BPS 疏水近似，根因是无固定几何的极性读出；ERRγ 预组织静态口袋以单酚互补主锚+疏水包围+非对称第二接触区分，我们在水稳 COF/HCP 刚性通道复现定距极性站与非对称判别面。
- **D**：ng/L 传质驱动力小是床层寿命根因；木质部多级分支以连续孔径梯度降低传质能垒，我们在分级孔活性炭复现介孔-微孔耦合。
- **E**：DOM 污堵是真实出水失效根因；细胞膜两性离子水化界面排斥大分子粘附，我们在识别相外接枝稀疏两性离子层维持痕量 BPA 位点可达。
- **F**：分子级识别几何可制造性差；生物矿化/自组装以模板获得刚性有序孔，我们用硅/肽/CNC 模板固化识别位点间距。
- **H**：真实出水芳香背景高；真黑色素以 π 堆叠+酚氢键+金属螯合成为天然广谱芳香吸附相，我们仿造聚多巴胺/单宁相并叠加判别维度。

## 7. 家族级已知风险（继承 §4 死路，逐族预挂）

- A：Cu 溶出与二级出水 EDTA/Cl⁻ 竞争；催化氧化在无 H₂O₂/O₂ 供给下能否发生；须证明配位判别非普通疏水分配。
- B：后修饰位点非均一；水中氢键被屏蔽/ DOM 堵孔降级为普通芳香吸附；BPA-3 谱系须量化真实位点比例。
- C：合成大环非生物源（原创/对应受限）；β-CD 广谱、先例拥挤、容量天花板。
- D/E：传质/抗污为通用工程增量，须证明与 BPA 选择性正交且非单纯孔容/亲水性。
- F：模板残留与规模化；周期性阵列的识别增量须超过等孔容随机体。
- G：印迹为材料技术非生物源，硬对应缺口，须诚实标注为杂化或非仿生。
- H：天然多酚相广谱吸附，BPA 专属增量须独立证明。
- I：定向氢键在水中被竞争；须非对称设计避免退化为广谱酚吸附（P1-MAPy 教训）。
- J：pH 响应/离子交换为操作约束非机制优势，作对照或增强，不单独承重。
