# BPA 组合攻击/淘汰记录（Ultimate 轮，隔离攻击者）

角色：隔离攻击者/排序者。输入：`BPA_PORTFOLIO_01-50/51-100.md` 100 方案、`research_contract.yaml`（通过线 85/0 未决 critical/high；选择性吸附硬门；硬对应 ≥1 静态且六字段齐备）、`INNOVATION_CHECKLIST.md`、继承死路（`00_DESIGN_BRIEF.md` §4.1）。仅用 Read/推理，不杜撰 DOI、数值、机制；BPA 专属水相增量未独立核到者标 `tier none`。

## 0. 攻击主轴

1. **硬对应 biological_feature + source_support**：把"材料机制冒充生物原型"作首要淘汰判据（G 印迹族、J 工程族、部分 F 模板族）。
2. **思路三过滤**：ERRγ 为毒性受体（B 族仅取静态几何，边界须显式声明）；A 族降解酶为真实处理方案，思路三气味低，优先。
3. **选择性单调性（创新清单 D）**：若排序可由 logKow/电荷/分子量单调解释即淘汰或降对照（多数 D/H/J 广谱条）。
4. **继承死路重演**：对称夹持、酰氨吡啶=羧酸受体、β-CD 广谱/天花板、水中卤键/CT、抗污纯 Cai 2017 重叠——命中即降级。
5. **水相增量 tier**：所有承重条的 BPA 专属水相选择性增量当前 tier none，须可溶门/固相证据；据此定"深化必答唯一问题"。

## 1. 保留方案（7，含 1 最弱标记）

### 保留 1｜U01 单核 Cu(II) 螯合活性炭（漆酶 T1 配位识别，79，partial）
- **存活理由：** 本轮最强新仿生对应。承重原型为漆酶多铜氧化酶（**自然界真实氧化酚的进化位点**），规避 ERRγ 毒性受体的思路三气味；两个硬对应（Cu(II)-酚配位 + 疏水底物槽）六字段可齐备；**活性炭为框架首选材料**，可制造性、水稳、成本全项目最佳；提供一个 logKow 无法解释的正交识别维度（可氧化酚的配位判别）。
- **关键风险：** Cu 溶出与二级出水 EDTA/Cl⁻ 竞争可能消除配位判别；Cu(II)-酚在中性水的判别强度 tier none；须证明非普通疏水/普通 IMAC 分配。
- **深化必答唯一问题：** 固定单核 Cu(II)-N 位点对 BPA 相对 BPF/BPS/苯酚/4-枯基酚的配位判别，能否在中性二级出水（含 EDTA/Cl⁻/DOM）保留、且删除 Cu 或饱和 Cu 后增益崩塌？若增益只随疏水或总 Cu 量变化，配位判别主张失败。

### 保留 2｜U13 非对称单酚锚 + 桥基判别 β-酮烯胺 COF（80，partial）
- **存活理由：** 本轮总分最高。相对 BPA-1（对称夹持已否）与 BPA-3（对称双极性站）的实质增量是**非对称主锚 + 显式桥基判别面**：一壁定距互补主锚读一个酚氧，对壁疏水凹面读异丙基桥体积——把"分 BPA/BPF/BPS"从疏水性转到几何。水稳 COF 有 class 级先例，四组对照（无锚/错位/无凹面/化学随机）直接证伪。
- **关键风险：** 后修饰位点非均一（须消解 NMR/元素分析定量）；水中氢键被屏蔽或 DOM 堵孔降级；桥基凹面判别增量 tier none。
- **深化必答唯一问题：** 在量化真实位点比例下，桥基判别凹面是否使 BPA 优于 BPF/BPS 且在 DOM 竞争中保留，删除主锚或填平凹面时增益崩塌？

### 保留 3｜U24 BPA-3 增量：双极性站 COF + 位点定量协议（79，partial）
- **存活理由：** 直接承接全项目 BPA 最高分 BPA-3（78），把其唯一待闭问题（限量后修饰位点非均一）写进设计（强制消解 NMR/元素分析定量 + 四组对照内建）。ERRγ 静态夹持 source_support 为 hard（PDB 2E2R 已核）。
- **关键风险：** 与 BPA-3 机制同源，增量主要在方法学（位点定量）而非新机制；若定量后发现真实双站比例极低，则退化为 BPA-3。
- **深化必答唯一问题：** 定量真实双极性站比例后，整通道对 BPA 中性水选择性是否超 BPF/BPS 且 DOM 中保留？（与 U13 二选一或合并，避免机制冗余。）

### 保留 4｜U18 非对称双端极性 COF（77，partial）
- **存活理由：** 直接纠正"对称双端夹持"死路——显式设主锚强/次锚弱。删除次锚增量不塌陷即证非对称主张失败，可证伪性强。
- **关键风险：** 与 U13/U24 同属 B 族 COF 机制族，须去重。
- **深化必答唯一问题：** 非对称（强主锚 + 弱判别次锚）相对对称双站是否给出可测的 BPF/BPS 判别增量？
- **族内去重：** B 族 COF（U13/U18/U24）深化只保留 U13 为承重、U24 为方法学对照，U18 降为机制对照。

### 保留 5｜U23 单酚锚 + 两性离子抗污外层（B×E，78，partial）
- **存活理由：** 正面处理 DOM 污堵这一真实二级出水失效根因；识别（单锚）与抗污（细胞膜界面）双原型、双硬对应。
- **关键风险：** 抗污原创受 Cai 2017 约束（须证明非纯抗污）；外层传质阻力；小级分 DOM 仍竞争。
- **深化必答唯一问题：** 稀疏两性离子外层能否在真实 DOM 背景下维持单锚位点对 BPA 的可达性与选择性增量（相对无外层核），且不只是降低总容量？

### 保留 6｜U09 Cu(II)-联吡啶超交联聚合物（A 族自支撑，75，partial）
- **存活理由：** A 族配位判别的自支撑高位点版本，规避活性炭 Cu 负载密度上限；无惰性核。
- **关键风险：** 同 U01 的 Cu 溶出/竞争；HCP 合成步骤多。
- **深化必答唯一问题：** 同 U01（配位判别是否在盐/DOM 保留、删 Cu 崩塌）；作 U01 的材料形态替代对照。

### 保留 7（最弱标记）｜U25 β-CD 次级口单极性受体（BPA-2R 确切衍生物增量，66，partial）
- **存活理由：** 保留 β-CD 生物主客体脉络的**唯一确切衍生物**候选（6-位单取代脲-β-CD），把 BPA-2R 的共定位证伪问题冻结为确切分子。
- **关键风险：** 总分最低；共定位增量 tier none；β-CD 广谱、先例拥挤、容量天花板。
- **深化必答唯一问题：** 确切次级口单极性衍生物的完整体，其 BPA 增量能否超未改性 β-CD + 单独极性单元统计加和？不能则退出。

## 2. 淘汰/降级方案（按主导原因分组）

### 因硬对应缺口（材料机制冒充生物原型 / 非生物源）
- **G 印迹族全体（U63–U72）**：印迹为材料技术，biological_feature 缺口；U65/U66/U72 因与仿生识别单元杂化保留为 revise 对照，其余降 control_only 或类别捕获。原创性受先例密度压制（MIP-炭、假模板均成熟）。
- **C 合成大环族（U26/U27/U28/U30/U31）**：杯芳烃/柱芳烃/CB/间苯二酚芳烃/CTV 为合成大环非生物源，BPA 专属水相包合 tier none，伴容量天花板；降 revise/control。C 族仅 U25（β-CD 生物寡糖）保留承重脉络。
- **J 工程族（U91/U95/U96/U97/U98）**：阴离子交换、磁分离、商业炭、β-CD 先例基线无独立仿生对应或先例已占；固定为 control_only 工程基线。

### 因选择性可被单调物化参数解释（创新清单 D 不通过）
- **D 传质族纯传质条（U36/U37/U39/U40/U41/U42/U43/U44）**：传质/结构为通用工程增量，无识别维度，选择性随孔容/疏水性单调；降 control_only 或工程对照。D 族仅 U35（承重传质原型）+ 交叉条（U22/U38）保留。
- **H 天然多酚族广谱条（U75/U76/U78/U79）**：黑色素/木质素/单宁/腐殖酸相广谱芳香吸附，BPA 专属增量 tier none；降 control_only。H 族仅叠加判别维度的 U73/U77/U80 保留为 revise。

### 因继承死路重演
- **U33 冠醚-芳香杂化腔（49）**：冠醚适配阳离子不适配中性双酚，设为**阴性对照**证伪"腔即识别"。
- **U64 半共价硼酸酯（59）**：BPA 无邻二醇不能成单个硼酸酯，化学窗口窄；control_only。
- **U82/U83/U84 阴离子型氢键受体**：方酰胺/Hamilton 脲/脒胍偏羧酸/阴离子，对中性单酚弱，重演 P1-MAPy"错把羧酸受体当酚受体"；降 control_only（U81 非对称修正版保留为 revise）。
- **F 牺牲模板条（U58/U59/U60/U61）**：肽/核酸/矿化/胶束模板为材料技术，对应 loose；control_only。F 族保留 U53（硅藻矿化承重）与 U62（周期阵列 + 单锚）。

### 因动态归属风险
- **U34/U92/U99 pH/温敏响应**：ERRγ 不支持动态门控，响应为材料再生机制非仿生动态；作再生维度 revise，不承重。

## 3. 淘汰总结

100 条：承重深化 7（U01/U13/U24/U18/U23/U09/U25，其中 B 族 COF 去重后 U13 承重、U24 方法学、U18 对照），机制/工程对照约 35（G/J/部分 C/D/H/F），其余为 revise 但受 tier none 与族内冗余限制暂不深化。

首要淘汰判据仍是硬对应 biological_feature + source_support：G 印迹、J 工程、C 合成大环、F 牺牲模板因"生物原型"实为材料机制或非生物源被降级；D 纯传质与 H 广谱多酚因选择性单调（创新清单 D 不通过）降对照。

保留 7 条里，**U01（漆酶 Cu 配位）与 U13（非对称单酚锚 + 桥基判别）是本轮相对既有 39 条的两处实质仿生对应增量**：U01 把承重原型移到自然界真实处理酚的降解酶（规避思路三），U13 给"分 BPA/BPF/BPS"一个几何判别机制（规避疏水单调）。二者与承接既有最高分的 U24 构成深化三角。无方案达 85；深化目标是闭合"配位判别/桥基判别在盐+DOM 下是否保留"这一共同 tier none 关口。
