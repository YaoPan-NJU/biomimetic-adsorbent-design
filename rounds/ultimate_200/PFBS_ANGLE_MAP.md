# PFBS 角度地图（设计空间 → 100 方案生成矩阵）

> 见 `00_DESIGN_BRIEF.md` §4.2。PFBS 的物理本质决定：绕开中心反例（阳离子=静电=盐敏感）的**唯一足够大的自由能杠杆是脱水选择性**（Hofmeister 反转）；且无任何生物系统进化处理全氟物质，思路三结构性不成立，仿生只能走思路二通用原理。本轮的实质增量是把 `shortchain_redirection_1` 的"概念 α"从假想升级为**成体系、可评分、可漏斗的脱水选择性 + 动力学门闩组合**，并如实记录电静电族/含氟族的方向级否定。

## 1. 污染物锚定事实

- PFBS（全氟丁烷磺酸根），`MW≈300.10`，C4 全氟链 + 磺酸头基，强酸阴离子（水中全解离）；弱水合阴离子。
- 竞争物：**硫酸根（0.1–1 mM，过量 10⁵–10⁶）**、碳酸氢根、氯离子、磷酸根、长链 PFSA（PFPeS/PFHxS/PFOS）、阴离子表面活性剂、DOM。
- 主目标场景：市政二级出水（PFBS ~ng/L–µg/L，约 3 nM），100% 检出（`E001`）。
- **中心反例（已核 `E056`）**：阳离子 β-CD 聚合物 PFBS 去除 1 mM Na₂SO₄ 下 94%→38%，PFOS 仍 ~79%。
- **物理杠杆**：水合自由能 硫酸根 ~−1080、磺酸根 ~−300 kJ/mol（教科书/Marcus 惯用值，原始出处待核）；脱水成本差 ~700 kJ/mol 是唯一够用的选择性来源。

## 2. 工程难题清单（思路二）

| 编号 | 工程难题 | 传统方法为何原理上做不到 |
|---|---|---|
| P1 | 硫酸根 10⁵–10⁶ 倍竞争下捕获痕量 PFBS | 电荷/强氢键给体对高水合二价硫酸根超额补偿，选择性方向被拉回硫酸根 |
| P2 | 长链 PFAS 置换短链、短链提前穿透（出水>进水） | 疏水分配按链长单调，短链本征弱、先被置换 |
| P3 | DOM 大分子孔口污堵 + 100–1000 Da 小级分位点竞争 | 允许 ~300 Da PFBS 通过的界面必放行同尺寸 DOM 小级分 |
| P4 | 无氟约束下的选择性识别 | 含氟骨架生命周期负担 + 监管反对；无氟识别化学在约束内文献层不存在 |
| P5 | 弱结合短链的可逆再生 | 强结合再生难、弱结合选择性差的两难 |

## 6. 承重家族痛点-仿生因果链

- **A（核心）**：PFBS 在硫酸根 10⁶ 倍过量下被捕获的根因障碍是电荷/强给体对高水合硫酸根超额补偿。离子通道选择性过滤器进化出"按脱水成本区分离子"——弱配位、无强给体、疏水屏蔽使强水合离子欠补偿、弱水合离子优先。提取该原理造**无强给体的脱水选择性弱配位孔**（anion-π 缺电子面 / C-H 给体大环 / 低极性醚环境），使弱水合 PFBS 磺酸头优于强水合硫酸根——这是唯一在物理上能反转硫酸根方向的机制。
- **B（抗置换）**：短链本征弱疏水、被长链置换先穿透（P2）。FABP 门区以静态几何门控配体进出。提取"孔口静态门闩 + 腔内分配"造动力学陷阱：PFBS 进入后驻留、抵抗长链置换。**仅取静态门区几何，不主张动态门控（S1 教训）**。
- **C（几何）**：~7 Å 一维通道几何排除 PFOS C8 全长、纳 PFBS。须证明窗口对小硫酸根也排除，否则须壁面脱水选择性协同（并入 A）。

## 7. 家族级已知风险（继承 §4.2 死路，逐族预挂）

- A：脱水选择性对 PFBS/硫酸根的分配窗口须 ≥10²–10³，外推不足即证不成立；水相 anion-π/C-H 给体强度数据稀疏（tier none/lead）；孔化学与 C4F9 尾相容性。
- B：门闩化学须从盐稳定识别族（A 的弱配位）选取，不得引入阳离子/氟；长链置换须预注册杀伤实验（置换即终止）。
- C：先例（Zeppuhar COF-PFAS）高容量实为二甲铵阳离子静电，非尺寸门；窗口不排除小硫酸根。
- D：anion-π/卤键/方酰胺原型多只测 F⁻/Ac⁻/Cl⁻/H₂PO₄⁻，未测硫酸根/磺酸根判别，易被优化成硫酸根受体（胍基陷阱）。
- E：抗污机制先验被占（Cai 2017）；小级分 DOM 结构性不可恢复；须证明非纯抗污。
- F：传质为通用增量，不解决硫酸根选择性。
- G：再生液硫酸根仍过量 ~10³ 倍，只改善约三数量级非消除；须定量回答"为何不直接用选择性吸附剂于主床"。
- H：含氟载体已被占据（Kasula 2026）、监管反对（Liu 2026）；仅作强制工程对照，标注 lifecycle。
- I：电荷/胍基=中心反例，1 mM Na₂SO₄ 下失效；仅作否定记录与阳性对照。
- J：β-CD 亲和力天花板 ~2.2×10³ L/kg 结构性不可达目标 10⁴；MIP 模板泄漏含氟负担；先例拥挤。
