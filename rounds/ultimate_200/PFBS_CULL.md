# PFBS 组合攻击/淘汰记录（Ultimate 轮，隔离攻击者）

角色：隔离攻击者/排序者。输入：`PFBS_PORTFOLIO_01-50/51-100.md` 100 方案、`research_contract.yaml`、`INNOVATION_CHECKLIST.md`、继承死路（`00_DESIGN_BRIEF.md` §4.2）。不杜撰 DOI/数值/机制；PFBS 专属"盐下优于硫酸根与长链"增量未核到者标 `tier none`。硬测试矩阵与阳性对照见 `PFBS_PORTFOLIO_01-50.md` 抬头。

## 0. 攻击主轴

1. **中心反例过滤（最高优先级）**：任何本质等同单电荷位/阴离子交换的方案在 1 mM Na₂SO₄ 下重演 94%→38% 失效——**I 家族全体（U77–U86）直接判否定记录/对照**，不进入承重。
2. **含氟生命周期负担**：机制与含氟单元不可分离者（H 家族 U69–U76）判强制对照，标注 lifecycle + 监管（Liu 2026）。
3. **思路三红线**：无生物进化处理 PFAS；承重原型只能是通用工程原理（脱水选择性、静态门区、矿化孔、传质、系统）。命中"找能结合 PFBS 的蛋白"即否。
4. **硫酸根判别缺口**：D 家族（anion-π/卤键/双脲/方酰胺）原型多未测硫酸根/磺酸根判别，易优化成硫酸根受体（胍基陷阱）——须正面回答，否则降级。
5. **选择性单调性 + β-CD 天花板**：J/F 广谱条与 β-CD（Kd 天花板 2.2×10³ < 目标 10⁴）判对照。

## 1. 保留方案（Concept α 主线 + 强制对照）

### 承重主线（合并深化为一条 Concept α）｜U13 + U21 + U01 + U11（74/72/72/73，partial）
- **存活理由：** 这是**物理上唯一能反转硫酸根方向的机制**。离子通道选择性过滤器进化出"按脱水成本区分离子"（真实进化机制），转译为**无强给体的脱水选择性弱配位孔**（anion-π 缺电子面 / cyanostar 类 C-H 给体 / 低极性醚），使弱水合 PFBS 磺酸头优于强水合硫酸根（脱水成本差 ~700 kJ/mol）；叠加 FABP 门区静态门闩囚禁抗长链置换（P2）。无氟、无阳离子，直接规避中心反例与含氟负担。相对既有 PFBS-11/6/17/19 的实质增量是把"脱水选择性"作为**明确承重杠杆**而非附带，并绑定四条预注册杀伤实验。
- **两个硬对应：** ① 脱水选择性弱配位孔（离子通道原理，Hofmeister 反转，source=离子通道选择性机制 class 级）；② 孔口静态门闩 + 腔内分配（FABP 门区静态几何，动力学囚禁抗置换）。
- **关键风险：** 脱水选择性对 PFBS/硫酸根的分配窗口能否 ≥10²–10³ 是 tier none 的核心赌注；水相 anion-π/C-H 给体强度数据稀疏；门闩化学水相强度未证；DOM 小级分对孔口机制结构性不可恢复。
- **深化必答唯一问题（可溶门预验）：** 以 Rekharsky-Inoue 汇编与树脂脱水选择性数据为边界，脱水选择性弱配位孔对 PFBS 相对硫酸根的分配窗口外推是否 ≥10²–10³？不足即如实终止 α，不换原理挽救。

### 几何维度承重｜U23 尺寸门 β-酮烯胺 COF（PFBS-11 增量，69，partial）
- **存活理由：** 几何非电荷维度，可与 α 壁面脱水选择性协同（U11/U27）。
- **深化必答唯一问题：** ~7 Å 窗口能否在保留 PFBS 全长的同时排除小硫酸根？若不能，须并入 α 由壁面脱水选择性承担判别。

### 强制对照｜U77 阳离子 β-CD（PFBS-1 中心反例）+ U69 氟化活性炭（Kasula 2026）
- **用途：** U77 为盐敏感失效的**阳性对照**（同批测 94%→38%）；U69 为**含氟工程对照**（标注 lifecycle），任何 α 候选须在无氟无阳离子下超越或匹配其容量而不带二者缺陷。

## 2. 淘汰/降级（按主导原因）

### 因中心反例（电荷=盐敏感）——否定记录
- **I 家族全体（U77–U86）**：阳离子 β-CD、胍基-双脲、季铵/鏻 IX、开放金属位/缺陷 MOF、强碱混合模式、胍基探针——全部本质单电荷位/阴离子交换或偏硫酸根。U78/U85 判 terminate（胍基优化成硫酸根受体 / 非水相萃取），其余 control_only。U86 为方向级否定记录条。

### 因含氟生命周期负担——强制对照
- **H 家族全体（U69–U76）**：氟化炭/部分氟化/PFPE/氟凝胶/氟化 PIM——机制与含氟不可分离，载体已占（Kasula 2026）、监管反对（Liu 2026）、链长反转本征不可能。全部 control_only，U76 terminate。

### 因硫酸根判别缺口
- **D 家族部分（U36 双脲、U37 方酰胺、U40 硼酸酯）**：原型偏硫酸根或对磺酸盐弱，易优化成硫酸根受体；降 revise/control。D 族仅 anion-π 条（U33/U34/U38）与卤键（U35/U41）保留为可与 α 协同的几何维度，且须正面测硫酸根判别（U42 探针预注册）。

### 因选择性单调 / 天花板 / 先例拥挤
- **F 纯传质条（U52/U56/U57/U58/U59/U60）**：传质通用增量，不解决硫酸根选择性；control_only。F 族仅承重交叉条（U51/U53/U54）保留。
- **J β-CD/印迹/混合族（U87/U89/U91/U92/U93/U94/U95/U96/U97/U98）**：β-CD Kd 天花板结构性不可达、MIP 模板泄漏含氟、先例拥挤、商业 IX 基线；control_only。J 族仅 U90（非氟假模板）保留 revise、U100（整合旗舰）承重。
- **G 系统架构（U61–U68）**：系统级贡献，但再生液硫酸根仍过量 ~10³ 倍、须定量回答"为何不直接用选择性吸附剂于主床"；保留 U63（两床抗置换）、U67（再生分离）为 revise，其余 control。
- **E 抗污族（U43/U46/U47/U50）**：纯抗污机制先验被 Cai 2017 占据；control_only。E 族仅与 α/门闩协同条（U44/U48）与蜕皮更新条（U45）保留 revise。

## 3. 淘汰总结

100 条：承重深化实为**一条 Concept α 主线**（U13/U21/U01/U11 合并 + U23 几何协同），加两条强制对照（U77 中心反例阳性对照、U69 含氟对照）。约 55 条为 control_only/terminate 的工程对照与否定记录（I 电荷族、H 含氟族、J/F 广谱与基线）——这些**如实保留是本轮的诚实价值**：它们把"电荷=盐敏感""含氟=lifecycle+监管""β-CD=天花板"三条方向级否定钉死，使 α 的承重地位有明确的阴性锚点。

首要淘汰判据是中心反例（电荷=盐敏感）与含氟负担：I 家族全体、H 家族全体因重演已核失效或不可分离含氟被降级；D 部分因硫酸根判别缺口降级；J/F/G/E 广谱与基线因选择性单调、β-CD 天花板或抗污先例被占降对照。

无方案达 85，PFBS 组封顶 74。深化目标是用**可溶门预验**回答 α 的唯一承重赌注——脱水选择性弱配位孔对 PFBS/硫酸根分配窗口是否 ≥10²–10³。这是 PFBS 能否写出高质量论文的分水岭；外推不足即如实记为方向级否定，与 `shortchain_redirection_1` 结论一致。
