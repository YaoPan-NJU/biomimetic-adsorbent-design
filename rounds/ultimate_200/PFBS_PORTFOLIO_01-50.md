# PFBS 组合：方案 U01–U50（家族 A–E）

> 见 `PFBS_ANGLE_MAP.md`。字段与 BPA 组合一致。PFBS 为更难目标（无生物专属原型、硫酸根 10⁶ 倍竞争），全组封顶分低于 BPA，如实反映。承重轴必须非电荷、非强氢键（否则重演中心反例）。PFBS 专属"盐下优于硫酸根与长链"的增量凡未核到者标 `tier none`。硬测试矩阵统一：PFBS/PFPeS/PFHxS/PFOS 同存；纳米纯水 / 1 mM Na₂SO₄ / 氯化物 / 真实二级出水；报批量去除 + PFBS 饱和 mg/g + 床层 mg/mL + 首先穿透物；以 PFBS-1 阳离子 β-CD 为同批阳性对照。

## 家族 A｜脱水选择性弱配位孔（Hofmeister 反转，离子通道过滤器原理）（U01–U12）

立论：硫酸根 10⁶ 倍过量下捕获 PFBS 的根因障碍是电荷/强给体对高水合硫酸根超额补偿。离子通道选择性过滤器进化出"按脱水成本区分离子"——**弱配位、无强给体、疏水屏蔽使强水合硫酸根欠补偿、弱水合 PFBS 磺酸头优先**。这是物理上唯一能反转硫酸根方向的机制（脱水成本差 ~700 kJ/mol）。材料为合成非蛋白弱配位孔。

### PFBS-U01｜anion-π 缺电子三嗪脱水选择性孔（PFBS-6 增量：Hofmeister 判据）
- **工程难题：** P1 硫酸根竞争下捕获 PFBS。
- **生物原型·原理：** 离子通道选择性过滤器脱水选择性（真实进化机制）；缺电子三嗪 π 面提供**弱配位、无强给体**的脱水选择性识别，使弱水合磺酸头优先于强水合硫酸根。
- **硬对应：** ① 脱水选择性弱配位孔（离子通道原理，Hofmeister 反转）；② 缺电子 anion-π 几何面（非电荷）。
- **识别单元·材料：** 三嗪多孔有机聚合物（无氟、芳腈三聚成型）。
- **竞争与停止：** 硬测试矩阵；若 PFBS/硫酸根分配窗口外推 <10² 或盐下坍缩为普通疏水吸附即停。
- **评分：** 因果15/选择性16/转译14/原创13/可证伪8/证据6 = **72**，revise。
- **对应强度·风险：** partial。水相 anion-π 对 PFBS 强度 tier none；须证明反转方向而非仅疏水分配。

## 家族 B｜孔口动力学门闩 + 腔内分配抗置换（Concept α，FABP 门区静态几何）（U13–U22）

立论：短链本征弱疏水、被长链 PFAS 置换先穿透（P2，出水>进水）。FABP 门区以静态几何门控配体进出。提取"孔口静态门闩 + 腔内分配"造**动力学陷阱**：PFBS 进入后驻留、抵抗长链置换。**仅取静态门区几何，不主张动态门控（S1 教训）**。门闩化学从 A 族盐稳定弱配位选取，不引入阳离子/氟。

### PFBS-U13｜Concept α 旗舰：脱水选择性门闩孔口 + 分级孔腔内分配 + 动力学囚禁
- **工程难题：** P1+P2 综合（硫酸根竞争 + 长链置换）。
- **生物原型·原理：** 离子通道脱水选择性（门闩化学）+ FABP 门区静态几何（囚禁）+ 木质部分级传质三原型协同。
- **硬对应：** ① 脱水选择性弱配位门闩（离子通道，Hofmeister 反转）；② 孔口门闩静态几何 + 腔内分配（FABP 门区，动力学囚禁抗置换）。
- **识别单元·材料：** 分级介孔碳 + 孔口脱水选择性弱配位门闩 + 腔内分配（**首选炭，无氟无阳离子**）。
- **竞争与停止：** 预注册杀伤实验：① 窗口外推 <10² 迭代；② 长链 PFSA 冲击置换即终止；③ 门闩水相/硫酸根判别不足即终止；④ DOM 小级分主导则降级。
- **评分：** 因果15/选择性16/转译15/原创13/可证伪9/证据6 = **74**，revise。
- **对应强度·风险：** partial。PFBS 组最高但门闩化学水相强度 tier none；双维协同归因难。
