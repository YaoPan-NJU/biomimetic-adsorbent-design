# BPA-U13 深化：非对称单酚锚 + 桥基判别 β-酮烯胺 COF

- 编号：`BPA-U13`（Ultimate 轮 B 家族承重）｜分诊分 80/revise（本轮 BPA 最高）
- 定位：相对既有 BPA-1（对称夹持已否）与 BPA-3（对称双极性站，全项目历史最高 78）的实质增量——**非对称主锚 + 显式桥基判别面**，把"分 BPA/BPF/BPS"从疏水性转到几何。
- 状态：实验级设计文档，未授权采购/合成/实验。

## 0. 结论先行

BPA-1 的"对称双端 9.27 Å 夹持"已被 ERRγ 突变证据否定（`E039`：同一酚羟基由 Glu275/Arg316 互补夹持为主锚，`4-α-cumylphenol` 单羟基仍强结合）。本方案据此改写为**一壁定距单酚互补主锚 + 对壁疏水凹面读取桥基几何**：BPA 异丙基桥（体积大、疏水）、BPF 亚甲基桥（小）、BPS 砜桥（极性、吸电子）在凹面中占位/极性不同，构成非疏水单调的判别维度。承重赌注（tier none）：桥基凹面判别增量在量化真实位点比例、DOM 竞争下是否成立。

## 2. 科学问题 + 工程难题

- **科学问题**：能否在水稳 COF 刚性通道内以**非对称单酚主锚 + 桥基判别凹面**复现 ERRγ 静态识别，使 BPA 相对 BPF/BPS 获得随桥基几何（而非疏水性）变化的、可被"删主锚/填平凹面/错位"证伪的选择性增量？
- **工程难题**：E1（判别双酚同系物）。传统疏水吸附按 logKow 单调排序原理上不能分 BPA/BPF/BPS。
- **闭合链**：BPA/BPF/BPS 疏水近似难判别 → 传统吸附做不到 → ERRγ 以单酚主锚 + 疏水包围静态识别 → 提取非对称主锚 + 桥基判别凹面 → 水稳 COF 定距布置 → 桥基几何判别 → 四组对照证伪 → 论文主张。

## 3. 硬对应矩阵（≥2，六字段齐备）

### 硬对应 ①：非对称单酚互补主锚（ERRγ 主锚）
| 字段 | 内容 |
|---|---|
| biological_feature | ERRγ 中同一 BPA 酚羟基由 Glu275/Arg316 互补供受氢夹持为**主锚**（非对称，单羟基类似物仍强结合） |
| source_support | PDB 2E2R（`E023/E033`，verified）；ERRγ 突变 + `4-α-cumylphenol` 结合（`E039`，verified_with_boundary） |
| material_realization | COF 一壁定距布置互补供受氢主锚站（如脒-羟基/酰胺-羟基对，避开偏羧酸的酰氨吡啶） |
| expected_function | 对单个酚氧的定向互补主锚，提供超疏水的极性读出 |
| measurement | 主锚删除体 vs 完整体的 BPA 分配增量；固相 NMR/元素分析定量主锚站真实比例 |
| causal_control | 主锚删除体、主锚错位体、主锚化学随机体、等孔容无站 COF |

### 硬对应 ②：桥基疏水凹面非对称判别（ERRγ Tyr326 疏水包围）
| 字段 | 内容 |
|---|---|
| biological_feature | ERRγ 以 Tyr326 等疏水残基包围 BPA 芳香骨架与桥中心（预组织静态环境） |
| source_support | PDB 2E2R 疏水包围（`E033`，verified，仅静态几何）；BPA 两环平面锐角 ~87.4°、双酚氧 9.274 Å（坐标派生） |
| material_realization | COF 对壁刻蚀**异丙基尺寸疏水凹面**，几何读取桥基（纳异丙基、拒砜桥极性/亚甲基欠填充） |
| expected_function | BPA（异丙基）> BPF（亚甲基欠填充）、> BPS（砜桥极性排斥），判别随桥基几何非单调变化 |
| measurement | BPA/BPF/BPS/BPAF 分配序列 vs 疏水性序列的偏离；凹面填平体对照 |
| causal_control | 凹面填平体、凹面放大/缩小体、凹面极性化体、纯疏水无凹面体 |

> 双硬对应正交可证伪：① 极性主锚（删除/错位）；② 几何桥基凹面（填平/尺寸错配）。若 BPA/BPF/BPS 排序仅随疏水性单调、删主锚或填凹面增益不塌陷，则识别主张失败（退化为普通芳香吸附，重演 BPA-3 待闭风险）。

## 4. 材料架构与台式合成 SOP（概要）

- **骨架**：水稳 β-酮烯胺 COF（TpPa 类，1,3,5-三甲酰间苯三酚 + 二胺缩合），酮烯胺互变异构赋予水/酸稳定。
- **主锚布置**：选一壁二胺组分带可后修饰位（如硝基→氨基→脲/脒），限量后修饰引入互补供受氢主锚站；**强制消解 NMR/元素分析定量真实位点比例**（吸取 BPA-3 待闭教训）。
- **桥基凹面**：对壁组分选带异丙基/叔丁基侧基的连接单元，制造异丙基尺寸疏水凹面。
- **对照体同批**：主锚删除/错位/随机、凹面填平/错配/极性化、等孔容化学随机 COF。
- **成型**：COF 粉体黏结成粒或原位生长于分级孔炭（并入 U22 交叉），报床层 mg/mL。
- **表征**：PXRD（结晶度/孔道）、N₂ 吸附（孔径）、固相 ¹³C/¹⁵N NMR + 元素分析（位点比例）、FT-IR。

## 5. 识别机制、选择性策略与竞争物谱

- **机制**：刚性通道内主锚站以固定几何读一个酚氧（极性维度），凹面读桥基（几何维度），两维正交；异丙基桥最佳占位 → BPA 优先。
- **竞争物谱**：BPF/BPS/BPAF/4-枯基酚/苯酚/壬基酚/EE2；DOM；盐。
- **停止规则**：分配序列 = 疏水性序列、或删主锚/填凹面不塌陷、或 DOM 堵孔降级为普通芳香吸附 → 识别主张失败。

## 6. 实验计划与成功/失败判据

1. **位点定量（门 0）**：消解 NMR/元素分析确证主锚站与凹面真实比例（避免 BPA-3 的位点非均一陷阱）；比例过低即先优化后修饰。
2. **等温/动力学 + 选择性**：BPA/BPF/BPS 混合竞争，分离因子 vs 疏水性预测偏离。
3. **对照矩阵**：主锚（删/错/随机）、凹面（填/错配/极性）四组，逐维证伪。
4. **基质**：超纯水→盐→DOM→真实二级出水，判别保留率。
5. **再生/稳定/浸出**：COF 水稳循环、容量保留。
- **成功判据**：BPA 相对 BPF/BPS 的分离因子显著偏离疏水性预测；删主锚且/或填凹面时增益塌陷；DOM 中保留；总干材料/床层容量可接受。
- **失败判据**：排序随疏水单调、对照不塌陷、DOM 堵孔——降级。

## 7. 创新清单自检（A–E）

- A 机制：✅ 桥基几何判别 + 极性主锚双维，超疏水分配。
- B 架构：✅ COF 通道非对称双壁功能分区（主锚壁 + 凹面壁）是新空间组织。
- C 转译：✅ ERRγ 静态几何非对称化 + 桥基读取，超官能团类比。
- D 选择性：✅ 不可由 logKow 单调解释（凹面填平对照隔离几何维度）。
- E 先例：⚠️ 双酚 MIP/COF 有先例，须以非对称 + 桥基判别的因果对照量化增量。
- 判定：A/B/C/D 四维创新，通过。

## 9. 对抗审计与待闭问题

- **High 风险**：限量后修饰位点非均一（BPA-3 同款风险）——本方案以门 0 位点定量正面处理；水中主锚氢键被屏蔽；DOM 堵孔。
- **待闭唯一问题**：在量化真实主锚/凹面比例下，桥基凹面是否使 BPA 优于 BPF/BPS 且 DOM 中保留、删主锚/填凹面增益崩塌？
- **与 U24 去重**：U24（BPA-3 增量对称双站 + 位点定量）作方法学对照；U13 承重（非对称 + 桥基判别新机制）。二者共享位点定量 SOP，深化二选一为主线（建议 U13）。
- **处置**：`revise`，承重深化候选。
