# PFBS Concept α 深化：脱水选择性弱配位门闩孔口 + 分级孔腔内分配

- 编号：`PFBS-U13/U21/U01/U11` 合并（Ultimate 轮 A+B 承重主线）｜分诊分 74/revise（本轮 PFBS 最高）
- 定位：把 `shortchain_redirection_1` 的"概念 α"从假想升级为**成体系、可评分、含四条预注册杀伤实验的承重方案**；无氟、无阳离子，正面攻硫酸根竞争的物理根源。
- 状态：实验级设计文档，未授权采购/合成/实验。

## 0. 结论先行

PFBS 在二级出水中被硫酸根（0.1–1 mM，过量 10⁵–10⁶）压制，且被长链 PFAS 置换先穿透（P1+P2）。电荷/强氢键给体对高水合硫酸根超额补偿——这是中心反例（阳离子 β-CD 94%→38%）的物理根源。**唯一足够大的反转杠杆是脱水选择性**：硫酸根 `ΔG_hyd≈−1080 kJ/mol`（Macreadie 2022 *Chem*，已核）而磺酸根仅 ~−300 kJ/mol，差 ~700 kJ/mol。自然界离子通道选择性过滤器以脱水成本区分离子（Solymosi 2025：脱水支配天然离子通道与合成离子筛的选择性，已核）；弱水合阴离子会脱去水化壳并吸附到非极性界面（Rogers et al.，已核）。本方案提取"**无强给体的脱水选择性弱配位孔**"，叠加 FABP 门区静态门闩囚禁抗置换。承重赌注（tier none）：脱水选择性弱配位孔对 PFBS/硫酸根的分配窗口能否 ≥10²–10³。深化把该赌注压到**可溶门预验**——不足即如实终止，不换原理挽救。

## 2. 科学问题 + 工程难题

- **科学问题**：能否用**无强给体、无阳离子、无氟**的脱水选择性弱配位孔，使弱水合 PFBS 磺酸头相对强水合硫酸根获得 ≥10²–10³ 的分配窗口（Hofmeister 反转），并以孔口静态门闩抵抗长链 PFSA 置换？
- **工程难题**：P1（硫酸根 10⁶ 倍竞争）+ P2（长链置换、短链早穿透）。传统电荷/氢键吸附**原理上**被硫酸根超额补偿；含氟路线有 lifecycle+监管负担且链长反转本征不可能。
- **闭合链**：PFBS 被硫酸根压制根因（电荷/给体对高水合硫酸根超额补偿）→ 传统吸附与含氟路线都做不到（前者盐敏感、后者 lifecycle）→ 离子通道以脱水成本区分离子（真实进化机制）→ 提取无强给体脱水选择性弱配位孔 + FABP 门区静态囚禁 → 合成非蛋白弱配位孔炭 → 脱水选择性 + 抗置换 → 四条预注册杀伤实验 → 论文主张或方向级否定。

## 3. 硬对应矩阵（≥2，六字段齐备）

### 硬对应 ①：脱水选择性弱配位孔（离子通道选择性过滤器原理，Hofmeister 反转）
| 字段 | 内容 |
|---|---|
| biological_feature | 离子通道选择性过滤器以脱水成本区分离子（弱配位、无强给体、疏水屏蔽使强水合离子欠补偿） |
| source_support | 脱水支配天然离子通道与合成离子筛选择性（Solymosi 2025, real.mtak.hu/231188, verified）；硫酸根高水合能阻碍水相选择性结合（Macreadie 2022 *Chem*, ΔG_hyd −1080 kJ/mol, verified）；弱水合阴离子脱水化壳吸附非极性界面（Rogers et al., verified）；anion-π 提取弱水合 ReO₄⁻（Qin 2023, verified 类比） |
| material_realization | 无强给体的弱配位孔：缺电子 anion-π 面（三嗪/NDI POP）/ C-H 给体大环（cyanostar 类）/ 低极性醚受限疏水孔；**无阳离子、无氟、无强氢键给体** |
| expected_function | 弱水合 PFBS 磺酸头分配 > 强水合硫酸根（脱水成本差驱动），窗口目标 ≥10²–10³ |
| measurement | 可溶弱配位受体/孔对 PFBS vs 硫酸根/氯/碳酸氢根分配系数；¹⁹F NMR/ITC；分配 vs 强给体阳性对照（应偏硫酸根） |
| causal_control | 强给体阳性对照（应偏硫酸根，反证方向）、纯疏水无弱配位对照、阳离子对照（应盐敏感）、Hofmeister 序列多阴离子 |

### 硬对应 ②：孔口静态门闩 + 腔内分配（FABP 门区静态几何，动力学囚禁抗置换）
| 字段 | 内容 |
|---|---|
| biological_feature | FABP 门区静态几何门控配体进出（**仅取静态门区，不主张动态门控——S1 教训**） |
| source_support | FABP 门区结构 class 级（tier lead）；动力学囚禁"约束结合"Cram 半囚禁经典先例（class 级） |
| material_realization | 分级介孔碳孔口布置脱水选择性弱配位门闩 + 腔内分配，PFBS 进入后驻留 |
| expected_function | PFBS 进入-驻留抵抗长链 PFSA 置换（正面攻 P2：短链早穿透、出水>进水） |
| measurement | 进入-驻留表型拆分；PFBS 负载后长链 PFSA 冲击置换实验；床层首先穿透物 |
| causal_control | 无门闩敞孔对照、门闩尺寸系列、长链冲击置换对照（置换即证门闩失效） |

> 两硬对应：① 提供选择性（脱水选择性，非电荷）；② 提供抗置换（静态门闩动力学囚禁）。门闩化学**必须**从 ① 的弱配位族选取，不得引入阳离子/氟（否则重演中心反例/lifecycle）。

## 4. 材料架构与台式合成 SOP（概要）

- **基底**：分级介孔-微孔活性炭（首选，无氟无阳离子）。
- **弱配位面**：表面接枝/共聚缺电子 anion-π 面（三嗪）或 C-H 给体大环（cyanostar 类）或低极性醚环境；控制强给体/阳离子为零。
- **孔口门闩**：介孔口引入尺寸受限的弱配位门闩单元，形成腔内分配的动力学陷阱。
- **对照体同批**：强给体阳性体（应偏硫酸根）、阳离子体（应盐敏感，= PFBS-1 类）、纯疏水无弱配位体、敞孔无门闩体、含氟对照（Kasula 类）。
- **表征**：N₂/CO₂ 吸附（分级孔）、XPS（无金属/无氟确证）、zeta（近中性）、¹⁹F NMR（PFBS 定量）。

## 5. 识别机制、选择性策略与竞争物谱

- **机制**：弱配位孔无强给体，硫酸根因高水合能欠补偿被排斥，弱水合 PFBS 磺酸头脱水后优先分配；门闩囚禁抵抗长链置换。
- **竞争物谱**：SO₄²⁻（核心）/HCO₃⁻/Cl⁻/PO₄³⁻；PFPeS/PFHxS/PFOS（链长置换）；DOM。
- **停止规则（预注册四杀伤）**：① PFBS/硫酸根分配窗口外推 <10² → 迭代；② 长链 PFSA 冲击置换发生 → 终止门闩；③ 门闩化学水相/硫酸根判别不足 → 终止；④ DOM 小级分主导损害 → 按边界降级。

## 6. 实验计划与成功/失败判据

1. **可溶门预验（门 0，决定性）**：以 Rekharsky-Inoue 汇编与树脂脱水选择性数据为边界，测/外推弱配位受体对 PFBS 相对硫酸根的分配窗口；<10² 即如实终止 α。
2. **等温/动力学 + 选择性**：硬测试矩阵全跑，PFBS 相对硫酸根/长链的分离因子。
3. **对照矩阵**：强给体（偏硫酸根）、阳离子（盐敏感）、纯疏水、敞孔、含氟——逐维证伪与锚定。
4. **抗置换**：PFBS 负载后长链冲击，进入-驻留拆分。
5. **基质/再生/浸出/稳定**：真实二级出水、床层突破、再生保留。
- **成功判据**：无氟无阳离子弱配位孔在 1 mM Na₂SO₄ 与真实二级出水中对 PFBS 的分配增量 ≥10² 且随脱水选择性设计变量变化（强给体对照偏硫酸根）；长链冲击不置换；容量/床层可接受。
- **失败判据（方向级否定）**：窗口 <10²、或随普通疏水单调、或长链置换、或盐下坍缩——如实记为方向级否定（与 `shortchain_redirection_1` 一致），不换原理挽救。

## 7. 创新清单自检（A–E）

- A 机制：✅ 脱水选择性（Hofmeister 反转）用于 PFAS 吸附，书目级为空。
- B 架构：✅ 弱配位门闩孔口 + 腔内分配的动力学抗置换架构。
- C 转译：✅ 离子通道脱水选择性 + FABP 静态门区，通用原理转译（非思路三）。
- D 选择性：✅ 明确非电荷、非强氢键；强给体对照隔离脱水维度。
- E 先例：✅ Hofmeister/脱水选择性明确用于 PFAS 吸附书目级为空（redirect 已核）；须四篇近邻全文核验确认无 Hofmeister 论述。
- 判定：A/B/C/D/E 多维创新，通过（PFBS 组唯一多维创新条）。

## 9. 对抗审计与待闭问题

- **Critical 赌注**：脱水选择性弱配位孔对 PFBS/硫酸根窗口能否 ≥10²–10³ 是全方案生死点，tier none；水相 anion-π/C-H 给体强度数据稀疏；"Absence of Hofmeister Selectivity in Hydrophobic Ion-pairing"（Solymosi 2025）提示某些疏水体系无 Hofmeister 选择性——正是须实验判定的边界。
- **High 风险**：DOM ~300 Da 小级分与 PFBS 尺寸接近，对孔口机制结构性不可恢复；长链置换。
- **待闭唯一问题（门 0）**：脱水选择性弱配位孔对 PFBS 相对硫酸根的分配窗口外推是否 ≥10²–10³？不足即终止 α。
- **处置**：`revise`，PFBS 承重深化唯一主线；门 0 可溶预验决定生死。诚实预期：这是 PFBS 能否写出高质量论文的分水岭，存在较高概率触门 0 而记为方向级否定。
